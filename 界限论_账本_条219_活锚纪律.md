## 〇、总纲：不钉是不可能的，不钉死是可以的

**不可能的一半（定理，防幻觉）**：落账必带锚——判定需外接态（T3/T4【零件】），每次赋真值都在跑双贷款（打桩＋转贷，条217）；有内容的逻辑必带锚——"背景无关的逻辑"只有两种空的：全已判（水晶棺）与永不判（泥沼）（本会话四格图【纪律·零件已封】）。**凡声称"完全不锚定就能落账/判定"的方法，都是两种空逻辑之一的伪装。**

**可以的一半（三根支柱，指导由此合法）**：

1. **锚不随记录传播**【零件，§7.2】：账本 $\mathcal{A}_{\text{phys}}$ 阿贝尔、余圈在其上恒等于一——锚在落账那一刻被消耗，账落定之后是**态无关**的。**落账带锚，账不带锚。**
2. **推理层有典范免锚的死片段**【A】：Gödel–Gentzen 双否翻译（经典可保守嵌入直觉主义，$\neg\neg$-稳定＝"不可翻案"＝落账不可逆性的证明论孪生）＋任何 topos 自带典范布尔子宇宙 $\mathrm{Sh}_{\neg\neg}$（不需任何选择）。**在已付讫内容上做经典推理，免锚是构造性的、典范的。**
3. **锚的支付按笔且不可逆**【零件，B1】：每次落账把参照不可逆地挪动一次——**锚是按笔借、按笔还的燃料，不是一次性买断的地基。**

**总纲一句**：**每一笔都钉，任何一笔都不钉死。** 钉＝为这一笔借锚、付费、落账、锚随笔消耗；钉死＝让某个锚（态、骨架、或判定参照）跨笔存续而不再出现在成本清单上。**报出成本不等于付完代价——锚是燃料，报得出只说明它在烧、烧的方向不可撤，不说明烧完了不用再借。** 三种钉死＝三种死（冻态→水晶棺／冻骨架→泥沼／冻锚→燃尽【纪律·四格图】）。

## 一、七条活锚纪律

**纪律一（态侧）：锚按笔借。** 每次判定显式带自己的 $\omega_{\text{ext}}$；判定完成，锚即耗尽，下一笔重借。禁止任何"从此以后都用这个参照"的默认。〔防：燃尽（冻锚）——失配累积、账单发散；定理根：锚不可内生（W.3）＋锚逐次暂时标定（条192 跟随结构）。〕

**纪律二（算子侧）：桩可拔，基无典范。** 投影族／粗粒化／指针基按轮备料（活世界不发现成投影，备料＝D，条217 §六）；禁止把任何一组基典范化为"世界本身的分法"。〔防：泥沼（冻骨架）——固定件全局拼不拢；定理根：无绝对指针基（第五章红线）＋嵌法无典范代表、无测度（A1 封顶）。〕

**纪律三（逻辑使用边界）：排中不过落账线。** 经典逻辑（二值、排中、反证出存在）只许用于**已落账或规定性落账**的内容；对未落定者——开着的未来、他者的内里、正在构成的东西——行二值即为走私一个冻结的锚。〔防：水晶棺式误用（把未判当已判）；定理根：经典＝直觉主义的 $\neg\neg$-稳定片段【A】＋筛真值朝过去开口、不含未来（已封）。〕

**纪律四（产出定义）：形式化到僵硬，产出取残差。** 用经典工具分析两本账，**要的不是闭合是残差**——推不平处＝干涉项裸露＝他者的信使＝下一次打桩的洞。尺子必须冻僵才能显影（漂着的锚吸收自己的失配——直觉算不出自己的残差，这正是要形式化的全部理由）。**分析完美闭合时，先怀疑尺子漂了，再怀疑领域死了。**（注意：此处说的是形式系统在活认知中的岗位——残差检测；纯数学内部，经典工具另有它的生活与价值，此条不裁决数学哲学。见§四·2。）〔防：漂移吸收（活判定的固有病）；定理根：残差＝$D(\alpha,\beta)\neq0$ 的裸露＋他者只在账与账拼不拢处显形（Bell/KS 型，结构引用）。〕

**纪律五（残差分类义务）：每个残差判"价格还是墙"。** 可付的（重奠成本、B1 的整笔）＝价格；不可付的（$[\sigma]$、三不可及）＝墙。两种误判各滑向一死：把价格误读成墙＝泥沼（尾声原句）；把墙误读成价格＝燃尽（为冻锚下不可能的拟合无限付费）。〔防：两种误读的对偶；定理根：$[\sigma]$ 板吃不下（条217）＋三不可及（第六章）。〕

**纪律六（节律）：平台持守，跃迁重奠。** 残差在跟随能力内（条192：慢变速率＜回归平衡率）——持守，付稳定燃料，**不许每见失配就拔桩**（无平台＝无落账＝滑向泥沼）；残差越过跟随能力——重奠，付整笔，**不许拒绝**（拒绝＝冻锚＝燃尽）。活＝平台与跃迁的交替；纪律＝守住交替本身。〔定理根：奠立＝跃迁（已封）＋条192/174。〕

**纪律七（账目结构）：两读分账，跨账比对，禁止合账。** 算子读与态读各记各账、各推到闭包，比对取残差；**禁止假设一本"总账"同时记线与类**——那是要一把不存在的迹（W.3），互补盲不可合并（条217 §七）。合账的冲动本身就是钉死冲动的另一种形式。

## 二、判据表：钉 vs 钉死（操作检验）

| 检验 | 活锚（钉） | 钉死（滑向某死） |
|---|---|---|
| 成本清单 | 每笔判定能报出自己的锚与价格 | 锚消失于清单——没人记得它是借的 |
| 残差去向 | 有登记处，被收割、被分类（纪律四五） | 被解释掉／被尺子的漂吸收 |
| 排中的位置 | 只在落账线之后 | 用于未来、他者、构成中者 |
| 公设的身份 | 标为桩：何时打的、替代什么、价码多少 | 称为"自明"（自明＝跟随顺畅的**报告**，明见＝$D=0$ 对齐的感受【零件】——它陈述我的锚定状态，不陈述命题的免锚地位） |
| 对称性的身份 | 标为规范固定：一次选锚动作（在多个等价固定方式中选一，A1） | 称为"世界本有的样子" |
| 重奠的身份 | 支付（可作预算） | 失败／背叛（于是被拒绝→燃尽） |
| 闭合的解读 | 触发怀疑（纪律四） | 当作完工 |

## 保什么：三条精确轴（弃"保态/保算子"gloss；★条220 faxing QDS 熵-切割问）

> **性质**：本节纠正一个反复绊人的表述——"历史一致性保态、topos 保算子"把两条不同的轴搅成一句。拆开三条精确轴，零件全已封，不添定理。

**病根**：三样东西被"保态/保算子"混掉了——**读法轴**（态读/算子读，描述性、对偶、永远一致，条215）／**操作轴**（可逆 σ_t 过程 vs 落盘承诺）／**切割轴**（同一切割 vs 重画切割）。"保态/保算子"当 preservation 属性用是错的：算子读里不动的恰恰是态（海森堡），"保算子"名不副实；两读只是描述选择、永远一致，不是"保不同东西"。

**轴一·读法（描述，对偶，永一致）**：算子读＝态不动/算子动（σ_t 搬）；态读＝算子不动/态动。互为对偶，永远一致（无外部钟，条215）。**不问"保什么"——它只是同一件事的两种记法。**

**轴二·操作（物理，问保不保 $\omega^*$）**：
- **可逆 σ_t 过程 → 保 $\omega^*$**：凡能写成 σ_t 公式的可逆过程（历史一致性判据 $D=\omega_{\text{ext}}(C_\alpha^\dagger C_\beta)$、推理/预演、算子搬运）都保 $\omega^*$（KMS 不变）、不承诺、无 B1。**这才是"历史一致性保态"的准确形——它是 σ_t 判据、保 $\omega^*$；但"topos 保算子"该弃：不落盘 topos（预演/推理）同样保 $\omega^*$。**
- **落盘承诺 → 破保态**：把可能 pin 成一条确定记录、付 B1（不可逆）。**这不是 σ_t 过程，是跳变事件。** 破保态的是"承诺成这一条"这一下，不是"保算子"。
- **落盘 topos＝一次观察＝承诺事件**：不是保态、也不是保算子，是"把态 pin 成 A_phys 记录、付 B1"。（技术：非选择的退相干 pinch 到 A_phys 对 $\omega^*$ 保态，因 A_phys 模不变；破保态的关键动作是"选定成这一条"的承诺。）

**轴三·切割（★条220：跃迁 vs 保态的 QDS 真相）**：faxing 问——跃迁态是低熵过程（调整与环境的划分、保新稳定），保态是自然耗散，物理上是否只有耗散？**核定**：
- **全局只有可逆 σ_t（熵恒定，非耗散）**——耗散只在有切割时才有意义。
- **相对一个切割（A_phys）：粗粒化 ⟹ 熵增 ⟹ 耗散**（Spohn：$S(\omega_{\text{ext}}\|\omega^*)$ 在 QDS 下单调降）。低熵维持＝把熵导出去（本地低熵、全局仍耗散，Prigogine 耗散结构）。
- **★跃迁不是"反耗散的力"，是重画切割（重定义 QDS）**：重切割后，**同一道可逆流**相对**新切割**朝**新平衡**耗散。无新物理过程加入——变的是 QDS 的**定义**。"跃迁低熵"是相对旧切割的假象（态离开旧平衡看似反耗散、靠导熵付账）；相对新切割只是朝新平衡走（就是耗散）。
- **答"物理上只有耗散吗"**：全局只有可逆流；唯一不可逆来源＝切割导致的耗散；**跃迁＝重画切割、非第二个力**。faxing"第一个就是通过不断修改 QDS 定义保新稳定"——**完全正确、且是它的全部**。
- **由此统一**：保态（直觉、骑流）与跃迁（经典落盘、重切割）＝同一道可逆流经过切割的两种样子，差别全在切割（动不动、重不重画）、不在流本身——与"两读共用同一道模流"（条215，描述轴统一）在操作轴同构。

**逻辑映射（精确版）**：直觉逻辑＝不落盘＝骑可逆流（σ_t 保 $\omega^*$、不承诺、活在未落定地带）；经典逻辑＝落盘承诺（B1）——容量内＝往现有账本钉记录（平台持守、同一 QDS）／超容量＝逼跃迁重奠（重画切割、新 QDS）。**经典逻辑逼你不断落盘承诺、承诺累积超容量才逼跃迁**（"逼我不断跃迁"收一档：经典贵在不断承诺、终将逼重切割；直觉廉在骑可逆流不承诺）。

**弃用清单**：不再写"历史一致性保态、topos 保算子"（混轴）；不再把"保态/保算子"当 preservation 二分。改用：可逆 σ_t 过程（保 $\omega^*$）vs 落盘承诺（破保态、B1）vs 跃迁（重切割、非新力）。

## 使用面并入（条177 观察侧对应 ＋ faxing 二问；读法级，零件全已封，措辞待数学会话终签）

> **以下为问答草稿，非条219正式正文。** 零件全已封，不添新定理，不进正文。

**逻辑二分的观察侧对应＝同一次观察的两个时态。**
- 经典逻辑辖区＝已落账片段（$\mathcal{A}_{\text{phys}}$ 阿贝尔、$D=0$、单 context）。读账≠测量：不扰动、不付 B1；任何态限制到沉积交换核给同一读数（翻转不变＝主体间稳定，档 2/3）。锚在落盘那一刻被消耗（支柱一：落账带锚、账不带锚）。＝**完成时**。
- 直觉主义逻辑辖区＝测量尚未完成处（context 未选、账未落）。观察侧签名＝**Kochen–Specker**：跨不相容 context 无全局二值赋值 ⟹ 全局排中被结构禁止 ⟹ 唯一可能真值＝筛真值（第六章，朝过去开口、位置索引）。＝**进行时**。
- 分界＝**落账线**：选定 context 落盘那一下锚被消耗；线下游免锚可读（经典），线上游笔笔带锚（直觉主义）。**两种逻辑＝同一次观察的完成时与进行时，非两种观察。**

**偷锚辨正（纪律三定理根注脚）。** “有没有锚”不是区分二逻辑的轴——有内容判定全带锚（T3/T4；免锚逻辑只两种空的：全已判棺／永不判沼，四格图）；真正的轴＝**锚在不在账上**。直觉主义＝锚戴明处（真值本身带位置索引，Kripke／层语义“在某阶段为真”；拒排中＝诚实记账，非胆小）。经典之免锚地盘＝已清账片段（Gödel–Gentzen／$\mathrm{Sh}_{\neg\neg}$），其免锚是“**付讫**”非“未付”。**越落账线对未落定者行排中＝走私冻锚（不上成本清单的冻结锚）＝第三死（冻锚→燃尽）入口。** 该审查者永远是自称无锚者，非公开带锚者。

**faxing 二问校正。**
- **问一（活观察要否态读＋算子读都读以“避免锚一头”）**：读两幅图与锚是**两条轴**（条177／两轴不叠），读两幅 ≠ 避免锚一头。读两幅＝翻转轴上兼顾过程（态读）与对象（算子读）两半互补内容、不把任一幅当唯一实在；锚是另一轴——有内容判定必带锚、避不开，可为者＝活锚（报得出、可重奠）非冻锚。“避免锚一头”＝活锚纪律，非读两幅。
