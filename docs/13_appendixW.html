W.1 预备:公理与基本定义

W.1.1 公理 \(\Sigma\)

\(\Sigma\) 只有两条:\(\mathcal{M}\) 为 III₁ 因子(无迹)、可分预对偶。III₁ 意味着:无极小投影、无迹、一切非零投影等价——世界里没有一个预先确定、可反复指认的"事实"。世界是纯粹的"作",不是"是"的集合。(前件:理解在发生——两流在场、\(\mathcal{A}_{\text{phys}}\subsetneq\mathcal{M}_{\mathcal{A}}\)——为讨论范围,非定理。) 因子性 = "无绝对客观"的代数身体(等价重述、非推导)。 公理里的"因子"(\(Z(\mathcal{M})=\mathbb{C}1\),中心平凡)不是一条与全书主旨无关的技术选择——它恰是核心断言"没有绝对客观的东西"在中心上的代数身体。等价链:非因子 ⟺ 中心 \(Z(\mathcal{M})\supsetneq\mathbb{C}1\) 有非平凡元 ⟺ 存在对所有内自同构不变的量(中心元与一切酉元对易)⟺ 存在"被给予之前就已固定、对谁都一样"的绝对经典参数(超选择标签、上帝手里的经典信息)。而这正是 G5(第九章)那个"绝对客观=对全体酉群 \(\mathcal{U}(\mathcal{M})\) 不变者"——因子 \(Z(\mathcal{M})=\mathbb{C}1\) ⟺ 对全体酉群不变者只剩标量 ⟺ 没有非平凡的绝对客观量。所以:因子性与"无绝对客观"(G5)是同一命题的两种说法。Remark(分寸):本书保持因子为公理,不声称从"无绝对客观"这条现象学/哲学断言推出它(不搭现象学→代数的桥,守元规则 M);此处只点破二者的代数等价,以收编"想要因子"那股引力——想要因子,不过是嗅到了"无中心"与"无上帝视角"是同一件事。因子不是额外强加的技术条件,是核心断言在代数侧的等价重述。

Remark(因子性为何是自然的工作假设：超选择扇区——一个说明，不是正当性证明)。 上文的等价重述回答"因子性*意味着*什么"，不回答"*为什么*该设因子"。这里给一个说明——它解释了为什么在因子里工作是物理学与本书的自然选择；**但必须先钉死它不是什么：它不是"\(\mathcal{M}\) 必须是因子"的证明，也不是"因子性有独立于哲学的正当理由"这种强宣称。**

说明的内容。 由中心分解 \(\mathcal{M}=\int^{\oplus}\mathcal{M}_z\,d\mu(z)\)（Takesaki I, Ch. IV），任一 von Neumann 代数按其中心摊成因子的直积分；中心元＝与一切可观测量对易者＝超选择荷，其谱把表示分成超选择扇区（扇区间可观测量矩阵元为零、无相干叠加）。物理学的标准做法是固定一个超选择扇区（所有超选择荷取定值）后在其中工作，那里的代数是因子。本书取同样的做法：把"描述认知的结构"取为一个因子，对应"所有超选择荷已取定值"这一设定。

⚠️ 三处必须诚实标死，否则此说明会被误当证明：
(1) 态落单一扇区 ≠ 中心平凡。 "锚态 \(\omega^*\) 在中心上取确定值"（即 \(\omega^*\) 是因子态）是态的性质；"\(Z(\mathcal{M})=\mathbb{C}1\)"是代数的性质。前者不蕴含后者：一个态选定了一个扇区，\(\mathcal{M}\) 的中心仍可非平凡（别的扇区还在，只是这个态没占）。
(2) 从"落单一扇区"到"因子"隐含一步约化，而那一步就是取因子。 要由"落在扇区 \(z\)"得到因子，须把 \(\mathcal{M}\) 约化到 \(\mathcal{M}_z\)；而 \(\mathcal{M}_z\) 是因子，是直积分的构造使然（纤维按定义即因子），不是被"在场"推出的。故"在场 ⟹ 落扇区 ⟹ 扇区内是因子"这条链，其末端是重述：约化到一个扇区，本身就是取因子。诚实的陈述只能是：若把代数取为第一人称锚态所在扇区的那一片，则它是因子——这是建模选择的说明，不是关于 \(\mathcal{M}\) 的定理。（等价的另一种说法：若把 \(\mathcal{M}\) 取为锚态的 GNS 表示生成的代数 \(\pi_{\omega^*}(\mathfrak{A})''\)，则"\(\omega^*\) 是因子态 ⟹ \(\mathcal{M}\) 是因子"确是定理；但前件"锚态是因子态"正是"落单一扇区"的代数写法，说明性同上——它把因子性放进了建模的入口，而非从别处推出。）
(3) 与 W.2.6 耦合注的纪律对齐：本说明不得读作"现象学的'在场'推出代数性质"。 那正是耦合注刚拆掉的那类桥（解读 ⟹ 类型）。这里的"在场／取定值"只在建模设定的层面出现：本书从一开始就把 \(\mathcal{M}\) 设定为一个因子（＝设定工作在一个已定的超选择扇区内），如同物理学固定扇区后再谈局域代数。因子性的地位始终是公理（与 III₁ 同级），本 Remark 只说明这条公理为何是自然、与物理学惯例一致的建模选择，不改其公理地位、不提供额外正当性。

与等价重述的关系：等价重述给因子性的*含义*（因子 ⟺ 无非平凡绝对客观量 ⟺ G5）；本 Remark 给的是*自然性说明*（为何这样建模合乎惯例）。二者都不是正当性证明——因子性是公理，本书不声称把它证出来。

W.1.3.6 "已知"的严格定义:靠"未来不可知"获得边界

"已知"定义为一条增滤流在一个不可内生截断处的那一节：

\[\text{已知} \;=\; \mathcal{A}_{s_0},\qquad\text{其中 } \{\mathcal{A}_s\} \text{ 是 } \sigma_t\text{-协变增滤流：}\]
\[\begin{aligned} &\text{(增)}\quad \mathcal{A}_s\subseteq\mathcal{A}_{s'}\ (s\le s');\\ &\text{(稳)}\quad \text{每节 }\mathcal{A}_s\text{ 皆 }\sigma_t\text{-稳定};\\ &\text{(易)}\quad \text{截面对易};\\ &\text{(满)}\quad \Big(\bigcup\nolimits_{s\le s_0}\mathcal{A}_s\Big)''=\mathcal{A}\quad(\text{到 }s_0\text{ 为止的并充满此刻账本})；\\ &\text{(断)}\quad s_0\text{ 是一个不可内生的截断（下段）}. \end{aligned}\]

⚠️ **(满) 为何必须限定在 \(s\le s_0\)(要紧,防两处误读):(a) 已知 ＝ \(\mathcal{A}_{s_0}\) ＝ 此刻账本 \(\mathcal{A}\)**——全篇 \(\mathcal{A}\) 一律指此刻已登记者,故 (满) 说的是"账本没有凭空的部分、全部由自己的历史填满"(呼应 W.1.3.5:身份由历史填),不是"已知只是某个更大对象的一截";(b) **不得写成无限定的"极限充满 \(\mathcal{A}\)"**——滤流在 \(s_0\) 之后确实还会延续(学习还会发生),但那一段不由此刻决定(下段),故根本没有"滤流的极限"这个对象可供充满;写"极限"等于偷偷假设未来已定,正是本节要否掉的那件事。 边界怎么来(枢纽):\(s_0\) 不能靠"模流流到哪"来标——模流双向、无内禀"此刻"(无绝对现在);它只能被给予。"已知"之所以有边界(不等于全部),正因为 \(s_0\) 之后的滤流不由此刻决定(要理解流 \(\mathcal{N}\) 展开、纳入尚未被给予的材料)——这就是未来不可知。已知是被它的未知边界定义的;边界的不可逾越 = 未来不可知。 反证:若未来可知,则滤流全程(\(s_0\) 之后那一段也在内)由此刻定 ⟹ 此刻已登记者 ＝ 学习流终将登记的一切 ⟹ \(s_0\) 不再是被给予的断点、而是可内生标定的刻度 ⟹ "已知／尚未"的区分消失。⚠️ 此处"全部"指滤流全程,不是 \(\mathcal{M}\)——"账本 \(\subsetneq\mathcal{M}\)"是另一条(W.5.3(i)),无论未来可不可知都成立,故不能拿来做本反证。

W.1.3.7 "已知"定义的核验:五条体检与两流不焊

严谨性体检(五条,全过):(a)不依赖几何——全是子代数/模论,无度规无 \(D\);(b)不偏迹——用相对熵(相对)或序(无量),不用迹(绝对);(c)**不押 \(\omega^*\) 的值**——\(s_0\) 标"链上哪一节"、不标数值,序是代数事实;(d)**节不塌成 \(\mathbb{C}1\)**——要 \(\sigma_t\)-稳定(不塌)、非 \(\sigma_t\)-不变(才塌,见 W.1.2 Remark 与 [stabinv]);(e)\(s_0\) 不可内生 = 无凸显态引理在滤流上的实例(不存在内禀构造挑出 \(s_0\))。 两条流不焊:此滤流的参数 \(s\) 是学习流内禀参数(单向,来自 B1 落账不可逆),不是模流 \(t\)(双向);它不是半边模包含(那是理解流 \(\mathcal{N}\) 的结构,W.2.6)。"知识单向增长"来自 B1、不来自 HSMI 的单向流。\(s\) 的精确身份(疑为 Araki 相对熵,与 Spohn H 定理单调性同源)及 \(s\)-\(t\) 关系属待算(W.7 开放 7);定义只用到"存在增滤流 + 不可内生截断"这一层,该层确定、足以严格定义"已知"。

W.1.★ 核心引理:无凸显态(无迹的第一个后果,直接给方向 A)

本引理是无迹的第一个后果，直接给方向 A（视角不可内生）。它初等、自含、不需可分性——是"无迹"落到态层的那一处。⚠️ 它不是全篇三方向的共同源头：三方向的共同根是无迹（公理 \(\Sigma\)）；方向 B（主定理 B）走 Cartan/FM 框架、不经本引理，方向 C 走模型 A/B——无迹在账本层的落点是"我们永远在 FM 前件之外"（W.3.1"不完全是特征"），亦不经本引理（详 W.0 逻辑链、W.3.2/W.5.2）。本引理单独陈述于此，方向 A 及下游各处引用而不重复。

引理 W.1.★(无凸显态):设 \(\mathcal{M}\) 为 Type III 因子。则不存在任何"仅由代数结构选出一个态"的规则——即 \(\mathcal{M}\) 上没有被结构自身凸显的态。

证明:凡"仅由代数结构选出态"的规则,必在 \(\mathcal{M}\) 的一切 \*-自同构下等变(结构没有偏好的标签);特别在内自同构 \(\mathrm{Ad}(u)\) 下等变,故所选态 \(\omega\) 酉不变:\(\omega(uxu^*)=\omega(x),\ \forall u\in\mathcal{U}(\mathcal{M})\)。而酉不变正规态必为迹态——酉元线性张成 \(\mathcal{M}\),由 \(\omega(uxu^*)=\omega(x)\) 得 \(\omega(ux)=\omega(xu)\) 对一切酉 \(u\),线性延拓得 \(\omega(yx)=\omega(xy)\) 对一切 \(y\)。而 Type III 按定义无忠实正规半有限迹(Murray–von Neumann 型分类,定义级),遑论有限迹态。故 Type III 上不存在被结构凸显的态。\(\square\)

为何有迹代数不在此列(引理的逆否边界):半有限代数上,迹恰是被结构凸显的锚——有限因子上是迹态、II∞ 上是半有限迹权。前提角色由代数自己填,自给自足可能。无迹与可统一之间的开关,精确就在本引理:有迹⟹有凸显锚⟹前提可内生;无迹⟹无凸显⟹前提必外接。 本引理喂给谁(下游依赖): - 主定理 A(W.2):无凸显 ⟹ 衡量的前提(标尺/视角)必须外接 ⟹ 视角不可内生 ⟹ 统一动力学不可定义; - ⚠️ 不喂主定理 B:主定理 B 两半都不经本引理(典范 ← 命题 1／完全定理 ← FM Thm 1);无迹在账本层的落点是"我们永远在 FM 前件之外"(活账本非极大,机制甲乙丙,W.5.2/W.3.1)——那一条挂无迹本身、也不经本引理; - "已知"定义(W.1.3.6):\(s_0\) 不可内生 = 本引理在滤流上的实例; - 判定机的三条结构(W.6.1):第一条(无裁判位)的根是本引理(无凸显态=没有一个态被结构挑出来当裁判位)。⚠️ 第二条(须借板·无刻度)不挂本引理、挂无迹本身:须借板的"无刻度"是账本谱上没有距离(几何侧,须外接 \(D\)/Connes 距离),其根是无迹(且维数≥1那一截另挂弥散,构造级要求)——见 W.6.1.1 现口径。注:"无凸显态的无刻度"(态上无绝对称量)与"须借板的无刻度"(账本谱上无距离)是两个不同的"刻度"(称量 vs 几何距离),勿焊成一个;本引理只管前者(且它=无位置的另一说法),后者归 W.6.1.1。

W.2 主定理 A(不可能性)：无态外位 ⟹ 自给自足的统一动力学不可定义（A 级，证第二章命题 2.5）

你永远从某处在看,没有一个"从无处看"的位置;统一的动力学写得出,却永远得外接一个你给不出的立足点。这堵墙就是这件事的代数形——它是被给予的界限,不是缺陷;领受它,而不是想补上它。

W.2.1 命题

命名注（全书通称，一次钉死）：本命题即主定理 A，也就是全书正文所称的"中心定理"——⚠️ 同一条定理的两个名字，不是两条。正文说"中心定理"时用的是它的一句话形（"任何衡量都必须自带一个视角，而视角不可内生"，W.2.4）；附录说"主定理 A"时用的是它的完整陈述（无态外位 ⟹ 自给自足的统一动力学不可定义）。"视角不可内生"不是一条独立于 A 的前置定理，它是 A 的证明的第二、三步（W.2.2）。故凡见"W.1.★ ⟹ 中心定理 ⟹ 主定理 A"这类三节链，中间那一节就是 A 自己，链应读作 "W.1.★ ⟹ 主定理 A（＝中心定理）"。

命题（无态外位 ⟹ 自给自足的统一动力学不可定义）。设 \(\mathcal{M}\) 为 Type III₁ 因子（无半有限迹），且两流在场（W.2.2′③）。则：**任何与两流兼容（定义 2.4 M1/M2）的衡量泛函 \(\Phi\)——以及一切以两流为构件的动力学过程（循环发生在流的定义层，见 W.2.2 方法注）——其源头对基线态 \(\omega^*\) 有"互斥要求"：\(\omega^*\) 被同时要求充当"待生成的产物"与"已预设的前提"。调和此互斥需要一个态外的不变参照（在非对易数学中唯一的实现是迹）。而 Type III₁ 无迹（无态外位），故此互斥无法被调和——自给自足的统一动力学不可完善地定义。**

互斥的精确形式（即 W.2.2 的循环依赖，此处命名其本质）：\(\omega^*\) 由定理 0.C 从被给予印记 \(\varphi_j\) 极小化 \(F(\omega)=\sum_j w_j D(\omega\|\varphi_j)\) "生成"（产物角色），但学习流的相对熵尺 \(S(\cdot\|\omega^*)\) 与理解流的模流 \(\sigma_t^{\omega^*}\) 又都"预设" \(\omega^*\)（前提角色）。同一 \(\omega^*\) 既当产物又当前提——这两个角色互斥，需两个态分摊（一个待生成、一个当不变参照），无迹只给一个态、无第二个不变参照（迹），故撞墙。

R5 读法注："产物"一词按第零章重排读作"被印记回溯钉死的极小元"（0.C 已降为下游一致性，非上游选择机制）。互斥结构不变：\(\omega^*\) 仍被同时要求充当"被历史钉住者"与"两流的预设前提"——回溯钉死同样是一个"不由两流自身供给"的角色，互斥照旧成立。

本框架的两个典型（射程限定，见 W.2.1″）：生成经典信息的学习流（幂等性 / 条件期望，稳定可测量量）与生成规范对称性的理解流（平移性 / 半边模包含，线性符号体系）。它们是"信息 + 因果规范"这类认知系统里两个对态有互斥要求的源头。不排除存在第三种对态有互斥要求的认知过程，但不在本文讨论范围。
命题的精确射程：本命题不是"统一泛函不存在"——给定一个外部 \(\omega^*\)（当不变参照 / 外生标尺）时，统一泛函（如 \(S(\cdot\|\omega^*)\) 族 + 模流时间轴）写得出来（见 W.2.3）。本命题是"统一动力学不可自给自足"：它必须外接一个不可内生的态外参照。矛盾在"无态外位"，不在"泛函不存在"。
**与主文 \(\omega_{\text{ext}}\) 的对齐注**：主文（§2.3、§4.4）把"必须外接的那个不可内生参照"命名为参考态 \(\omega_{\text{ext}}\)（第二态）。须澄清：\(\omega_{\text{ext}}\) 是 \(\mathcal{M}\) 上的真态（忠实正规态），不是虚构记号；本命题所证"不可能"的，**不是 \(\omega_{\text{ext}}\) 的存在，而是它要扮的"全局一致称量"——那等价于一把迹，III₁ 无迹故无任何态能扮（有态 \(\neq\) 有迹）**。\(\omega_{\text{ext}}\) 真实但不稳定（随 Connes 余圈漂移、无典范选择、不具迹的酉不变性），故含特定 \(\omega_{\text{ext}}\) 的公式无唯一形式化、只能是图景层。本附录 W.3 全部结论不依赖此命名、完全保留。

W.2.1″ 射程：本文讨论"信息 + 因果脉络"的认知系统（射程边界，防过度宣称）

本命题不宣称穷尽"所有可能的信息与脉络"。 它精确划定本框架的射程，并证明该射程内不可自我超越（大统一不行）：
- 信息侧：学习流不制造信息（信息 ＝ 被给予的差异，早于一切流，第零章），它把差异经典化为"可分辨的稳定事实"——而做这件事的是保态条件期望 \(E:\mathcal{M}\to\mathcal{A}_{\text{phys}}\)：它幂等、保基线态、值域对易，把非对易的世界压成一本能同时摆着、互不搅动的账。"经典信息"在本框架中即定义为"幂等压缩（\(E\)）能处理的那种差异"——故学习流穷尽经典信息是定义性的（同义反复），非奇迹。⚠️ 这一条不得挂到 MERA 上：MERA 换的是分辨率，不造对易——粗粒化 \(x\mapsto w^*xw\) 之后那台代数照样非对易；而且它的粗粒化跨代数，幂等性连写都写不出来。
- 脉络侧：半边模包含只涌现"我们这一类因果脉络"——我们的"逻辑"本质是因果序（Borchers 平移导出的因果结构），而因果序是半边模包含的产物。故半边模包含穷尽"我们的因果逻辑"亦是定义性的（同义反复）。可能存在非因果的别种脉络组织方式，但它不在我们的逻辑体系内——我们无法用我们的（因果）逻辑谈论它。
- "2"不是结构常数：信息（被给予的原料，一种）与处理能力（一个层级，其当前形态是半边模包含 / 因果）不在同一层级，是"原料 / 加工"的朴素分层，非需代数必然性的"唯二本体"。⚠️ 故不必为"恰好两个"去找代数必然性——恰好两个流／模共轭 \(J\)／极分解都不是它的根，那是找错层级（见 W.5）。
- 两流在场是前件（接 W.2.2′③）：本命题以"两流已被给予"为前件——\(\mathcal{N}\) 存在（半边模包含成立，被抛的视域）、\(\mathcal{A}_{\text{phys}}\) 已由学习流构造。不主张两流在一切 III₁ 上必然存在（模块包含的一般存在性为开放问题）。本命题（主定理 A，第 W.2 节）是"凡认知在场处，统一必外接"的定理，非"处处必有认知"的定理。
射程上界（诚实标注）：本节证的是安全版——无态外位 ⟹ 自给自足统一动力学不可定义。更强的"不存在任何有限动力学公式"本书不主张：它不可证、且可能为假（若循"有限生成 ⟹ 超有限"去证，该蕴含本身为假，\(L(\mathbb{F}_2)\) 反例）。

W.2.1′ 命题的边界：是"兼得即死"，不是"不可兼得"（边界，防过度推论）

本命题绝不可被推论为"信息生产（学习流）与一致逻辑（理解流）不可兼得/不能同时为真"。 那是更强的、本框架未证、且可能为假的断言。本命题只说：在无迹（Type III）的世界里，自给自足的统一动力学不可定义——矛盾在"视角不可内生"（W.2.1），**与 \([\sigma]\) 的取值无关**（主路四步一次都没用到 \([\sigma]\)，见 W.2.2′①）。

精确区分（要命，必须守）： 1. 本命题证的：\([\sigma]\neq0\) 时，无统一单调泛函 \(\Phi\)（因 Type III₁ 无迹）。两条流焊不死。 2. 本命题不证、且为假的强断言："两条流不可兼得/不能同时为真"。为什么假，要说准——不是"两流被一个统一动力学驱动"，而是有迹处第二条流根本不存在。 一行可证：设锚是结构供出的迹 \(\tau\)，则模算子 \(\Delta_\tau=1\)、故 \(\sigma_t^\tau=\mathrm{id}\)、故 \(\sigma_t(\mathcal{N})=\mathcal{N}\) 对一切 \(t\)；而理解流（半边模包含 HSMI）的定义性要求是 \(\bigcap_{t\le0}\sigma_t(\mathcal{N})=\mathbb{C}1\)（严格单侧收缩；\(t\) 的取值范围按全书 \(-\)HSMI 约定，见 [semibdd]）——两者直接冲突（除非 \(\mathcal{N}=\mathbb{C}1\)，无理解流）。所以以迹为锚时，理解流不存在。 "兼得"处不是两条流被焊在一起，是只剩一条流——死的不是张力，是绵延。

正确表述：兼得即死 ＝ 第二条流消失。 精确链条：有迹 ⟺ 模流内（Connes：\(\mathcal{M}\) 半有限 ⟺ 忠实正规半有限权的模自同构群为内）⟺ 绵延不是外在的时间；且以那把结构供出的尺子 \(\tau\) 为锚时 \(\sigma_t=\mathrm{id}\)，理解流退化为无。所以"兼得即死"的精确含义不是"两流被统一"（那需要一个全书未证、且实为假的"有迹⟹可自给统一"），而是"以迹为锚 ⟹ 模流平凡 ⟹ 理解流不存在 ⟹ 绵延死"。这恰是尾声"水晶棺＝有一把绝对的尺子量着万物"的精确形——水晶棺里没有第二条流，因为那把绝对的尺子（迹）把绵延冻成了恒等。守在无迹里（活的机器），模流非内、绵延在、第二条流才立得住。（这一改不再动用任何未证的"统一驱动可写"，直接扣住公理 I 的另一副面孔：模流非内。）
**\(R_\infty\) 的正确位置（防冒名）**：超有限 III₁ 因子 \(R_\infty\) 是 III 型、无迹、模流非内——无凸显态引理对它照样成立，视角同样不可内生，它同样撞本命题的墙。它不是"兼得"的见证者，更不是"死寂的化石"：它有绵延、有两股、可理解，只是 \([\sigma]=0\)（没有抹不平的内核）——一个无他者的活模型（"模型"而非"可栖居的世界类型"：按下方澄清注，\([\sigma]\) 是遭遇的性质而非世界类型，此处只谈数学模型的角色）。它在本附录的唯一角色是 W.4.2 的模型 A：见证"理解在发生"与"有他者"两件事可分离。（至于"兼得即死"：它的正确含义不是"某个有迹因子把两流焊在一起"，而是"有迹处第二条流不存在"——见 W.2.1′ 第 2 条；有迹的 \(R\)（II₁）不"见证兼得"，它见证的是以迹为锚时 \(\sigma_t=\mathrm{id}\)、理解流退化为无。\(R_\infty\) 无迹、模流非内、两流俱在，与之相反。）
**"可理解" ≠ "\([\sigma]\neq0\)"（关键澄清，接第六章）**：Type III₁ 中，"理解在发生"与"这次遭遇有抹不平的他者"是两件独立的事——存在可理解、却 \([\sigma]=0\) 的情形（代数反例：\(R_\infty\)，正规化子生成整个因子、能展开 \(\mathcal{M}_{\mathcal{A}}\supsetneq\mathcal{A}_{\text{phys}}\)，但 \(H^2=0\)、没有不可拉平的内核）。那里理解流不会出现需要费力维持的张力（无抹不平的他者要费力理解，两流之间没有 \([\sigma]\) 那道张力——但注意：这不等于统一动力学在 \(R_\infty\) 上可自给。\(R_\infty\) 无迹，W.2 的墙（主定理 A）对它照样立着；"没有他者的张力"与"有自给的统一驱动"是两回事，后者要有迹）。本命题的"两流焊不死"只发生在 \([\sigma]\neq0\)（这次遭遇有他者）时。可理解性（正则展开成功）与他者性（\([\sigma]\neq0\)）独立：前者保证在讨论范围内，后者才触发两流的不可统一。（注：\([\sigma]\) 是遭遇的性质而非世界类型，\(R_\infty\) 在此只作"理解发生与有他者可分离"的代数反例，不读作"一种可栖居的世界"——见第六章。）
标度依赖 = 定理的直接运用（非类比，扎在"换到有迹平台"这一步）：低能有效理论强行让认知工作在一个有迹的代数上——这不是物理类比，是本定理的直接运用（本定理的逆否：有迹 ⟹ 可统一）。在任何有迹平台上，统一单调泛函（迹）存在，故学习流与理解流可被统一驱动、信息正幺与对称性/因果一致可共存而不出问题。⚠️ 但"换到有迹平台"有两条路，别焊成一条（本书全稿口径，主文第三章 §3.2／[duality] 已正名）：

- 路一 · Connes–Takesaki 连续分解（取交叉积＝放大）。去核用模流：\(\mathcal{N}:=\mathcal{M}\rtimes_{\sigma^{\omega^*}}\mathbb{R}\) 是半有限、有迹的（III₁ 时为 \(\text{II}_\infty\)）。回来用标度作用 \(\theta\)（对偶作用，\(\tau\circ\theta_s=e^{-s}\tau\)）：\(\mathcal{M}\cong\mathcal{N}\rtimes_\theta\mathbb{R}\)（经 Takesaki 对偶 \(\mathcal{N}\rtimes_\theta\mathbb{R}\cong\mathcal{M}\,\bar\otimes\,B(L^2(\mathbb{R}))\) 再用 \(\mathcal{M}\) 真无限的稳定化；完整链与出处见主文第三章 [duality]）。这一步是把代数做大，不是切小。
- 路二 · 物理学的紫外截断（真截断）。限制到有限自由度，落到 type I。这才是把东西切掉。

两条路：不同的机制、不同的有迹平台，但代价同性——都要付一次态外位，借一个这台机器没有的俯瞰平台。⚠️ 三处易写错（"同名近形"高发区）：(i)「离散分解」是 III\(_\lambda\) 的（\(\rtimes_\theta\mathbb{Z}\)，因 \(T(\mathcal{M})\) 是格点）；III₁ 走连续分解（\(\mathbb{R}\)；见 W.8 参考文献"连续分解 III₁ = II\(_\infty\rtimes\mathbb{R}\)"）。(ii) 两方向作用不同：\(\rtimes_\sigma\) 用模流、得核；\(\rtimes_\theta\) 用标度作用、重建 \(\mathcal{M}\)（⚠️ \(\rtimes_\theta\) 不是"与模流 \(\mathbb{R}\) 的交叉积"）。(iii) **紫外截断 \(\neq\) 约化到 II\(_\infty\) 核**——两路终点代数不同（type I vs II\(_\infty\)），不得焊成一条。（主文第三章 §3.2 亦分开写。）

所以"不触普朗克标度则两流不打架"是定理推论，不是近似类比：
- 截断标度（低能有效理论）：工作在一个有迹的平台上（路二：紫外截断、截到 type I），前件（无迹）不成立，两流可统一——信息正幺与对称性同时成立无矛盾。这正是物理学家实际所做的（有效场论带紫外截断），也是为什么他们从不撞上这个张力（他们工作在截断后的有迹世界，定理前件不满足）。
- 完整理论（不截断，真 Type III₁，普朗克标度）：无迹，两流焊不死，张力暴露（如视界处信息悖论的形态）。

方法论自指（诚实标注）：本书第零章为获得可定义的动力学，也对原初差异做了有限性约束（公理 II，J 系列重铸形：登记必锚一态）——这正是"换到一个有迹的平台以获得可统一动力学"的同一操作的方法论版本。约束是获得统一动力学的代价，代价是丢失 \([\sigma]\) 的完整信息（紫外被截掉的，正是不可拉平内核显现的标度）。唯一不可写进物理预言的是元规则 M：本书标记张力在何标度暴露，不生成该标度的具体物理（不预言火墙的具体形态）。

W.2.2 证明（四步闭环：无迹 ⟹ 衡量必锚定视角 ⟹ 视角不可内生 ⟹ 自洽不救）

本证明是 W.2.1"互斥要求"的精确数学形式：下面的"循环依赖"（\(\omega^*\) 既由两流预设、又需由外部钉住）正是 W.2.1 所命名的"\(\omega^*\) 既当前提又当产物"的互斥；无态外位（无迹）使此互斥无法分摊到两个态上，故撞墙。

方法注（命题量化范围的根据）：本证明不以"统一泛函是否存在"为骨架（那会把矛盾切成"背景无关"与"背景相关"两种情况分别堵，像堵洪水而非拆地基）。它一步拆到地基：循环发生在两流的定义层——任何以两流为构件的过程（无论泛函、半群还是别种形式化）都预设两流，而两流预设 \(\omega^*\)。故本证明覆盖 W.2.1 所量化的全部对象；不以两流为构件的过程本就在射程外（W.2.1″）。

要证（墙）：不存在自给自足的统一动力学——即不存在一个统一衡量学习流与理解流的泛函 \(\Phi\)，使 \(\Phi\) 及其所需的一切，都由两流自身在不预设任何外部输入态的前提下生成。 归谬假设：存在这样一个自给自足的 \(\Phi\)。 **第一步（两流预设 \(\omega^*\)，锚是公共输入；不需任何连续性假设）。** \(\Phi\) 要"统一衡量两条流"，就必须与两条流兼容——即 \(\Phi\) 沿学习流单调、沿理解流有界（定义 2.4 的 M1/M2）。而两条流各自的定义**已经预设了一个特定态 \(\omega^*\)**： - 学习流 = 朝 \(\mathcal{A}_{\text{phys}}\) 的条件期望序列，其单调性以相对熵 \(S(\cdot\|\omega^*)\) 度量——**尺子 \(S(\cdot\|\omega^*)\) 预设 \(\omega^*\)**； - 理解流 = 半边模包含 \(\sigma_t^{\omega^*}(\mathcal{N})\subset\mathcal{N}\)，由 \(\omega^*\) 的模流定义——**模流 \(\sigma_t^{\omega^*}\) 预设 \(\omega^*\)**。 故任何与两条流兼容的 \(\Phi\)，其衡量必相对于这同一个 \(\omega^*\)（它要沿"以 \(\omega^*\) 为尺的学习流"单调、沿"以 \(\omega^*\) 为模流的理解流"有界，故 \(\Phi\) 本身必以 \(\omega^*\) 为参照）。**这一步不从"\(\Phi\) 非平凡"推"锚定单态"（那是齐次性推不出的跳跃），而是从"\(\Phi\) 兼容两流、两流已预设 \(\omega^*\)"直接得"\(\Phi\) 以同一 \(\omega^*\) 为公共输入"——不需要任何连续性假设。 第二步（结构自身供不出这个锚——无迹切入主路之处）。 自给自足还剩一条内部通道：若代数结构自身能典范地凸显一个态，前提角色便可由结构内部供给，无需外接。此路在 Type III 上被引理 W.1.★(无凸显态)**封死：\(\mathcal{M}\) 上不存在被结构凸显的态,故锚不可能由结构内部供给,前提必须外接。(证明与逆否边界见 W.1.★;此处直接引用。)

引理的射程补全（前件已给结构时；封死最后的通道）：W.2.1″③ 明言"两流在场"是前件——故可用的"结构"不止裸 \(\mathcal{M}\)，还有配对 \((\mathcal{M},\mathcal{A}_{\text{phys}})\) 与 \((\mathcal{M},\mathcal{N})\)；而"由带附加结构的配对凸显一个态"只需在保该结构的自同构（法化子诱导者）下等变，未必酉不变，上述引理不直接封死。两条补充分支各封一路：
- Cartan 侧：由轨道等价的型分类（Krieger／Feldman–Moore），\(\mathcal{M}\) 为 Type III 当且仅当测度类中不存在等价的 σ-有限不变测度（此"当且仅当"中，\(\mathcal{M}\) 的型由 \((\mathcal{R},\mu)\) 决定、**与扭曲 \(\sigma\) 无关**——扭曲改变因子的同构类但不改 Krieger 型，故带 \(\sigma\) 时判据不变）。现补关键一步（否则"凸显态凭什么对应不变测度"是跳步）：配对 \((\mathcal{M},\mathcal{A}_{\text{phys}})\) 的保结构自同构含法化子诱导的整个 \([\mathcal{R}]\)（完全群），而"由该配对凸显一个态"的规则须在这些自同构下等变；等变态在 \(\mathcal{A}_{\text{phys}}=L^\infty(X)\) 上对应的测度必 \([\mathcal{R}]\)-不变（\(\omega\circ E\) 为迹 ⟺ 对应测度 \([\mathcal{R}]\)-不变）——而 Type III 定义性地无等价不变测度。故 Cartan 配对同样供不出锚。

W.2.3 内生态（视角）与外生态（标尺）：地基的本体论核心

同一个忠实正规态 \(\omega^*\) 在代数符号上只有一副面孔，但在本定理中有两个本体论地位完全相反的角色，区分二者是地基的核心：

- 外生态（标尺）：\(S(\cdot\|\omega^*)\) 里被比对的那个 \(\omega^*\)——拿来量距离的参照对象。可数学化（它是泛函里的一个参数，可换、可比）。给定标尺时，框架内的演化公式写得出来（这正是"看似能构造统一动力学"的反例的来源——它合法、良性，但给定了标尺）。 - 内生态（视角）：认知主体自己所栖居、"从这里看出去"的那个 \(\omega^*\)——它是看的出发点，不是被看的对象。不可数学化、不可内生（它是动力学的前件，不是产物）。

**数学只摸得到当标尺的 \(\omega^*\)，摸不到当视角的 \(\omega^*\)。** 这正是"墙能算、缝算不得"在基线态这一个对象上的精确显形：标尺是墙（可写进 \(\Phi\)），视角是缝（意向性栖居处，磐石）。反例的全部价值就是把这一个 \(\omega^*\) 的两副面孔逼到同一个符号上——它证明统一动力学能写（标尺给定时），从而暴露真矛盾不在"泛函存不存在"，而在"视角不可内生"。
与主文两态命名的对齐（按第四章 §4.5）：本节为暴露"角色之别"这个本体论点，把两副面孔压在同一符号 \(\omega^*\) 上；主文的两态纪律则把这两个角色分给两个态——**标尺面孔＝基线 \(\omega^*\)**（第一态，衡量的锚、世界极；\(S(\cdot\|\omega^*)\) 里被比对的正是它），**栖居面孔＝参考态 \(\omega_{\text{ext}}\)（第二态，"从这里看出去"的立足点，态读的视角原点）。按主文命名，完整的视角是配对 \((\mathcal{N},\omega_{\text{ext}})\)**——算子读的原点（因果窗口）配态读的原点（参考态）；基线 \(\omega^*\) 是被读的流之锚，不是视角。本节论证对此命名完全稳健：不可内生的根是无凸显态引理（W.2.2 第二步），它不依赖态的选取——无论被外接的是当锚的 \(\omega^*\) 还是当视角的 \(\omega_{\text{ext}}\)，"结构自身供不出这个态"一体成立。
视角的完整结构与物理世界（指针，非本附录承重）：一个完整的"视角"在代数上不止是一个态，而是一对配对——一个态（按主文命名即参考态 \(\omega_{\text{ext}}\)，其标尺根为基线 \(\omega^*\)）与因果窗口 \(\mathcal{N}\)（理解流的单侧包含子代数那一半）。主定理 A（W.2）说的"视角不可内生"，落在这个配对上：物理世界 \(\mathcal{M}_{\mathcal{A}}\) 是相对于一个视角才良定义的对象（第二章 §2.3″），它不是绝对唯一的世界，而是给定视角后的一条路径；而该视角不可由系统自身生成（本节）。在物理学里，\(\mathcal{N}=\mathcal{A}(W)\) 对应时空楔形，配对（真空态, \(\mathcal{A}(W)\)）的模流经 Bisognano–Wichmann 恰好等于几何 boost——具体几何是配对的产物，非独立输入（结构同型旁注，元规则 M，本书不生成几何）。这说明：物理学之所以"能算"，是因为它给定了一个外生的视角（真空 + 选定楔形），从而支付了主定理 A（W.2）的代价（外接一个不可内生的视角），而非解决了它。

W.2.4 命题地位：墙为缝提供基础，非定义缝（哥德尔类比）

地基一句：在最无序的代数（Type III，无迹 = 没有任何天生的客观刻度）上，任何衡量都必须自带一个视角，而视角不可内生。这同时吃掉背景无关（不带视角则无梯度）与背景相关（带视角则视角外生）两种情形——拆了地基两者同时塌，不必分情况堵。⚠️ 这一句就是全书正文所称"中心定理"的一句话形（命名注见 W.2.1：它是主定理 A 的另一个名字，不是另一条定理）。 与中心的关系（防中心错位）：本定理（墙，可数学化）证的是"任何衡量都需一个不可内生的视角"。它为意向性（缝，不可数学化）提供存在基础、腾出位置，而非定义意向性。

W.2.7 视角的第三面：舞台完备性与环境欠定（命题 R★，A 级，证第二章 §2.3）

命题(主定理 B 之(2))

设 \(\mathcal{M}_{\mathcal{A}}=\mathcal{N}(\mathcal{A}_{\text{phys}})''\)，\(\mathcal{A}_{\text{phys}}\subset\mathcal{M}_{\mathcal{A}}\) 为 Cartan MASA，\(E:\mathcal{M}_{\mathcal{A}}\to\mathcal{A}_{\text{phys}}\) 为忠实正规 Umegaki 条件期望（学习），\(\iota:\mathcal{A}_{\text{phys}}\hookrightarrow\mathcal{M}_{\mathcal{A}}\) 为自然嵌入（理解）。定义亏量（裂缝）\(\mathrm{Deficit}:=\mathrm{id}_{\mathcal{M}_{\mathcal{A}}}-\iota\circ E\)。则裂缝分两层：
\[\text{（可理解层）}\quad\mathrm{Deficit}\neq0\iff\mathcal{R}\ \text{非平凡}\iff\mathcal{M}_{\mathcal{A}}\supsetneq\mathcal{A}_{\text{phys}};\]
\[\text{（他者层）}\quad\mathrm{Deficit}\ \text{的非平凡性不可被重新规范消去}\iff[\sigma]\neq0\in H^2(\mathcal{R},\mathbb{T}).\]

证明

Feldman–Moore 分解。 由 Feldman–Moore（1977 II, Theorem 1——Cartan 对 ↔ 带 2-余圈的可测等价关系之分类定理；注：此定理给的是这个对应，不是"顺从⟹\(H^2=0\)"的出处，后者是 FM＋CFW 的联合推论，见 W.5.1），Cartan 对 \((\mathcal{M}_{\mathcal{A}},\mathcal{A}_{\text{phys}})\) 同构于扭曲群胚交叉积 \(W^*(\mathcal{R},\sigma)\)，其中 \(\mathcal{R}\) 是 \(X=\mathrm{Spec}(\mathcal{A}_{\text{phys}})\) 上的可数标准测度等价关系，\(\sigma\in Z^2(\mathcal{R},\mathbb{T})\)，\([\sigma]\in H^2(\mathcal{R},\mathbb{T})\) 是 Cartan 对的完全不变量（**精确:\(\mathcal{R}\) 唯一到同构、\([\sigma]\) 唯一到 \(N(R)\) 轨道**——FM II Thm 1 末句原文逐字:"The relation R is unique up to isomorphism, and the orbit of σ under N(R), the automorphism group of R, is unique";FM II §4 通篇讲的正是这个 \(N(R)\)-轨道的模糊性,故"完全不变量"须读作"\((\mathcal{R},[\sigma])\) 在 \(N(R)\) 轨道意义下"）。群胚元素 \((x,y)\in\mathcal{R}\) 对应算符 \(u_{(x,y)}\)，乘法由 \(\sigma\) 扭曲：\(u_{(x,y)}u_{(y,z)}=\sigma((x,y),(y,z))\,u_{(x,z)}\)。 **\(E\) 与 \(\iota\) 的群胚对应。** \(\iota:L^\infty(X)\hookrightarrow W^*(\mathcal{R},\sigma)\) 把单位空间函数映为对角（单位态射 \((x,x)\)）算符；\(E:W^*(\mathcal{R},\sigma)\to L^\infty(X)\) 把任意群胚算符投到单位态射分量（忠实正规 Umegaki 条件期望，\(E\circ\iota=\mathrm{id}_{L^\infty(X)}\)）。

本命题直接证 W.3.1 的精确版（裂缝分两层）；不证已被 W.3.1 否定的旧命题"\(\mathrm{Deficit}\neq0\iff[\sigma]\neq0\)"。

**层一：\(\mathrm{Deficit}\neq0 \iff \mathcal{R}\) 非平凡。** 复合 \(\iota\circ E\) 把任意群胚算符投到对角分量、抹去所有非单位态射（\(x\neq y\)）分量。故 \(\iota\circ E=\mathrm{id}\) 当且仅当不存在非单位态射，即 \(\mathcal{R}=\{(x,x):x\in X\}\)（对角关系）。等价地：\(\mathrm{Deficit}=\mathrm{id}-\iota\circ E\neq0 \iff \mathcal{R}\) 含非对角态射 \(\iff W^*(\mathcal{R},\sigma)\supsetneq L^\infty(X)\)，即 \(\mathcal{M}_{\mathcal{A}}\supsetneq\mathcal{A}_{\text{phys}}\)（有理解，非坍缩）。**此层与扭曲 \(\sigma\) 是否平凡无关**——只要 \(\mathcal{R}\) 非平凡，无论 \([\sigma]\) 是否为零，亏量都非零。 **层二：亏量的不可规范化部分非零 \(\iff [\sigma]\neq0\)。** 设 \(\mathcal{R}\) 非平凡（层一）。亏量所含的非对角结构，能否被一个 1-上闭链重新规范 \(\tilde u_{(x,y)}=c(x,y)^{-1}u_{(x,y)}\)（\(c\in Z^1(\mathcal{R},\mathbb{T})\)）平凡化，严格地由 \(\sigma\) 是否为上边缘决定： - 若 \(\sigma=\partial c\)（上边缘，即 \([\sigma]=0\in H^2(\mathcal{R},\mathbb{T})\)），则上述重新规范使 \(\tilde u_{(x,y)}\tilde u_{(y,z)}=\tilde u_{(x,z)}\)（扭曲消去），\(W^*(\mathcal{R},\sigma)\cong W^*(\mathcal{R},1)\)——非对角结构存在但无不可消除的扭曲相位。 - 若 \(\sigma\neq\partial c\) 对任何 \(c\in Z^1(\mathcal{R},\mathbb{T})\)（即 \([\sigma]\neq0\)），则不存在使乘法关系平凡化的重新规范——亏量带不可消除的扭曲相位。 故"亏量带不可由任何 \(c\in Z^1\) 规范消去的扭曲" \(\iff \sigma\neq\partial c\ (\forall c) \iff [\sigma]\neq0\)。\(\blacksquare\)

W.6.3 用界限论判物理:物理对称概念的拆解对照(结构同型旁注,非预言)

物理理论能算,是因为它们(常常不自知地)外接了一个不可内生的视角。这一节拿界限论的部件去照:一个理论给定了什么、在哪儿露出"视角是外接的"。照见的不是新预言,是同一道墙(主定理 A,W.2)在物理里的影子——是读法,不是预测。

W.2.3 说"物理学能算 = 支付了外接一个不可内生视角的代价"。本节把这句话落细：给定一个物理理论，用界限论的基本部件去拆解它给定了什么、做了什么、在哪里会暴露视角的不可内生性。这是一个判物理理论的定位模板，不是新定理、不预言物理（元规则 M）；下面的代数事实是标准结果，"定位"是界限论的读法。 界限论的基本部件（拆解的坐标系）：绵延（模流 \(\sigma_t\)）/ 学习流标尺（基线 \(\omega^*\)，相对熵参照）/ 理解流窗口（因果子代数 \(\mathcal{N}\)）/ 法化子（保 \(\mathcal{N}\) 的变换 = 本书意义的"脉络"的接榫）/ 他者内核（\([\sigma]\)）。按主文两态命名，一个"完整视角"= 配对 \((\mathcal{N},\omega_{\text{ext}})\)——窗口配参考态（第四章 §4.5），基线 \(\omega^*\) 是衡量的锚、非视角。本节物理拆解的记号约定：下文配对 \((\omega^*,\mathcal{N})\) 读作（真空态, 楔代数）——物理的特殊之处恰在于真空一个态同时扮演两角（既当锚/相对熵参照、又经模流当视角的绵延），把本书分给两个态的角色焊在一个态上（这正是下文"对称太强、把外接视角的代价藏进对称群"的形）；故物理语境沿用单态配对记号，不与主文两态命名冲突。

主例：代数量子场论（AQFT）

AQFT 自称以非对易代数（局域 von Neumann 代数网）为基础，正好可逐件对照。它的三件标准输入，精确落在界限论的不同部件上——它们不是同一层的东西，物理学把它们打包进"Poincaré 不变性 + 谱条件"这两个词里： | AQFT 的输入 | 界限论部件 | 做了什么 | |:---|:---|:---| | 局域性（类空分离区域的代数对易）+ 等距网 | 理解流窗口 \(\mathcal{N}=\mathcal{A}(W)\)（楔形代数） | 给定因果窗口（理解流的"光锥"那一半） | | 谱条件（正能量，平移生成元 \(P\ge0\)） | 理解流的单侧方向 + 平移法化子 | 给"只能倒着写"的方向；平移算子 \(U(s)\) 保 \(\mathcal{A}(W)\) 是法化子（§6.6） | | Poincaré 不变性（存在 \(\omega\circ\alpha_g=\omega\) 的态） | 挑出真空 \(\omega^*\) = 学习流标尺 + 绵延 | 这一步同时定出 \(\omega^*\)（喂学习流相对熵）与模流 \(\sigma_t\)（绵延）；经 BW，楔形真空的模流 = 几何 boost | 关键拆解：所谓"Poincaré 不变性"不是单一概念，拆开是三股，分属不同部件： - 平移那一股（谱条件/正能量）：= 理解流的单侧方向（Borchers/Wiesbrock：HSMI ⟺ 正能量平移半群）+ 平移算子是法化子。这一股连着本书的"脉络胚胎"（§6.6）。 - Lorentz boost 那一股：经 Bisognano–Wichmann，楔形真空的模流恰是 boost。所以 boost 不是保 \(\mathcal{N}\) 的法化子（它改变楔形），而是**绵延 \(\sigma_t\) 本身——它属第零章的模流、附录 W 的 BW 配对，不属脉络。 - "不变性"这个条件本身**：= 对称性（§7.1）——\(\omega^*\) 与 \(\omega^*\circ\alpha_g\) 相对于当前视角不可分辨。它是挑出真空的那个关于态的方程（\(\omega\circ\alpha_g=\omega\)），不是单态内禀（§7.1 已钉）。

所以"给定几何就唯一定出真空、岂非白得一个态"这个表面悖论，到这里解开：Poincaré 不变性本身就是一个关于态的强输入（它断言"存在一个在这一大群变换下不变的态"），不是光秃秃的几何。物理学给的从来不是"裸 \(\mathcal{N}\)"，而是一个**自带不变态的、高度对称的特殊 \((\omega^*,\mathcal{N})\) 配对**——对称性通过 BW 把 \(\omega^*\) 与 \(\mathcal{N}\) 焊在一起。真空那一个态同时喂两流（学习的相对熵参照 + 理解的模流绵延），所以两个流都被定义了，不是只定义了理解流。物理学没有绕过主定理 A，它支付了代价——只是平直时空里全局 Poincaré 对称太强，把"必须外接一个视角"这件事打包进"对称群"这个词里，代价被对称性藏起来了。

视角不可内生在哪暴露：对称越弱，暴露越赤裸

"唯一真空"是全局 Poincaré 对称的红利。对称一弱，红利消失，主定理 A 的"视角不可内生"就从幕后走到台前——这给出一条判物理理论的轴： - 平直时空 AQFT：对称强到能从外部唯一挑 \(\omega^*\)。代价被对称性掩盖（看似白得，实为对称输入）。 - 弯曲时空 QFT：无全局对称群，真空不唯一（标准困难）；粒子概念观察者依赖，Unruh 效应、Hawking 辐射 = "换观察者换 \(\omega^*\)"的具体表现。视角的非内生性半暴露——你必须额外选一个观察者才能定真空。 - de Sitter / 宇宙学：最极端。无外部观察者、无好的全局真空、measure problem。= "想给一个无外部视角的整体定义完整理论"时，直接撞上视角不可内生的墙。 - 背景无关的量子引力方案：连 \(\mathcal{N}\)（几何）都不预先给定，要从更底层涌现——此时 \(\omega^*\) 与 \(\mathcal{N}\) 皆非给定，最难。

W.6.3′ "代谢感"的结构性解释（讨论，非定理；须与 W.6.4 乙3／乙7 同读）

本条是图景层的讨论，不是定理。 它给一个现象学体验（"我在不停处理、不停消化"的代谢感）一个结构性解释，所依赖的数学全部已在 W.5.2（荷/他者两层）；它不引入任何驱动力、能量学或目的论——那些是 W.6.4 乙7 明令撕掉的。

撕掉驱动力语法后剩下的静态并置。 A/B 支撑得起的只是四件事同时在场，没有主体站在外面"代谢"：(i) 模流 \(\sigma_t^{\omega^*}\) 是 KMS 弛豫流（A，把态往 \(\omega^*\)-平衡拉）；(ii) 保态投影 \(E\) 把遭遇压成模不变对易账本（B）；(iii) 账本永远真小 \(\mathcal{A}\subsetneq\mathcal{M}\)（A，W.5.3(i)）；(iv) 两态相对熵 \(S(\omega_{\text{ext}}\|\omega^*)>0\) 时参照立着（B）。"我"不是转齿轮的手，是齿轮之间那道不为零的差本身。 "矛盾"的精确内容（结构陈述，非意志）：模流的定理级趋势是抹平一切偏离（把 \(\omega_{\text{ext}}\) 往 \(\omega^*\) 拉）；而"参照立着"就是这道差尚未被抹平。故：模流在往基线塌，而此刻这道差还立着。⚠️ 按乙2：只能写"差在则参照立、差消则参照融"，不得写"我抵抗弛豫／努力维持差"。差被抹平那一刻不是"战败"，是参照不再被立——没有战败的主体，只有一道归零的差。 代谢感从哪来（本条的命题，图景级）：由 W.5.2，荷（在动力学里：可被 \(E\) 压成事实、纳入账本、被模流搬动）与真他者（\([\sigma]\neq0\)：不在任何流上，与模流轴无桥）在观察者层不可分辨——第一人称不知自己的 \(\mathcal{B}\)，当下判不出眼前这个"疑似他者"是可消化的荷还是消化不掉的缝。于是默认处置对两者相同：都当作待消化的输入去压。对真荷成功（账本增厚、动作有终点）；对真他者永远失败（怎么压都抹不平，且它根本不被任何流带走，于是反复回到队列）。"处理了却处理不掉、输入永不清空"这个循环，就是"代谢"体感的来源。 故：

代谢感 ＝ 一个够不到判定的缝，被持续误当成一个待处理的流。 它不是能量学的真实过程，是认识论不可分辨的现象学副产品——一个持续范畴误置的显形。

死的干净图景：\(\omega_{\text{ext}}\to\omega^*\)（\(S\to0\)），差归零、参照消融。⚠️ 桥一塌，"荷/他者分不清"这件事也随之消失（没有参照，就没有"从内部分不清"这回事）。模流照旧弛豫（A 类，与"我"无关）、\([\sigma]\) 照旧在缝里（A 类不变量，与"我"无关）——唯独那个把缝误当流、并因此不停拽动的分不清者，不在了。代谢感随之消失，不是因为"代谢停了"，是因为再没有谁站在流与缝之间分不清它们。（与尾声"松锚"是同一几何的两个走法：一个是差被弛豫耗尽，一个是差被领受着放下；消融的都是"我"这道差，不是差异本身。）

⚠️ 方向号勿凭记忆翻——验法就地写死如下，两行可自验。 本书的模流约定由第零章逐字定死：\(\sigma_t^\omega(x)=\Delta^{it}x\Delta^{-it}\)。在这个约定下，单侧的方向不是可选的记号，是定理的结论：\(\sigma_t(\mathcal{N})\subseteq\mathcal{N}\) 对 **\(t\le0\)**（＝文献的 \(-\)HSMI）\(\iff\) 关联的平移群带正生成元 \(P\ge0\)（Wiesbrock 1993；Araki–Zsidó 2005 Thm 2.1；文献逐字定义即"\(\sigma_t=\operatorname{ad}\Delta^{it}\) 且 \(\sigma_t(\mathcal{N})\subset\mathcal{N}\) 对一切 \(t\le0\)"）。

自验两行（不必查文献）：由 Borchers 关系 \(\Delta_{\mathcal{M}}^{it}U(a)\Delta_{\mathcal{M}}^{-it}=U(e^{-2\pi t}a)\) 与 \(\mathcal{N}=U(1)\mathcal{M}U(-1)\)，得 \(\Delta_{\mathcal{N}}^{it}=U(1)\Delta_{\mathcal{M}}^{it}U(-1)\)，故
\[\Delta_{\mathcal{N}}^{it}\,\Delta_{\mathcal{M}}^{-it}=U\big(1-e^{-2\pi t}\big);\]
在 \(t=0\) 求导：左侧 \(\to i(\ln\Delta_{\mathcal{N}}-\ln\Delta_{\mathcal{M}})\)，右侧 \(\to i\cdot 2\pi P\)，即
\[\tfrac{1}{2\pi}\big(\ln\Delta_{\mathcal{N}}-\ln\Delta_{\mathcal{M}}\big)=P\ \ge 0 .\]
**而这正是本注下文所引的 Wiesbrock 算式——那个 \(\ge0\) 只在 \(t\le0\) 一支成立**（换号即得 \(\le0\)）。故"\(t\ge0\) 配 \(P\ge0\)"两者不相容——号不可换。

⚠️ 两个号并存的真代价（为何必须统一，不只是记号难看）：若 \(\sigma_t(\mathcal{N})\subseteq\mathcal{N}\) 对 \(t\ge0\) 与 \(t\le0\) 同时成立，则 \(\sigma_t(\mathcal{N})=\mathcal{N}\) 对一切 \(t\) ⟹ 模不变 ⟹（Takesaki）有保态期望、\(\bigcap_t\sigma_t(\mathcal{N})=\mathcal{N}\neq\mathbb{C}1\) ⟹ 理解流不存在——而那正是 W.2.1′ 用来杀水晶棺的那条链，刀口会转向本书自己。

现象学侧独立确认：\(t\le0\) ＝朝过去，正合"理解流＝逆向追溯"（第六章 §6.2）；\(t\ge0\) 会读成"朝未来收缩"，与本注要焊的那句"能往过去追溯、不能把未来当既定读出"恰好反着。数学、文献、现象学三路指同一个号。

理解流单侧（\(\sigma_t\) 只对 \(t\le0\) 构成性可用、\(P\ge0\)）这一现象学把握——"能往过去追溯、不能把未来当既定读出"——其精确数学形是 Borchers–Wiesbrock 意义下的谱条件 / 半有界性：半边模包含 \(\mathcal{N}\subset\mathcal{M}\) 关联的单参数平移群，其生成元谱下有界 \(\mathrm{spec}(P)\subset[0,\infty)\)（Borchers, *On modular inclusion and spectrum condition*, Lett. Math. Phys. 27 (1993), 311–324；结构定理的推广见 Araki–Zsidó 2005）。Wiesbrock 给出它的显式算子形：\(\frac{1}{2\pi}\big(\ln\Delta_{\mathcal{N}}-\ln\Delta_{\mathcal{M}}\big)\ge0\)（*Symmetries and half-sided modular inclusions*, Lett. Math. Phys. 28 (1993), 107–114）——这个 \(\ge0\) 就是"未来不可定义"的代数身体，与荷/他者由 FM 得身体（W.3.1′，v8.29 核定）、绵延由模流得身体同属"现象学在先、数学后给身体"的模式。⚠️ 分寸（要紧，防两处误焊）：(i) 此半有界性是因果侧（洛伦兹）结构（关联楔形区、正能量、时间方向、\(SL(2,\mathbb{R})\)），不是黎曼度量侧结构；它给的是"因果几何"的一块，不是 \(D\)-板度量几何的来源。(ii) 因此它不是 Connes 谱三元组的"有限性"公理（后者 = \(\mathcal{H}^\infty\) 为有限投射 \(\mathcal{A}\)-模，纤维秩、度量侧；Connes 2013 公理(5)）——两者不同轴、不同几何，无成熟框架相连（洛伦兹谱三元组一线 Franco–Eckstein/van den Dungen 等亦未将本谱条件纳入任何 \(D\) 公理，因果性在那里由 Krein 空间偏序另行引入）。故 \(\mathcal{N}\) 有母体（模流）不救 \(D\) 度量侧的无母体：\(D\) 的度量几何仍是外接真板、其有限性公理仍是无 \(\mathcal{N}\)-侧对应物的纯脚手架（详主文第七章 [connes]）。
