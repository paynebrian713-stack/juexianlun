W.2.1′ 命题的边界：是"兼得即死"，不是"不可兼得"（边界，防过度推论）

本命题绝不可被推论为"信息生产（学习流）与一致逻辑（理解流）不可兼得/不能同时为真"。 那是更强的、本框架未证、且可能为假的断言。本命题只说：在无迹（Type III）的世界里，自给自足的统一动力学不可定义——矛盾在"视角不可内生"（W.2.1），**与 \([\sigma]\) 的取值无关**（主路四步一次都没用到 \([\sigma]\)，见 W.2.2′①）。

精确区分（要命，必须守）：

1. 本命题证的：\([\sigma]\neq0\) 时，无统一单调泛函 \(\Phi\)（因 Type III₁ 无迹）。两条流焊不死。 2. 本命题不证、且为假的强断言："两条流不可兼得/不能同时为真"。为什么假，要说准——不是"两流被一个统一动力学驱动"，而是有迹处第二条流根本不存在。 一行可证：设锚是结构供出的迹 \(\tau\)，则模算子 \(\Delta_\tau=1\)、故 \(\sigma_t^\tau=\mathrm{id}\)、故 \(\sigma_t(\mathcal{N})=\mathcal{N}\) 对一切 \(t\)；而理解流（半边模包含 HSMI）的定义性要求是 \(\bigcap_t\sigma_t(\mathcal{N})=\mathbb{C}1\)（严格单侧收缩）——两者直接冲突（除非 \(\mathcal{N}=\mathbb{C}1\)，无理解流）。所以以迹为锚时，理解流不存在。 "兼得"处不是两条流被焊在一起，是只剩一条流——死的不是张力，是绵延。

正确表述：兼得即死 ＝ 第二条流消失。 精确链条：有迹 ⟺ 模流内（Connes：\(\mathcal{M}\) 半有限 ⟺ 忠实正规半有限权的模自同构群为内）⟺ 绵延不是外在的时间；且以那把结构供出的尺子 \(\tau\) 为锚时 \(\sigma_t=\mathrm{id}\)，理解流退化为无。所以"兼得即死"的精确含义不是"两流被统一"（那需要一个全书未证、且实为假的"有迹⟹可自给统一"），而是"以迹为锚 ⟹ 模流平凡 ⟹ 理解流不存在 ⟹ 绵延死"。这恰是尾声"水晶棺＝有一把绝对的尺子量着万物"的精确形——水晶棺里没有第二条流，因为那把绝对的尺子（迹）把绵延冻成了恒等。守在无迹里（活的机器），模流非内、绵延在、第二条流才立得住。（这一改不再动用任何未证的"统一驱动可写"，直接扣住公理 I 的另一副面孔：模流非内。）

**\(R_\infty\) 的正确位置（防冒名）**：超有限 III₁ 因子 \(R_\infty\) 是 III 型、无迹、模流非内——无凸显态引理对它照样成立，视角同样不可内生，它同样撞本命题的墙。它不是"兼得"的见证者，更不是"死寂的化石"：它有绵延、有两股、可理解，只是 \([\sigma]=0\)（没有抹不平的内核）——一个无他者的活模型（"模型"而非"可栖居的世界类型"：按下方澄清注，\([\sigma]\) 是遭遇的性质而非世界类型，此处只谈数学模型的角色）。它在本附录的唯一角色是 W.2.2 的模型 A：见证"理解在发生"与"有他者"两件事可分离。（至于"兼得即死"：它的正确含义不是"某个有迹因子把两流焊在一起"，而是"有迹处第二条流不存在"——见 W.2.1′ 第 2 条；有迹的 \(R\)（II₁）不"见证兼得"，它见证的是以迹为锚时 \(\sigma_t=\mathrm{id}\)、理解流退化为无。\(R_\infty\) 无迹、模流非内、两流俱在，与之相反。）

**"可理解" ≠ "\([\sigma]\neq0\)"（关键澄清，接第六章）**：Type III₁ 中，"理解在发生"与"这次遭遇有抹不平的他者"是两件独立的事——存在可理解、却 \([\sigma]=0\) 的情形（代数反例：\(R_\infty\)，正规化子生成整个因子、能展开 \(\mathcal{M}_{\mathcal{A}}\supsetneq\mathcal{A}_{\text{phys}}\)，但 \(H^2=0\)、没有不可拉平的内核）。那里理解流不会出现需要费力维持的张力（无抹不平的他者要费力理解，两流之间没有 \([\sigma]\) 那道张力——但注意：这不等于统一动力学在 \(R_\infty\) 上可自给。\(R_\infty\) 无迹，W.3 的墙对它照样立着；"没有他者的张力"与"有自给的统一驱动"是两回事，后者要有迹）。本命题的"两流焊不死"只发生在 \([\sigma]\neq0\)（这次遭遇有他者）时。可理解性（正则展开成功）与他者性（\([\sigma]\neq0\)）独立：前者保证在讨论范围内，后者才触发两流的不可统一。（注：\([\sigma]\) 是遭遇的性质而非世界类型，\(R_\infty\) 在此只作"理解发生与有他者可分离"的代数反例，不读作"一种可栖居的世界"——见第六章。）

W.2.2 证明（四步闭环：无迹 ⟹ 衡量必锚定视角 ⟹ 视角不可内生 ⟹ 自洽不救）

本证明是 W.2.1"互斥要求"的精确数学形式：下面的"循环依赖"（\(\omega^*\) 既由两流预设、又需由外部钉住）正是 W.2.1 所命名的"\(\omega^*\) 既当前提又当产物"的互斥；无态外位（无迹）使此互斥无法分摊到两个态上，故撞墙。

方法注（命题量化范围的根据）：本证明不以"统一泛函是否存在"为骨架（那会把矛盾切成"背景无关"与"背景相关"两种情况分别堵，像堵洪水而非拆地基）。它一步拆到地基：循环发生在两流的定义层——任何以两流为构件的过程（无论泛函、半群还是别种形式化）都预设两流，而两流预设 \(\omega^*\)。故本证明覆盖 W.2.1 所量化的全部对象；不以两流为构件的过程本就在射程外（W.2.1″）。

要证（墙）：不存在自给自足的统一动力学——即不存在一个统一衡量学习流与理解流的泛函 \(\Phi\)，使 \(\Phi\) 及其所需的一切，都由两流自身在不预设任何外部输入态的前提下生成。

**第一步（两流预设 \(\omega^*\)，锚是公共输入；不需任何连续性假设）。** \(\Phi\) 要"统一衡量两条流"，就必须与两条流兼容——即 \(\Phi\) 沿学习流单调、沿理解流有界（定义 2.4 的 M1/M2）。而两条流各自的定义**已经预设了一个特定态 \(\omega^*\)**：

- 学习流 = 朝 \(\mathcal{A}_{\text{phys}}\) 的条件期望序列，其单调性以相对熵 \(S(\cdot\|\omega^*)\) 度量——**尺子 \(S(\cdot\|\omega^*)\) 预设 \(\omega^*\)**； - 理解流 = 半边模包含 \(\sigma_t^{\omega^*}(\mathcal{N})\subset\mathcal{N}\)，由 \(\omega^*\) 的模流定义——**模流 \(\sigma_t^{\omega^*}\) 预设 \(\omega^*\)**。

故任何与两条流兼容的 \(\Phi\)，其衡量必相对于这同一个 \(\omega^*\)（它要沿"以 \(\omega^*\) 为尺的学习流"单调、沿"以 \(\omega^*\) 为模流的理解流"有界，故 \(\Phi\) 本身必以 \(\omega^*\) 为参照）。**这一步不从"\(\Phi\) 非平凡"推"锚定单态"（那是齐次性推不出的跳跃），而是从"\(\Phi\) 兼容两流、两流已预设 \(\omega^*\)"直接得"\(\Phi\) 以同一 \(\omega^*\) 为公共输入"——不需要任何连续性假设。**

第二步（结构自身供不出这个锚——无迹切入主路之处）。 自给自足还剩一条内部通道：若代数结构自身能典范地凸显一个态，前提角色便可由结构内部供给，无需外接。此路在 Type III 上被引理 W.1.★(无凸显态)封死：\(\mathcal{M}\) 上不存在被结构凸显的态,故锚不可能由结构内部供给,前提必须外接。(证明与逆否边界见 W.1.★;此处直接引用。)

引理的射程补全（前件已给结构时；封死最后的通道）：W.2.1″③ 明言"两流在场"是前件——故可用的"结构"不止裸 \(\mathcal{M}\)，还有配对 \((\mathcal{M},\mathcal{A}_{\text{phys}})\) 与 \((\mathcal{M},\mathcal{N})\)；而"由带附加结构的配对凸显一个态"只需在保该结构的自同构（法化子诱导者）下等变，未必酉不变，上述引理不直接封死。两条补充分支各封一路：
- Cartan 侧：由轨道等价的型分类（Krieger／Feldman–Moore），\(\mathcal{M}\) 为 Type III 当且仅当测度类中不存在等价的 σ-有限不变测度（此"当且仅当"中，\(\mathcal{M}\) 的型由 \((\mathcal{R},\mu)\) 决定、**与扭曲 \(\sigma\) 无关**——扭曲改变因子的同构类但不改 Krieger 型，故带 \(\sigma\) 时判据不变）。现补关键一步（否则"凸显态凭什么对应不变测度"是跳步）：配对 \((\mathcal{M},\mathcal{A}_{\text{phys}})\) 的保结构自同构含法化子诱导的整个 \([\mathcal{R}]\)（完全群），而"由该配对凸显一个态"的规则须在这些自同构下等变；等变态在 \(\mathcal{A}_{\text{phys}}=L^\infty(X)\) 上对应的测度必 \([\mathcal{R}]\)-不变（\(\omega\circ E\) 为迹 ⟺ 对应测度 \([\mathcal{R}]\)-不变）——而 Type III 定义性地无等价不变测度。故 Cartan 配对同样供不出锚。
- 窗口侧：\(\mathcal{N}\) 由 HSMI 定义，而 HSMI 的定义本身引用 \(\sigma_t^{\omega^*}\)——**\(\mathcal{N}\) 预设 \(\omega^*\)**（正是第一步的定义层循环），它不能反过来供出 \(\omega^*\)，否则循环。
三路合龙（裸 \(\mathcal{M}\)·酉不变／Cartan 配对·无不变测度／窗口配对·预设循环）才真正封死"结构自身供不出锚"，且三路与第一步咬合。

第三步（锚的实际来源外接）。 由第一、二步：锚 \(\omega^*\) 是 \(\Phi\) 与两流的公共输入，且不可由结构供给。它的实际来源是：\(\omega^*\) 被局部印记 \(\varphi_j\) 经 \(F(\omega)=\sum_j w_j D(\omega\|\varphi_j)\) 的极小化回溯钉死（定理 0.C，R5 读法：下游一致性），而 \(\varphi_j\) 是被给予的他者印记（元定理 W：印记选取独立于公理）。要由两流生成 \(\omega^*\)，得先有 \(\omega^*\) 才能定义两流——循环。故 \(\omega^*\) 不可由动力学内生，必外部被给予。

第四步（自洽不动点不救——堵死循环定义的标准出路）。 循环定义在数学中有一条标准出路——自洽方案：用候选态 \(\omega\) 定义两流，令两流运转产出印记/记录，记录经 0.C 型一致性产出 \(\omega'\)，求 \(\omega=\omega'\) 的不动点（自洽场论式）。此路不救"自给自足"，两刀封死：

- (a) 外接只被搬运、未被消除（承重）：任何此类方案的回路数据是印记 \(\varphi_j\)；元定理 W 已证印记的选取独立于公理——被给予。不动点把外接输入从 \(\omega^*\) 挪到 \(\varphi_j\)，"一切由系统自身生成"照样破产。（W.2.4 现象学回灌"最终递归到他者的印记"即此事。） - (b) 不动点的选取无典范（加固）：即便某方案有不动点，一般非唯一；"选哪一个作锚"落在参照的 torsor 上——无典范代表元（A1），无可归一化的自然测度（E5）。以"自洽"之名挑锚，挑的动作本身又是一次不可内生的外接。

故不动点方案不构成反例。

矛盾。 "自给自足"破产：统一衡量两流的 \(\Phi\) 与两流共享一个既不可由结构供给（第二步）、也不可由动力学内生（第三步）、亦不可由自洽回路兜底（第四步）的公共输入 \(\omega^*\)。**统一动力学可以写（给定 \(\omega^*\) 时），但永远不可能自给自足——它必然外接一个不可内生的视角。** \(\blacksquare\)

W.2.3 内生态（视角）与外生态（标尺）：地基的本体论核心

同一个忠实正规态 \(\omega^*\) 在代数符号上只有一副面孔，但在本定理中有两个本体论地位完全相反的角色，区分二者是地基的核心：

- 外生态（标尺）：\(S(\cdot\|\omega^*)\) 里被比对的那个 \(\omega^*\)——拿来量距离的参照对象。可数学化（它是泛函里的一个参数，可换、可比）。给定标尺时，框架内的演化公式写得出来（这正是"看似能构造统一动力学"的反例的来源——它合法、良性，但给定了标尺）。 - 内生态（视角）：认知主体自己所栖居、"从这里看出去"的那个 \(\omega^*\)——它是看的出发点，不是被看的对象。不可数学化、不可内生（它是动力学的前件，不是产物）。

**数学只摸得到当标尺的 \(\omega^*\)，摸不到当视角的 \(\omega^*\)。** 这正是"墙能算、缝算不得"在基线态这一个对象上的精确显形：标尺是墙（可写进 \(\Phi\)），视角是缝（意向性栖居处，磐石）。反例的全部价值就是把这一个 \(\omega^*\) 的两副面孔逼到同一个符号上——它证明统一动力学能写（标尺给定时），从而暴露真矛盾不在"泛函存不存在"，而在"视角不可内生"。

与主文两态命名的对齐（按第四章 §4.4·半）：本节为暴露"角色之别"这个本体论点，把两副面孔压在同一符号 \(\omega^*\) 上；主文的两态纪律则把这两个角色分给两个态——**标尺面孔＝基线 \(\omega^*\)**（第一态，衡量的锚、世界极；\(S(\cdot\|\omega^*)\) 里被比对的正是它），**栖居面孔＝参考态 \(\omega_{\text{ext}}\)（第二态，"从这里看出去"的立足点，态读的视角原点）。按主文命名，完整的视角是配对 \((\mathcal{N},\omega_{\text{ext}})\)**——算子读的原点（因果窗口）配态读的原点（参考态）；基线 \(\omega^*\) 是被读的流之锚，不是视角。本节论证对此命名完全稳健：不可内生的根是无凸显态引理（下文 W.2.2 第二步），它不依赖态的选取——无论被外接的是当锚的 \(\omega^*\) 还是当视角的 \(\omega_{\text{ext}}\)，"结构自身供不出这个态"一体成立。

视角的完整结构与物理世界（指针，非本附录承重）：一个完整的"视角"在代数上不止是一个态，而是一对配对——一个态（按主文命名即参考态 \(\omega_{\text{ext}}\)，其标尺根为基线 \(\omega^*\)）与因果窗口 \(\mathcal{N}\)（理解流的单侧包含子代数那一半）。W.3 说的"视角不可内生"，落在这个配对上：物理世界 \(\mathcal{M}_{\mathcal{A}}\) 是相对于一个视角才良定义的对象（第二章 §2.3″），它不是绝对唯一的世界，而是给定视角后的一条路径；而该视角不可由系统自身生成（本节）。在物理学里，\(\mathcal{N}=\mathcal{A}(W)\) 对应时空楔形，配对（真空态, \(\mathcal{A}(W)\)）的模流经 Bisognano–Wichmann 恰好等于几何 boost——具体几何是配对的产物，非独立输入（结构同型旁注，元规则 M，本书不生成几何）。这说明：物理学之所以"能算"，是因为它给定了一个外生的视角（真空 + 选定楔形），从而支付了 W.3 的代价（外接一个不可内生的视角），而非解决了它。

W.2.4 命题地位：墙为缝提供基础，非定义缝（哥德尔类比）

W.2.6 视角的另一半：主观原点与唯我论不可内生（A 方向，双半合龙）

本节补齐 W.3。 W.2.1–W.2.4 证的是视角的态半的不可内生（论证以标尺态 \(\omega^*\) 为符号给出；按主文两态命名，视角的态半是参考态 \(\omega_{\text{ext}}\)、完整视角＝\((\mathcal{N},\omega_{\text{ext}})\)——而无凸显态引理不依赖态的选取，故"态半不可内生"对锚 \(\omega^*\) 与视角态 \(\omega_{\text{ext}}\) 一体成立，见 W.2.3 对齐注；视角作为标尺可写、作为出发点不可内生）。视角还有**窗口半 \(\mathcal{N}\)**（因果窗口，理解流的半边模包含子代数）。本节证 \(\mathcal{N}\)-半有其独立的不可内生理由，不搭态半的便车。

W.2.6.1 主观原点非身体。 \(\mathcal{N}\)="够得着的范围"，从一个原点向外够；\(\mathcal{N}\) 与其对易补 \(\mathcal{N}'\) 的交界不是墙（身体）而是点（原点）——"我从这里够出去"的那个"这里"。故 \(\mathcal{N}\) 标定的是主观原点（够得着范围的圆心，零支撑点），非身体（身体是耗散稳态，另一对象，本卷不展开）。

W.2.6.2 客观对偶可平凡。 主观原点的客观对偶 = 相对对易子 \(\mathcal{N}'\cap\mathcal{M}\)（"够不着我的那片"的代数）。由半边模包含理论：\(\mathcal{N}'\cap\mathcal{M}\) 可以平凡（\(=\mathbb{C}1\)）——存在相对对易子平凡的 HSMI。此即"主观是一切够出去的原点，客观可以是零"的代数形：主观原点是实结构（\(\mathcal{N}\) 非平凡），其客观对偶在退化情形什么都没有。

W.2.6.3 标准条件 = 主观原点能否撑起客观世界的判据。 一个 HSMI 称标准，若公共循环分离矢量 \(\Omega\) 对 \(\mathcal{N}'\cap\mathcal{M}\) 循环。标准 HSMI 与 \(S^1\) 上强可加 Möbius 协变网一一对应（可从模算子重构时空对称群）。于是： - 标准 = 客观补非平凡 = 有真世界（有"够不着我的"他者区域、可重构时空）； - 非标准 = \(\mathcal{N}'\cap\mathcal{M}=\mathbb{C}1\) = 唯我论极限（够得着范围之外客观上空，世界收缩成一个点）。

"主观原点是否对应一个客观世界"不是哲学问题，是标准条件成立与否的数学问题。

W.2.6.4 唯我论不可内生。 判定标准条件，需检验相对对易子 \(\mathcal{N}'\cap\mathcal{M}\)——它是 \(\mathcal{M}\) 之内、与 \(\mathcal{N}\) 因果不相交的那块（GLW 图景里"区间 \((0,1)\)"的代数）是否被 \(\Omega\) 循环撑起。而认知构成性地够不到它，两条路都封死：

- 沿动力学（单侧半群）够不到：理解流的可用操作是单侧模流 \(\sigma_t\)（\(t\ge0\)，\(\sigma_t(\mathcal{N})\subset\mathcal{N}\)，Borchers 正能量 \(P\ge0\)）。单侧半群只在 \(\mathcal{N}\) 内部收缩，把够得着的范围越收越小，**永远搬不到与 \(\mathcal{N}\) 因果不相交的那块** \(\mathcal{N}'\cap\mathcal{M}\)——它在因果上不可达，不是"能到但没到"。 - 沿结构操作（反演）够不到：理解流构成性可用的结构操作只有 \(\sigma_{t\ge0}\)（单向收缩）与模共轭 \(J\)（\(J\mathcal{N}J=\mathcal{N}'\)，反酉、时间反演型）——这是关于认知可用工具的范围声明；其中能把 \(\mathcal{N}\) 侧映到其对偶侧的，是 \(J\)。但 \(J\) 把正能量条件翻成 \(P\le0\)（负能量方向），认知的理解流被 \(P\ge0\) 钉死、构成性用不了这个反演。

**故"我够不着的世界（\(\mathcal{N}'\cap\mathcal{M}\)）到底非空还是平凡，是有客观世界还是只有我"这一判定，要求一个认知原则上到不了的位置 ⟹ 唯我论不可内部判定。** 它既不可证也不可反驳：反驳它（确认客观补非平凡）需站到那个因果不相交、且只能经反酉到达的一侧，而那侧对沿单侧正能量流运作的认知封闭。这不是"世界不存在"，而是"世界存不存在这个判定本身，落在一个不可内生的位置上"。

W.2.6.5 双半合龙。 态半（不可内生，W.2.1–W.2.4；对锚 \(\omega^*\) 与视角态 \(\omega_{\text{ext}}\) 一体成立）+ \(\mathcal{N}\)-半（主观原点的客观世界不可内生 = 唯我论不可判定，本节）——**视角配对 \((\mathcal{N},\omega_{\text{ext}})\) 两半各有独立不可内生理由**。W.3 从"态不可内生"扩为"态与主观原点双半皆不可内生"。

严谨度：A 方向（engine = 半边模包含 / Borchers 单侧性 / 标准 HSMI 对应定理，皆标准；"标准↔Möbius 网一一对应"据 Guido–Longo–Wiesbrock 一线，其 HSMI 结构定理的完整证明据 Araki–Zsidó 2005）。两叉的严谨度分标：第二叉（\(J\) 反演须负能量、理解流用不了）是硬的——\(J\mathcal{N}J=\mathcal{N}'\) 与 \(J\) 反转能量符号皆标准。第一叉（单侧半群因果不可达相对对易子）为A 方向、非完全形式化：它依赖"单侧半群轨道的可达闭包不含与 \(\mathcal{N}\) 因果不相交之区"这一直觉，在 GLW 具体结构里成立、但本附录未给完整形式证明；正文引用以第二叉为主承重、第一叉为佐证。诚实标注：与 W.3 态半同性质——本节是"凡认知在场处，唯我论不可内生"的条件定理，非"唯我论为真/为假"的断言（它恰恰证明这个判定不可内生）。

耦合注（公理 I 的一条代数旁证；标 A 级·前件=标准性）。先钉死一件事，防一个自然的误读。 因子类型 III₁ 是代数事实（公理 I，或由"理解流在场"的代数刻画推出，见下）；唯我论／非唯我论是加在同一代数结构上的现象学解读——解读不改变因子类型。无论你把"我够不着的对偶" \(\mathcal{N}'\cap\mathcal{M}\) 读成唯我论（它平凡、只是我的镜像）还是非唯我论（它丰富、是真他者），\(\mathcal{M}\) 还是那个 \(\mathcal{M}\)、还是 III₁，理论照常描述认知。下面这条蕴含不是"解读⟹类型"，是"代数条件⟹类型"。

把标准 HSMI 结构定理（以 Araki–Zsidó 2005 为准）正着读：**若半边模包含 \(\mathcal{N}\subset\mathcal{M}\) 是标准的（公共向量 \(\Omega\) 不仅对 \(\mathcal{N}\)、\(\mathcal{M}\) 循环分离，对相对对易子 \(\mathcal{N}'\cap\mathcal{M}\) 亦循环分离）、非平凡（\(\mathcal{N}\subsetneq\mathcal{M}\)，平移生成元 \(P\ge0\) 非零）且理解流在场，则 \(\mathcal{M}\) 必为 Type III₁。 这里的前件"标准性"是一个代数条件**（对偶非平凡、\(\Omega\) 对之循环分离），不是"世界非唯我论"这个解读态度——只是这个代数条件，恰好是"非唯我论解读为真"在代数侧对应的那一个。对应不是推出：能逼出 III₁ 的是标准 HSMI 这个代数结构（＝理解流在场的一种强刻画），不是选了哪种解读。就此而言，公理 I（无迹）除了第零章的排除链，还有这条代数旁证：它与"理解流以标准 HSMI 形式在场"相容、乃至被其推出——纯代数，不经过"有没有客观世界"这一解读，不新增任何"被给予"。

三处钉死：(i) 承重引用以 Araki–Zsidó 2005（*Rev. Math. Phys.* 17 (2005), 491–543）为准——它把 Wiesbrock 结果从"公共循环分离向量"推广到"公共忠实正规半有限权"，并填补了 Wiesbrock 1993 原证中的一处缺口（缺口恰在需 \(\Omega\) 对 \(\mathcal{N}'\cap\mathcal{M}\) 也循环分离这一步），故不引 Wiesbrock 1993 Main Theorem 承重；(ii) 前件"标准性"（\(\Omega\) 对 \(\mathcal{N}'\cap\mathcal{M}\) 循环分离）正是 W.2.6.3/6.4 证明不可内部判定的那个代数条件——所以这条旁证不能把公理 I 降为无条件定理（要用它，得先假定一个第一人称判不了的前件），且奇异 HSMI（\(\mathcal{N}'\cap\mathcal{M}=\mathbb{C}1\)）存在；(iii) 在奇异 HSMI 那一侧，这条蕴含的前件不满足、III₁ 不再被它保证——但这不是"唯我论那侧因子类型变了"，只是"这条特定的代数旁证在那侧不适用"；\(\mathcal{M}\) 的类型仍由公理 I 独立钉定。此注只立一条代数旁证，既不改公理 I 的地位，也不把唯我论真假与因子类型焊在一起——"没有绝对刻度"（无迹，代数）与"有没有客观世界"（唯我论，解读）是两个层次，本注不混。

W.2.7 视角的第三面：舞台完备性与环境欠定（命题 R★，A 级，证第二章 §2.3）

本节与 W.2.6 成对。 W.2.6 证窗口方向的够不到——"我够不着的客观对偶（世界）是否非空，不可内生"（唯我论不可判定）。本节证舞台方向的够不到——"我所在的环境（更大的 \(\mathcal{M}\)）究竟是哪一个，被一切被活之物欠定"。两面合起来：第一人称被钉死在 \((\mathcal{N},\ \mathcal{M}_{\mathcal{A}},\ \omega^*)\) ＝（窗口，舞台，锚）三件套上，三件套之外无第一人称事实。

设定与唯一框架前提。 \(\mathcal{A}_{\text{phys}}\subseteq\mathcal{M}\) 是学习流沉积的账本（弥散阿贝尔子代数），\(\omega^*\) 忠实正规，\(\mathcal{A}_{\text{phys}}\) 模不变（\(\sigma_t^{\omega^*}(\mathcal{A}_{\text{phys}})=\mathcal{A}_{\text{phys}}\)，学习流的构造性要求，非额外假设）。定义舞台为账本的正规化子闭包 \(\mathcal{M}_{\mathcal{A}}:=\mathcal{N}_{\mathcal{M}}(\mathcal{A}_{\text{phys}})''\)。唯一的框架前提（可及性读法，明标一次）：第一人称的全部通道 ＝ 账本登记（经 \(E_0:\mathcal{M}\to\mathcal{A}_{\text{phys}}\)）＋ 舞台内的动力学操作（含理解流窗口）＋ 被活的绵延（锚态模流）。此为本书一贯的操作读法（与元定理 W 同族），非定理；下游 R1–R8 全为定理级。

命题 R★（舞台完备性／环境欠定）：第一人称的全部数据，只依赖于舞台连同它上面的基线态 \((\mathcal{M}_{\mathcal{A}},\ \omega^*|_{\mathcal{M}_{\mathcal{A}}})\)。任意两个把该舞台包含在内、且在舞台上限制到同一基线态的更大环境，生成逐字相同的第一人称数据。换言之：环境超出舞台的那部分，在账本上零指纹、在绵延里零手感。

措辞钉死（防形而上误读）：结论用"欠定"（underdetermined，逻辑词），不用"舞台就是世界"（形而上断言）。本命题只断言"环境超出舞台的部分不留任何第一人称指纹"——这是定理；不断言"舞台之外无物"。"我当下的模流反推成哪个 \(\mathcal{M}\)"没有第一人称答案；就一切能被登记、被理解、被活出的而言，舞台即世界的全部代数。

证明（R1–R8，除可及性读法外全 A 级；标准件组装：Takesaki 条件期望理论 ＋ 保态期望唯一性 ＋ 塔式分解）。

- R1（舞台模不变，A）：\(\sigma_t^{\omega^*}(\mathcal{M}_{\mathcal{A}})=\mathcal{M}_{\mathcal{A}}\)。 证一行：设 \(u\mathcal{A}_{\text{phys}}u^*=\mathcal{A}_{\text{phys}}\)（\(u\) 在正规化子中），由 \(\sigma_t(\mathcal{A}_{\text{phys}})=\mathcal{A}_{\text{phys}}\) 得 \(\sigma_t(u)\mathcal{A}_{\text{phys}}\sigma_t(u)^*=\sigma_t(u\mathcal{A}_{\text{phys}}u^*)=\mathcal{A}_{\text{phys}}\)，故 \(\sigma_t\) 保正规化子群 \(\mathcal{N}_{\mathcal{M}}(\mathcal{A}_{\text{phys}})\)；\(\sigma_t\) 为正规自同构，保双对易闭包，故保 \(\mathcal{M}_{\mathcal{A}}\)。\(\square\) - R2（舞台期望与流限制，A）：存在唯一忠实正规 \(\omega^*\)-保态条件期望 \(E_{\text{st}}:\mathcal{M}\to\mathcal{M}_{\mathcal{A}}\)，且 \(\sigma_t^{\omega^*}\big|_{\mathcal{M}_{\mathcal{A}}}=\sigma_t^{\,\omega^*|_{\mathcal{M}_{\mathcal{A}}}}\)。 据 Takesaki（*Conditional expectations in von Neumann algebras*, J. Funct. Anal. 9 (1972), 306–321）：子代数模不变 \(\iff\) 保态条件期望存在，且模群限制到子代数 ＝ 限制态自己的模群。由 R1 前件满足。**被活的绵延在舞台上自洽封闭：它就是限制态 \(\omega^*|_{\mathcal{M}_{\mathcal{A}}}\) 自己的模流。 - R3（舞台 Cartan 对，A）**：\((\mathcal{M}_{\mathcal{A}},\mathcal{A}_{\text{phys}})\) 为 Cartan 对（正规性按构造——正规化子生成整个舞台；MASA 性由 \(\mathcal{A}_{\text{phys}}\) 在 \(\mathcal{M}\) 中极大交换遗传到 \(\mathcal{M}_{\mathcal{A}}\)；保态期望由 \(E_0\) 限制到 \(\mathcal{M}_{\mathcal{A}}\) 给出）。此即全书机器（Feldman–Moore、\([\sigma]\)）运转的构造级基础，不欠任何开放问题。 - R4（塔式分解／账本零指纹，A）：由保态条件期望唯一性，\(E_0=E_0\big|_{\mathcal{M}_{\mathcal{A}}}\circ E_{\text{st}}\)。故对一切 \(x\in\mathcal{M}\)，\(x\) 的账本登记 ＝ \(E_{\text{st}}(x)\) 的账本登记，且 \(E_0\big(x-E_{\text{st}}(x)\big)=0\)——环境超出舞台的部分在账本上零指纹。 - R5（环境欠定，A，主命题）：第一人称全部通道（账本登记及其统计、舞台内动力学、被活绵延）只依赖 \((\mathcal{M}_{\mathcal{A}},\ \omega^*|_{\mathcal{M}_{\mathcal{A}}})\)。任取两个环境 \(\mathcal{M}_1,\mathcal{M}_2\supseteq\mathcal{M}_{\mathcal{A}}\)（各配忠实正规态、限制到同一 \(\omega^*|_{\mathcal{M}_{\mathcal{A}}}\)、满足模不变），由 R2 绵延在两者中皆 ＝ 限制态模流（相同）、由 R4 账本登记只经 \(E_{\text{st}}\)（相同），故生成逐字相同的第一人称数据。\(\blacksquare\)（R2＋R4 直接合成） - R6（环境级"正则性"问题的消解，A）：老稿"\(\mathcal{A}_{\text{phys}}\) 在 \(\mathcal{M}\) 中是否 Cartan／\(\mathcal{M}_{\mathcal{A}}=\mathcal{M}\) 吗"这一问，＝ 对"舞台之外是否有物"作全通道清点 ＝ 态外位（W.2 的墙、第三章禁格）。由 R5，它对一切被活之物无差别——不是"很难判定"，是"其答案不接触任何能被登记、被理解、被活出的东西"。它够不到的根是"嵌入非内蕴"（同一舞台可嵌入不同大小的环境，内部数据读不出嵌进了哪个），与无迹（W.2 墙的根）是两个不同的根，别混（详见第三章 [gejing-math]）。 - R7（剩余的身份，结构陈述）：越过舞台的操作，恰是那些不保持我的经典框架的操作——其中包括把账本 \(\mathcal{A}_{\text{phys}}\) 共轭成另一份账本 \(u\mathcal{A}_{\text{phys}}u^*\) 的酉元（另一个 Cartan 截面／另一个舞台的框架）。它们不是舞台内的操作；触及它们的唯一方式是换锚重沉积（不可逆，付 B1）。主体从不观测"舞台的叠加"，只在事后发现自己已在某一个舞台里。这是第八章多舞台转化的接口。 - R8（适配不可携带，A）：R2 前件（模不变）是相对于当前锚的。换锚之后（新 \(\omega^{*\prime}\)），旧账本一般不再对新模流不变 ⟹ 新锚下保态期望不存在 ⟹ 旧舞台对新锚不是舞台。"正则性（适配义）"随锚生随锚灭，不可跨锚携带。 这正是 B1（参照不可回）在正则性寄存器上的显形。

严谨度：R1–R6、R8 全 A 级（Takesaki 期望理论 ＋ 保态期望唯一性 ＋ 构造，无一新数学）；唯一非定理件是设定处那条可及性读法（框架前提，已明标一次，与元定理 W 同族）；R7 为结构陈述（"剩余恰含框架改换者"精确，"剩余尽是框架改换"不主张）。\(\blacksquare\)

与 W.2.6 的对称、公理 I 卸重（两句）：窗口半（W.2.6）证"我够不着的客观对偶存不存在，不可内生"（唯我论）；舞台半（本节 R5）证"我舞台之外的环境剩余是什么，欠定"。环境从两个通道（因果窗口／经典账本）都欠定——第一人称被钉死在（窗口，舞台，锚）三件套上，三件套之外无第一人称事实。 又：公理 I 钉的是结构类型（III₁——被四步锚链钉住的是被活结构的型），从不宣称固定"是哪一个 \(\mathcal{M}\)"或舞台之外的延展；R★ 恰说明此设定不多花一分钱——它对一切被活之物零附加主张。锚链钉型、R★ 卸重，公理 I 因此轻装。

W.3 主定理 B(刻画):缝与他者的代数身份

W.3.1 定义与地位:缝的两层

层一·缝:\(\mathrm{Deficit}=\mathrm{id}-\iota\circ E\neq0\iff\mathcal{R}\) 非平凡。MASA-free、恒在、基本他者性。 层二·扭:\([\sigma]\neq0\in H^2(\mathcal{R},\mathbb{T})\)=缝连规范都抹不平。需阿贝尔(无内部酉元拉平)+迷向平凡(FM 有定义域)。 **\([\sigma]\) 的地位**:在给定的、被给予的配对 \((\mathcal{M}_{\mathcal{A}},\mathcal{A}_{\text{phys}})\) 上有确定值;不可判定的是我的账本是否极大(=我算的是不是这个舞台的那个类)。

**\([\sigma]\) 的典范性 vs 完全性:非极大不完全是特征、非缺陷。 要把两件此前易混的事分开。典范性(有确定值、零选择歧义):由命题1 给,只依赖阿贝尔、不依赖极大**——非极大账本上 \([\sigma]\) 照样典范。完全性(是完全不变量、\((\mathcal{R},[\sigma])\) 完整刻画 Cartan 对):由 FM Thm 1 给,只在极大[Cartan]时成立。二者不是一回事:非极大账本上,\([\sigma]\) 有典范值、但不是完全不变量。Remark(不完全是特征):这个"不完全"不是需要补的洞:它精确等于"我看到的障碍,是真 \([\sigma]\)(换任何更大账本都抹不平),还是只因我账本不够大(还没覆盖能吸收它的对易子代数)"这一问的不可判定。而这个不可判定,正是学习流永远有得扩张的空间(接 W.1.3.2"\(\mathcal{A}'\cap\mathcal{M}_{\mathcal{A}}\) 里冒出新可登记者=我还能学"——这是它的判别论版本)。(分寸,详 W.6.1.3:不可判定供的是空间〔永远推得动〕,不是推力——缝隙惰性、不产生力;真正推动扩张的是他者(遭遇抹不平逼重切)。没有他者,不可判定的缝隙静静敞着、谁也不动〔泥沼〕;不可判定保证学习永不完成,他者保证学习真在发生。)彻底判定(=\([\sigma]\) 变完全)只在极大 MASA 达成,而那正是死点(视域填满、\(K=1\)、无可再学,W.1.3.2)。⟹ 健康认知永远停在非极大:永远带着"这障碍可能是真他者、也可能是我不够大"的不可判定活着、继续学。这条同时消解"想要极大 MASA"的引力:极大 MASA 不是"更完备的账本",是"唯一让他者判定停机的账本"=停止学习=死;想要它=想要判定停机=想死。所以本书"账本不必极大"不是权宜、不是绕开 Cartan 的补丁——是"完全性=死"这个正面事实的推论:\([\sigma]\) 的不完全,与认知的活着,是同一件事。

Remark(遭遇机制,拦一个循环,):可见锚 vs 已纳入。 "可见=对同一 \(\mathcal{A}\) 限制一致"是稳态定义;而遭遇他者恰恰是"对方对我旧账本呈现为不一致/异常"(账本连法容不下)。故遭遇先于重切,不能用重切后的 \(\mathcal{A}\) 定义遭遇——否则循环。分两个账本破环。可共享层:我俩都能用的那些**\(\sigma_t\)-稳定的对易子代数**——它们的读数落在 \(\ker(\omega_{\text{我}}-\omega_{\text{他}})\) 里(两态在其上一致)。Remark(子空间 ≠ 子代数):\(\ker(\omega_{\text{我}}-\omega_{\text{他}})\) 只是子空间、不是子代数(两算子在核里、乘积不一定在),所以"可共享"不是"核里的最大账本"这么一个东西——它是一族我俩都能用的稳定对易子代数,且极大者不唯一(Zorn、补全非唯一,同 MASA 病)。故不取"最大 \(\mathcal{A}_{\text{共}}\)",只取:存在共享上界——对我此刻的账本 \(\mathcal{A}_{\text{我}}\),存在一个我俩都能用的、更大的稳定对易子代数把它含住。**已纳入 \(\mathcal{A}_{\text{我}}\)**=我此刻实际登记的。遭遇他者=撞见"我俩本可共享、我还没长到"的东西(某个共享上界里、\(\mathcal{A}_{\text{我}}\) 之外的算子)。我重切改变的是 \(\mathcal{A}_{\text{我}}\)(朝某个共享上界长),不改变"我俩潜在能共享什么"(那族稳定对易子代数,由两态在 \(\ker\) 上的一致性定,与我登记到哪无关)。可共享(潜在,一族/有上界)与已纳入(实际,一本)是两回事,别混。承重结论不需要"最大共享账本"唯一,只需"存在共享上界":可见性在共享层不动、重切只改 \(\mathcal{A}_{\text{我}}\)——弱化后全保。此机制与 W.1.3.5(身份由历史填)、W.1.3.6(已知=增滤流一节)一致:\(\mathcal{A}_{\text{我}}\) 朝 \(\mathcal{A}_{\text{共}}\) 长,正是学习流被遭遇他者所推。

W.3.2 主定理 B 之(2):裂缝 = \([\sigma]\)(A 级,证第二章命题 2.4;Cartan 实现层内)

学习把世界压成可说的事实,总漏下一点压不平的东西。这一节去量那个"漏下的",并证明它压不平的内核恰是他者 \([\sigma]\)。漏是被给予的——你领受它,补不上。

命题(主定理 B 之(2))

设 \(\mathcal{M}_{\mathcal{A}}=\mathcal{N}(\mathcal{A}_{\text{phys}})''\)，\(\mathcal{A}_{\text{phys}}\subset\mathcal{M}_{\mathcal{A}}\) 为 Cartan MASA，\(E:\mathcal{M}_{\mathcal{A}}\to\mathcal{A}_{\text{phys}}\) 为忠实正规 Umegaki 条件期望（学习），\(\iota:\mathcal{A}_{\text{phys}}\hookrightarrow\mathcal{M}_{\mathcal{A}}\) 为自然嵌入（理解）。定义亏量（裂缝）\(\mathrm{Deficit}:=\mathrm{id}_{\mathcal{M}_{\mathcal{A}}}-\iota\circ E\)。则裂缝分两层：
\[\text{（可理解层）}\quad\mathrm{Deficit}\neq0\iff\mathcal{R}\ \text{非平凡}\iff\mathcal{M}_{\mathcal{A}}\supsetneq\mathcal{A}_{\text{phys}};\]
\[\text{（他者层）}\quad\mathrm{Deficit}\ \text{的非平凡性不可被重新规范消去}\iff[\sigma]\neq0\in H^2(\mathcal{R},\mathbb{T}).\]

证明

Feldman–Moore 分解。 由 Feldman–Moore（1977 II, Theorem 1——Cartan 对 ↔ 带 2-余圈的可测等价关系之分类定理；注：此定理给的是这个对应，不是"顺从⟹\(H^2=0\)"的出处，后者是 FM＋CFW 的联合推论，见 W.5.1），Cartan 对 \((\mathcal{M}_{\mathcal{A}},\mathcal{A}_{\text{phys}})\) 同构于扭曲群胚交叉积 \(W^*(\mathcal{R},\sigma)\)，其中 \(\mathcal{R}\) 是 \(X=\mathrm{Spec}(\mathcal{A}_{\text{phys}})\) 上的可数标准测度等价关系，\(\sigma\in Z^2(\mathcal{R},\mathbb{T})\)，\([\sigma]\in H^2(\mathcal{R},\mathbb{T})\) 是 Cartan 对的完全不变量（**精确:\(\mathcal{R}\) 唯一到同构、\([\sigma]\) 唯一到 \(N(R)\) 轨道**——FM II Thm 1 末句原文逐字:"The relation R is unique up to isomorphism, and the orbit of σ under N(R), the automorphism group of R, is unique";FM II §4 通篇讲的正是这个 \(N(R)\)-轨道的模糊性,故"完全不变量"须读作"\((\mathcal{R},[\sigma])\) 在 \(N(R)\) 轨道意义下"）。群胚元素 \((x,y)\in\mathcal{R}\) 对应算符 \(u_{(x,y)}\)，乘法由 \(\sigma\) 扭曲：\(u_{(x,y)}u_{(y,z)}=\sigma((x,y),(y,z))\,u_{(x,z)}\)。

**\(E\) 与 \(\iota\) 的群胚对应。** \(\iota:L^\infty(X)\hookrightarrow W^*(\mathcal{R},\sigma)\) 把单位空间函数映为对角（单位态射 \((x,x)\)）算符；\(E:W^*(\mathcal{R},\sigma)\to L^\infty(X)\) 把任意群胚算符投到单位态射分量（忠实正规 Umegaki 条件期望，\(E\circ\iota=\mathrm{id}_{L^\infty(X)}\)）。

本引理直接证 W.1.3 的精确版（裂缝分两层）；不证已被 W.1.3 否定的旧命题"\(\mathrm{Deficit}\neq0\iff[\sigma]\neq0\)"。

**层一：\(\mathrm{Deficit}\neq0 \iff \mathcal{R}\) 非平凡。** 复合 \(\iota\circ E\) 把任意群胚算符投到对角分量、抹去所有非单位态射（\(x\neq y\)）分量。故 \(\iota\circ E=\mathrm{id}\) 当且仅当不存在非单位态射，即 \(\mathcal{R}=\{(x,x):x\in X\}\)（对角关系）。等价地：\(\mathrm{Deficit}=\mathrm{id}-\iota\circ E\neq0 \iff \mathcal{R}\) 含非对角态射 \(\iff W^*(\mathcal{R},\sigma)\supsetneq L^\infty(X)\)，即 \(\mathcal{M}_{\mathcal{A}}\supsetneq\mathcal{A}_{\text{phys}}\)（有理解，非坍缩）。**此层与扭曲 \(\sigma\) 是否平凡无关**——只要 \(\mathcal{R}\) 非平凡，无论 \([\sigma]\) 是否为零，亏量都非零。

**层二：亏量的不可规范化部分非零 \(\iff [\sigma]\neq0\)。** 设 \(\mathcal{R}\) 非平凡（层一）。亏量所含的非对角结构，能否被一个 1-上闭链重新规范 \(\tilde u_{(x,y)}=c(x,y)^{-1}u_{(x,y)}\)（\(c\in Z^1(\mathcal{R},\mathbb{T})\)）平凡化，严格地由 \(\sigma\) 是否为上边缘决定： - 若 \(\sigma=\partial c\)（上边缘，即 \([\sigma]=0\in H^2(\mathcal{R},\mathbb{T})\)），则上述重新规范使 \(\tilde u_{(x,y)}\tilde u_{(y,z)}=\tilde u_{(x,z)}\)（扭曲消去），\(W^*(\mathcal{R},\sigma)\cong W^*(\mathcal{R},1)\)——非对角结构存在但无不可消除的扭曲相位。 - 若 \(\sigma\neq\partial c\) 对任何 \(c\in Z^1(\mathcal{R},\mathbb{T})\)（即 \([\sigma]\neq0\)），则不存在使乘法关系平凡化的重新规范——亏量带不可消除的扭曲相位。

故"亏量带不可由任何 \(c\in Z^1\) 规范消去的扭曲" \(\iff \sigma\neq\partial c\ (\forall c) \iff [\sigma]\neq0\)。\(\blacksquare\)

精确化(防虚假等价)

两层结构说明。 层一表明 \(\mathrm{Deficit}=0\) 严格等价于"\(\mathcal{R}\) 退化为对角关系"（无非对角态射），这比"\([\sigma]=0\)"强——\([\sigma]=0\) 但 \(\mathcal{R}\) 非平凡时，\(\mathrm{Deficit}\neq0\)（有非对角态射，但扭曲可被规范消去）。故早期单层表述"\(\mathrm{Deficit}\neq0\iff[\sigma]\neq0\)"不精确，本节以两层版为准（命题框即两层版）。

精确版（A 级）：裂缝有两个层次，需分清—— - 亏量非零（\(\mathrm{Deficit}\neq0\)）\(\iff\) \(\mathcal{R}\) 非平凡（有非对角态射）\(\iff\) \(\mathcal{M}_{\mathcal{A}}\supsetneq\mathcal{A}_{\text{phys}}\)（有理解，非坍缩）。这是可理解性层（学习压缩丢的是"非对角信息"，无论扭曲是否平凡）。 - 亏量带不可消除扭曲（\(\mathrm{Deficit}\) 的非平凡性不可由重新规范 \(\tilde u=c^{-1}u\) 消去）\(\iff\) \([\sigma]\neq0\)。这是他者层（学习压缩丢的信息里，有一部分带不可拉平扭曲）。

**所以"裂缝 = \([\sigma]\)"的精确陈述是**：裂缝（亏量）总非零（只要有理解，\(\mathcal{R}\) 非平凡）；裂缝中不可由规范消去的那部分 = \([\sigma]\)。他者 = 裂缝里抹不平的内核，不是裂缝全部。这与第三章命题甲一致（他者 = 带不可拉平扭曲的剩余，非全部剩余）。

严谨度：A（Feldman–Moore 分解 + 群胚代数标准结构；精确化后无虚假等价）。

对第二章命题 2.4 的回灌：命题 2.4 应表述为"裂缝的不可拉平内核 = \([\sigma]\)"，而非"裂缝 = \([\sigma]\)"。裂缝本身 = 有理解（\(\mathcal{R}\) 非平凡）；其抹不平的内核 = 他者（\([\sigma]\neq0\)）。

严谨度：拆解所依据的代数事实（BW = boost、谱条件 ⟺ 正能量平移半群、弯曲时空真空不唯一、Unruh/Hawking、DHR 重构）均为标准结果（A）；将它们定位到界限论部件上的读法，是结构同型旁注（元规则 M，非预言、非本书新定理）；"脉络与物理规范的精确关系"已核（不同层级、分述、不焊，与 Coleman–Mandula 自洽）；进一步撕开 \(G\) 已核为派生对象（输入=局域网 + 真空 + 时空维度=视角 \((\omega^*,\mathcal{N})\) + 几何，模流不作 \(G\) 重构的显式原料、仅经 \(\omega^*\) 间接在场，故 \(G\) 非原始颗粒、非"差"、非 \([\sigma]\) 群化），界限论不为其单设部件。

W.7 开放问题

1. **\((\star)\)【承重级·首位】**:存在遍历 \(\mathcal{R}\) 使 \(H^2(\mathcal{R},\mathbb{T})\neq0\)。归结 Popa 2007 §6.6 的期望式表述(带问号 \(H^n(\mathcal{R}_\sigma)\cong H^n(\Gamma,\mathbb{T})\)?;归结=给欠债具名出处,非还债)。三支改述(承载 \([\sigma]\) 的 \(\mathcal{R}\) 是舞台的、舞台型未定):舞台 III₁/半有限/型立前件;**\((\star_{\mathrm{II}_1})\)**=存在 pmp 遍历 \(\mathcal{R}\) 使 \(H^2\neq0\)=Popa 猜想原生形态(若舞台可 II₁,gap 消失)。护栏:有迹的是舞台非世界(\(\mathcal{M}\) 仍 III₁、理解流住 \(\mathcal{M}\),不违公理 I)。

2. **\((\star'')\)**:非阿贝尔窄窗口(纤维=有非平凡中心的 II₁ 代数、\(G\) 无限非顺从、扭不被轨道吸收)——unknown 地带,并列不押。

3. LHS 谱序列(情形② 荷/他者分离算子)——存在性已确认(Sauer–Thom,离散测度群胚扩张的 HS 谱序列,\(E_2\)=底×纤维、\(d_2\) 有明确代数描述);**具体到我们设置的 \(H^2\) 微分尚未算出**(习题级、非墙);核清前情形② 他者与 \((\star)\) 分层挂、勿混同。

4. 定理 2 测度论细节（"MASA⟺Weyl 群胚迷向本质平凡"非孤立定理,是 FM+Renault[C\*-代数版:effective Weyl 群胚]+Aoi–Yamanouchi[正规化群胚]的核心思想整合;vN 代数版为其自然推论）;\(K\) 随落账单调;奇异 MASA 在 III₁ 的存在性已解决（Houdayer–Popa 2017,"Singular MASAs in type III factors":满足 Connes 双中心化性质[CBP]的 III₁ 因子含奇异 MASA、且为正规条件期望值域;Araki–Woods 等重要因子满足 CBP;"全部 III₁ 是否都含"仍开放）;\(\mathcal{R}\) 由群实现之前件。

5. 开放二(单 \(\mathcal{M}\) 上多 Cartan 一零一非零;Connes–Jones 1982/Popa–Vaes 2014 已挂)——真开放、不承重。

6. 开放一(永远开):裂缝自身的动力学——缺内部原则上生不出的语言(W.2 保证);全书那道缝的数学位置。

7. **"已知"滤流的进度参数 \(s\)(待算·非承重)**:W.1.3.6 定义只用"存在增滤流 + 不可内生截断"(确定、足够严格)。\(s\) 作排序坐标不需值(已证);作有意义刻度,最自然候选是 Araki 相对熵(单调性由数据处理不等式给)。**\(s\) 与模流 \(t\) 的函数关系不追求**——两条流无桥(学习流 B1 单向 vs 模流双向),\(s=f(t)\) 若成立反是焊接警号;强给关系会引入几何、违元规则 M,故意不算。此条不影响"已知"定义的严格性(W.1.3.6 体检五条已过)。
