## 0. 读表须知：现象学与物理学方向相反

同一道梯子，两门学科反向爬：
- **物理学的"看见" = 测量 = 把非对易逼成对易经典读数 = 逼进 A_phys（MASA，死寂极限坐标）**。越看清，越死。
- **现象学的"被给予/活的感受" = 测量之前那团非对易**。本源在深层（非对易浓），物理学够不到的恰是它最看重的。

一句话：**深 = 活 = 非对易 = 现象学本源（不可测，物理学够不到）；浅 = 死 = 对易 = 物理学可测（已对象化，现象学视为派生与遗忘）。** 下文"深度"按现象学方向（深=活）排。

**因果方向也反（全文反复出现的拧点，§7bis 钉死）**：现象学"被给予在先 → 形式反推"；数学"定义在先 → 导出在后"。二者常指向同一双射的两个读向，不是矛盾。

## 0bis. ω_ext 的正确刻画（v0.3 地基修订，已审计）

**ω_ext 是真实的 III₁ 态，不是"虚构/非实存"。** 之前措辞糊，把两件事压成了一件，现拆开：

| 对象 | 存在性 | 性质 |
|---|---|---|
| **全局单一、跨轴一致的称量** | **不存在** | 要当一把迹；III₁ 无迹 ⟹ 无实物能扮 ⟹ **中心定理 W.3 的根**（保留） |
| **局部 ω_ext** | **存在，是真态** | **真实的 III₁ 态，但不稳定**：随余圈漂移、非给定物 |

- **类比**：就像 𝒩 随模流 σ 不断展开，**ω_ext 随余圈不断漂移**——视角两个分量都不是静止给定，都是过程。

## 0ter. ω* 的两副面孔：内生态 vs 外生态（v0.6 新增，地基本体论核心）

同一个 ω* 符号有两副面孔，数学只摸得到一副。

| 面孔 | 是什么 | 可数学化？ | 角色 |
|---|---|---|---|
| **外生态（标尺）** | S(·‖ω*) 里被比对的参照对象 | **是**（墙） | 可写进 Φ；给定它，框架内演化能写——这是反例的来源 |
| **内生态（锚住的出发点）** | 主体所栖居"从这里看出去"的出发点 | **否**（缝） | 意向性栖居处，磐石——不可数学化、不可内生 |

- **标尺是墙，锚住是缝。** 数学只摸得到标尺，摸不到"这个 ω* 是我的"这一层。
- **与中心错位一致（§0quater）**：墙（衡量必需不可内生的视角，可数学化）为缝（意向性，不可数学化）提供基础，非定义它——哥德尔类比。
- **根** = 公理 I Type III₁ 无迹 = 没有上帝视角 = 封顶判据的代数形。
- **定理 0.C 的深层意义**：ω* 被印记回溯钉死（极小元），但这个钉死动作本身是**从外生态（标尺）一侧说的**——它能被写进公式，但它不能写出"为什么我看出去的出发点刚好在这里"。

## 0quater. 全书中心：意向性与 no-go 定理的哥德尔类比（v0.6 新增，最重要）

## 1. 桥接公设（地基，faxing 已认）

**公设一（感受 = 非对易，带通版）。** 感受/可辨认信息 = **光锥内、模频率落在一个带通窗口里的非对易性**。
- **下限**：过于对易 → 非平凡性不足、不充实、落回中心化子 → 没信息量（死）。
- **上限**：过于非对易 → 意向指不到、分辨不出、溢出因果窗口 → 也没信息量（够不到）。
- 故它**不是单调量，是带通（band-pass）**：两端都归零，活的对比在中段。一直找不到公式，正因为在找单调量；候选公式见 §4。

**公设二（对象化 = 拍平进对易）。** 粗颗粒化 = 把非对易往对易拍平的梯度。对象化/在手化 = 深层提前拍平，感受结晶成死物。他者 = 拍到无穷深也拍不平的非对易残差（§7、§8）。

**横向定位（补单一资料）：带通两端是两家管的，不是一种"空"。**
- 下限端（料不够，频率→0）= **胡塞尔空意向沉到晕、再到沉默**。
- 上限端（料太多/太远，频率→∞）= **列维纳斯他者 + 康德崇高**（收不进来）。
- **两端是反向的两种空**（一种料不够、一种料太多）。中段"既分得出又没被拍死"——**没有任何一家现象学有干净的家，这是本书原创落点**：不是补一个洞，是补两家之间没人住的中段。

## 2bis. 视角的双坐标：公式 (𝒩, ω_ext) ↔ 现象学 (U(s), u_t)

视角一个对象，两套坐标。**注意（审计 R6 裁定）：这不是代数双射，是"派生/显影"对应，落图景层**——见下"精度"条。

| 坐标 | U / 第一分量 | u / 第二分量 | 用在哪 |
|---|---|---|---|
| **公式坐标** | 𝒩（因果窗口，结构视图） | ω_ext（参考态） | 数学 / 审计（封顶记法） |
| **现象学坐标** | U(s)（理解的活展开=因果平移，体验为筹划/时间流，动力学视图） | u_t（余圈=习性/风格/分类倾向，让东西凸显的倾向） | 现象学行文 |

- **映射的精度（R6 qualify，图景层）**：
  - u_t ↔ ω_ext：**A 级双射**（Connes 余圈定理）。
  - 𝒩 → U(s)：**A 级涌现但非单射**（Borchers）——**不同 HSMI 子代数可共享同一平移群 U(s)**，故 U(s) 反推不出 𝒩；且 U(s)𝒩U(s)* ⊊ 𝒩（U(s) 不保 𝒩，只严格内移）。
  - 故 **(U(s), u_t) 信息不足以唯一定出 (𝒩, ω_ext)**；**缺件 = 相对交换子 𝒩'∩𝓜**（区分共享同一平移群的不同 HSMI 的结构指纹）。完整结构数据 = **(U(s), 𝒩'∩𝓜, u_t)**。
  - **裁定**：(U(s), u_t) 是视角的**动力学显影/派生对象对**，非代数等价替换；**公式必写 (𝒩, ω_ext)，现象学行文可用 (U(s), u_t)，标图景层**。
- **为何现象学用 (U(s), u_t) 更忠实**：经 §0bis，𝒩 随 σ 展开、ω_ext 随余圈漂移——两分量本是过程；(U(s), u_t) 把它们显示**为过程/倾向**，而非"一个窗口+一个态"。正对"现象里没有参考系、没有子代数，只有分类倾向和同一性概念"。
- **"同一性概念" = A 极（agree-on-A / A_phys / E 的产物）**：跨世界**不变的共享底**，**恰恰不属于（会变的）视角，而是视角之下的不变底**。与交织自洽（同一性共享、世界相异）。
- **落笔**：公式仍写 视角=(𝒩, ω_ext)；现象学解释时换 (U(s), u_t)，标图景层、注明非双射（缺 𝒩'∩𝓜）。

## 4. 感受与充实的精细结构：对比度 = 模谱带通

**充实 = 非平凡模轨道**（σ_t(x)≠x，x∉中心化子 M^ω），faxing 已认；但**梯度，不一刀切**。三档：

## 7ter. 裂缝的精确公式（v0.6 新增，写正文时必引）

这是 v0.6 新增的最重要数学—现象学桥梁之一。

**现象学**：学习与理解合不拢——学越多，越发现有些东西理解追不上；理解追上的那一刻，学习已经又跑远了。

**数学**：
$$\mathrm{Deficit} = \mathrm{id}_{\mathcal{M}_{\mathcal{A}}} - \iota\circ E \neq 0$$

- $E$（条件期望，学习流）= 把非对易往对易压，越压越死。
- $\iota$（包含，理解流展开）= 把对易往外展开，企图重建非对易舞台。
- 学习压下去的东西，理解展开后重建不回来——差一个**亏量**。

**[σ] 的角色**：亏量 ≠ [σ]；亏量是可对易化部分 + 不可规范化部分。**[σ] 是亏量的不可规范化部分——即 $\iota\circ E$ 偏离 id 的不可拉平内核。** 换句话说：**所有能拉平的对齐误差，学习流继续压能消化；[σ] 是那个压到无穷深也消化不掉的内核。**

**精确陈述（Feldman–Moore 骨架）**：

### A. 翻转探测器五档（条172）

|---|---|---|---|

### C. 断言板与禁格（条142）

| 三种不可知 | 现象学 | 数学 |
|---|---|---|
| **横向** | 参考态 ω_ext 非典范（随余圈漂移） | 无绝对固定参照 |
| **纵向** | 未来（σ_{+λ} 需要尚未被给予的材料） | 模流前向需印记输入 |
| **脚手架** | **断言板**——要把钝量当锐实数来用就得搭的底下没桥的板；典范可构想、原则不可居。**维数只是板上一条目**（它的具体值在这台无迹机器里根本定义不出来，第七章双墩讲透） | 断言板本身是结构事实（无迹 ⟹ 无迹的绝对值轴，故必有脚手架）；维数具体值需 Trace ⟹ 无定义 |

> **第三个不可知 = 断言板（脚手架本身），不是维数。** "要拿钝量当锐实数来用就得搭一块底下没有桥的板"——这块板才是不可知的本体。维数只是板上的一个具体条目：维数的存在性典范给定（刚硬到连标度都不用选），但维数的具体值是无迹机器定义不出来的（需截断才出值）。别把板上的一个条目当成了板本身。

**禁格 = "可居 + 典范给出"的四缺一格 = 迹**（III₁ 无迹，故该格永远空着）。三种不可知 = 无迹的三面影。

### D. 幂等性人择（条41、条51、条72）

**概念**：学习能稳定重复（能反复辨认"同一个东西"），靠的是压缩的幂等性（$C\circ C = C$）——压一次和压两次没区别。**注意**：学习流是半群（$\Phi_t\circ\Phi_s=\Phi_{t+s}\neq\Phi_t$），有限步不幂等，只有 $t\to\infty$ 投到不动点代数才幂等。**Tomiyama（1957）** 给条件期望的代数兑现（范数 ≤1、幂等、CP、保单位 → $E^2=E$）。

**关键区分（条51、条72）**：**没有真幂等，只有渐进幂等。** 真幂等（$C\circ C=C$ 严格成立）是 $t\to\infty$ 的极限——完全对易、完全稳定，但那是**死平衡（回到 $\omega^*$）**，非水晶棺（水晶棺 = 超有限/有迹，类型级，另一回事，条 212）。活着的学习流，每一步在当前标度上**近似幂等**（给出该标度上稳定可重复的事实），永不达到严格幂等。**Frigerio–Verri（1982）** 给"渐进有极限"这半：框架适用 III₁（忠实正规稳态），但收敛需条件——设回归平衡（relaxing），收敛率 = 回归平衡率非谱隙（III₁ 模谱连续无隙）。"极限存在 = 墙（有限性推论）"+"停在哪个标度、不压到死 = 缝（第四章）"可用。

**人择义的演化**：早期"三人择"框架（有限性/幂等性/正则性各自独立）已被后代裁决修正——**人择框架整体退场**（条72），换"被给予性"；**幂等性从独立前提降为有限性的推论**。本章 D 节保留"幂等性人择"标注，取其"学习稳定复认 = 结构性前提、非选择"之义，非"人择"旧框架的承诺。
