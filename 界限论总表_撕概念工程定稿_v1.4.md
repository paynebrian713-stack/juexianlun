### 三条脊梁命题

**W★（被给予侧）**：被给予生成一切**结构**（A），不决定任何**后果**。

**W★★（单向耦合）**：
$$\underbrace{\text{被给予}}_{\Sigma} \;\xcancel{\longrightarrow}\; \underbrace{[\sigma]}_{\text{自由}} \qquad\text{但}\qquad \underbrace{[\sigma]}_{\text{自由}} \;\longrightarrow\; \underbrace{\text{被给予结构的动力学后果}}_{\text{后果}}$$
后向断 = 元定理 W；前向通 = 持久性/度规/图灵型-b 的总闸。**后果 = 自由（A）× 几何实现（C/□）；B 是未拆时的幻觉。**

**W★★★（开放间隙）**：封闭间隙（F1）= 知道它不可知（自由，问题关闭）；开放间隙（□）= 知道它出不来、但还不知道是哪种不知道（问题敞开）。**几何、经验都不是垃圾桶。**

**W◆（自由的减法定义、双向断与范围量化）**：

*（0）$\mathcal{M}$–$[\sigma]$ 关系*——第三种关系（决定值域，不决定取值）：
$$\mathcal{M} \text{ 决定 } [\sigma] \text{ 的取值空间 } \mathcal{C}(\mathcal{M}),\ \text{但不决定取哪一点。}$$
$\mathcal{M}$ 是"值域的给予者"（容器被给予，A），$[\sigma]$ 是"值域内未被给予的取值"（容器里取哪点不被给予，F）。这是 W★★ 后向断的精确机制：被给予给的是容器 $\mathcal{C}(\mathcal{M})$，不是值。

*（1）减法定义*——不正面定义自由，用两个限制之外的余：
$$\text{自由} = \big(\Sigma \text{ 推不出的}\big) \cap \big(\mathcal{A}_{\text{phys}} \text{ 测不到的}\big) = [\sigma]$$
- 代数未限制：$\Sigma\nvdash[\sigma]$（元定理 W）。
- 观测未限制：$[\sigma]$ 在 $\mathcal{A}_{\text{phys}}$ 内无投影（交换层不可见嵌入扭曲），经典时空仪器测不到。
- 无遗漏：代数可及（公理推出）∪ 观测可及（$\mathcal{A}_{\text{phys}}$ 内）之外，只剩嵌入的非交换扭曲 = $[\sigma]$。

*（2）双向断*：
$$\underbrace{\text{代数 }\Sigma}_{\text{上头}} \;\xcancel{\longrightarrow}\; [\sigma] \;\xcancel{\longleftarrow}\; \underbrace{\text{观测 }\mathcal{A}_{\text{phys}}}_{\text{下头}}$$
上头给不出值（W◆-1，元定理 W，给的是值域）；下头是读法的产物，反限制不了取值（W◆-2，$[\sigma]$ 在 $\mathcal{A}_{\text{phys}}$ 无投影）。自由严格在两头都够不着的交集。

*（2-bis）W◆-2 的主观侧补全（自由不可自知）*：W◆-2 的"下头"原讲**物理世界** $\mathcal{A}_{\text{phys}}$（交换截面）测不到 $[\sigma]$。同一个不可及在**主观**一侧也成立：主观 = 模流 + 主观群胚 $\mathcal{R}$（Feldman–Moore 群胚），**不是**物理世界 $\mathcal{A}_{\text{phys}}$。但主观群胚也自知不了自己的 $[\sigma]$——
$$\text{站在 }\mathcal{R}\text{ 内的某态/某轨道上（GNS 相对一个态，公理 I 视角主义：无绝对视角）}\;\Rightarrow\;\text{只拿到 }\mathcal{R}\text{ 的局部轨道资料}$$
而 $[\sigma]\in H^2(\mathcal{R},\mathbb{T})$ 是 $\mathcal{R}$ 的**全局上同调类**；**局部轨道资料定不出全局类**。故主观判不了自己的 $[\sigma]$。
> **单机制（钉死，勿写"上同调阶数高"）**：此不可及的机制**就是**视角主义——局部 $↛$ 全局，**一个根**，不是"阶数限制 + 视角限制"两个根。给定 $(\mathcal{R},\sigma)$，$[\sigma]$ 完全确定（Feldman–Moore），不存在"阶数高所以够不着"。"自由不可自知"恰是自由的标志：**能被主观判定的都是被给予的结构（$\mathcal{R}$ 的局部），不是自由（全局 $[\sigma]$）**——若自由能被自己判定，它就被该判定决定了，便不再是自由。
>
> **合起来是完整图景**：自由**两头都够不着**——物理世界 $\mathcal{A}_{\text{phys}}$ 测不到（W◆-2 下头），主观群胚 $\mathcal{R}$ 自知不到（本条，主观侧）。内外两侧都够不着，才是自由"够不着"的完整代数图景。

*（3）范围量化*（纯代数）：
$$\text{自由的范围} = \dim\mathcal{C}(\mathcal{M}) = \mathcal{M} \text{ 的不共轭 Cartan MASA 模空间维数}$$
| $\mathcal{M}$ 类型 | $\dim\mathcal{C}$ | 依据 |
|:---|:---|:---|
| 超有限（ITPFI） | $0$（Cartan 唯一） | Connes–Feldman–Weiss |
| 非顺从（某些 Popa 型因子） | 可达 $\mathfrak{c}$（不可数多不共轭 Cartan） | Popa 学派 deformation/rigidity |
| 无 Cartan（如 $L(F_n)$） | $\mathcal{C}=\varnothing$（无经典截面） | Voiculescu |

**回应余地 = $\dim\mathcal{C}(\mathcal{M})$**（= $\mathcal{M}$ 的不共轭经典截面数 = 我能以多少种方式回应；纯代数，不配"跃迁/体验"语言）。**注意三件事**：(1) $\dim\mathcal{C}$ 是**回应余地/回应空间大小**，**不是"他者丰富度"**——带他者（$[\sigma]\neq0$）与回应余地（$\dim\mathcal{C}$）是两个独立维度（Ozawa–Popa：非顺从 + Cartan 唯一 = 有他者却唯一回应）。(2) $\dim\mathcal{C}$ 的**大小**被 $\mathcal{M}$ 确定（是 $\mathcal{M}$ 不变量 = 被给予的"自由容量/命运"），**不被物理世界、不被他者确定**（后两者可证）；**不被任何东西确定的是"在 $\dim\mathcal{C}$ 内取哪点（实际回应）"**——那才是自由。(3) $\dim\mathcal{C}$ 小须拆两种：(a) 顺从无他者（真决定论）；(b) 有他者却被锁死回应（Ozawa–Popa）。

#### 4.A.3 号差（洛伦兹负号从哪来）—— 撕了三轮，本节为最终版

**框架天然给出退化度规** $(0,+,+,\dots)$：模流方向是 $g$ 的零方向（KMS：$\omega^*\circ\sigma_t=\omega^*$），横截方向正。**不是**洛伦兹的 $(-,+,+,+)$。

**负号不可内生定理（DeepSeek，裁决 A）**：Araki 相对熵 Hessian 在实切空间严格正半定（相对熵凸性 + $\omega^*$ 是极小点）；号差在实变换下不变（Sylvester 惯性律）；模流方向是零本征值非负本征值。故**洛伦兹负号不可能内生于实信息几何**——出负号必离开"$\omega^*$ 是 $F$ 极小值"这一前提，即引入复化（Wick）或外部对合（Krein）。

**$J$ 路径的精密核查（DeepSeek 计算）**：
$$J v_t J^{-1} = J(i\ln\Delta)J^{-1} = (-i)\cdot\ln(\Delta^{-1}) = (-i)(-\ln\Delta) = i\ln\Delta = v_t$$
即 **$J$ 与模流方向对易，不是反对易**。后果：模流方向**不自动**落进 $J$ 的负本征空间。$J$ 提供"可以出负号"的材料（$\pm1$ 本征空间，A），但"负号对齐模流方向（时间）"代数不强制。

**号差最终归属**：
| 成分 | 归属 |
|:---|:---|
| 退化度规 $(0,+,+,+)$ | A（相对熵 Hessian + KMS） |
| 复结构 | A（模流需要，在 $\mathcal{M}$ 层，见 4.B.4） |

### 4.C-bis 等谱型（谱→几何不一对一）—— 认识论欠定，非自由

**真实现象**："能听出鼓的形状吗"——不能；等谱非等距流形大量存在。读者必问"这种多义算不算自由"。

**判定：不算，是认识论欠定（G），不是本体自由（F）。**

- **是什么**：给定一个谱（可观测数据），多个不同几何（流形）给出同一谱。从可观测反推结构的多义。
- **为什么不是自由**：那些等谱的几何，每一个**自己都是确定的**（被给予的），只是我从谱**区分不开**它们。这是"我反推不出唯一结构"（认识论：数据不足以定），不是"结构本身不被决定"（本体论：自由）。把它当自由 = 把"我够不着"误当"它不被决定"——与图灵型同款错误。
- **落槽**：**G**（认识论边界，有确定答案但观测够不着）。

**与 $\dim\mathcal{C}$ 的关系（关键分界，勿混）**：$\dim\mathcal{C}$ = 同一个 $\mathcal{M}$ 的多个不共轭 Cartan（**一个对象，多读法**）；等谱型 = 多个不同流形给同一谱（**多对象，同投影**）。前者一对象多读法，后者多对象同投影，分属不同层。
> **不与 $[\sigma]$ 建实质联系**：等谱型归 G（几何被决定，只是我够不着），$[\sigma]$ 是 F（不被决定），分属不同槽，不用"投影/剩余"把二者连起来（防软焊缝）。等谱型只在下表与 F1 **并列对照**。

**三种"够不着/不被决定"的并列**：

| 型 | 性质 | 算术层级/归宿 |
|:---|:---|:---|
| 图灵型-a | 算法够不着（有答案算不到） | $\Sigma_1$/$\Pi_2$，**G** |
| 等谱型 | 谱反推不出唯一几何（有结构反推不出） | 认识论欠定，**G** |
| 哥德尔型（F1） | 本体不被决定（$[\sigma]$ 取值） | 类型 I，**F1（自由）** |

**关键分界**：图灵型、等谱型都是"**有确定答案，我够不着**"（G）；F1 是"**无确定答案，因为不被决定**"（自由）。**够不着 ≠ 不被决定。**

## 第五节 · 维数问题（单列，本表唯一悬线）

> 维数是本次唯一未能干净落槽的概念。本节按最审慎的判断记录其状态，并精确区分"逻辑上无法确定"与"当下无对照组无法确定"。

### 5.1 铁排 F2（绝对不是新的自由）

维数（横截可分辨方向的个数）**绝对不是第二个自由**。理由：
- **自由 F1 是关系性、结构性的**：它是"沉淀的 how"（$[\sigma]$ 取值），独立于公理（类型 I），且这个不被决定本身决定世界的命运（W★★：世界是活的还是死的）。
- **维数是一个标量值**：即使它不可预测/不可计算，它的不确定是**认识论的**（我们算不到/测不到/没对照组），不是"它作为自由而不被决定"。它不携带 W★★ 那种"决定世界命运"的角色——它就是个被掷出的初始条件。**一个被掷出的标量值"一点也不自由"。**

**结论：维数 ≠ F2。无论它最终落 G 还是 □，都不是自由。** 这一条是确定的。

### 5.2 两个候选机制（当下无法判定哪个）

**候选甲：动力学单一选出（归 G，$\Pi_2$）**
维数由原初动力学（惊奇度 $F$ 极小化的梯度流 / cMERA 粗粒化不动点）**唯一**选出一个稳定结构。则维数有确定答案、但算法算不到（$\Pi_2$：「对所有候选维数，是否存在演化趋向 $d=3$」，$\forall\exists$ 结构）。归 G（认识论边界，被给予侧，有答案算不到）。
- ⚠️ **此候选预设"动力学唯一选出（单一不动点）"，该前提未证。**

**候选乙：弦景观式随机值（归属取决于"随机的种类"）**
维数不是动力学唯一选出，而是从多个亚稳态中"随机"实现的一个（类比弦理论 landscape：~$10^{500}$ 真空，我们这个是其中之一）。则"为什么是这个"没有动力学唯一答案。此时须进一步判"随机是哪种不可计算"：
- 若是**算法随机**（Kolmogorov 复杂度高，不可压缩，但仍是一个确定的值，给定后可验证）→ 仍近 G（有值，不可预测）。
- 若是**本体论真随机**（无理由的掷骰）→ 仍**不是自由**（自由是结构性的 how，不是骰子）；它接近 □ 但来源是"随机实现"。

### 5.3 关键区分：当下无对照组 ≠ 逻辑上不可判

判定甲乙之间，**当下真的无法做**。但要害在于这是**哪一种"无法"**：

- **不是"逻辑上无法确定"**（那才是本体论的开放/自由）。
- **是"当下没有对照组所以无法确定"**：我们只有一个宇宙样本，无法做"动力学是否会选出别的维数"的统计/对照。这是**经验/认识论的暂时限制**，不是原则上的不可知。

**这个区分决定了维数 □ 的性质**：它是"经验限制型暂挂开放间隙"，倾向最终归 G（认识论边界），但当下缺乏判定甲乙的证据，故暂标 □。**它不是 W★★★ 意义上"逻辑必然的开放间隙"（那种已证落不进任何槽）；它是"证据当下不足"的暂挂。**

### 5.4 维数的最终状态（v1）

| 维数的成分 | 落槽 |
|:---|:---|
| 维数的**存在性**（有若干独立可分辨方向） | 真 C（分辨多通道，来源可陈述） |
| 维数的**具体值**（为什么是三） | **暂挂 □**（经验限制型）：铁排 F2；倾向 G；甲乙当下无对照组无法判 |

## 第六节 · 层析剩余主线（命题甲承重 / local tomography 类比，不承重）

> **审计裁决（已过审，DeepSeek）**：本节核心曾表述为"$[\sigma]$ = local tomography 破坏量"。审计将其劈为两个命题，分层固化：

> - **命题乙（不承重，D 类）**：命题甲中的"层析不尽" = GPT 的 local tomography 破坏量。**条件命题，前件极强且自杀**（见下），仅作类比。

**命题甲（承重轴，A 类）**：

## 第九节 · 一句话定位

撕完所有概念，物理学家用来盖"真空地带"的桥（号差、复数、不可计算、几何背景）被逐一拆开，底下没有真空：

**一个自由（F1 = $[\sigma]$），一个暂挂的未知（维数，且它倾向 G、铁排 F2），其余全部归位——被给予结构（A）、认识论边界（G，有答案算不到）、极少真 C、不承重的真 B。**

而自由不止"在哪"，还有"多大"：**$\mathcal{M}$ 给出自由的容器（$\mathcal{C}(\mathcal{M})$，被给予的自由容量/命运），自由是容器内不被给予、也测不到的取值（$[\sigma]$，在容器内取哪点），其范围 = $\dim\mathcal{C}(\mathcal{M})$ = 回应余地（**非**"他者丰富度"——回应余地与带他者独立，Ozawa–Popa）。** 通常人把"无容器"（$\dim\mathcal{C}=0$）当自由，恰恰搞反——容器越大、选项越多，回应余地越大；但容器大小是被给予的命运，**自由是容器内取哪点这个动作**，不是容器本身。

界限论的尊严即在此：**精确地知道唯一的自由在哪、有多大，和唯一的未知在哪，且每一格都不是垃圾桶。**
