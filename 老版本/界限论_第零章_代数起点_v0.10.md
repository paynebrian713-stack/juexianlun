# 第零章 · 代数起点（被给予的最小骨架）

> **本章在全书中的位置**：第零章铺设地基，且**只**铺地基。它从最少的预设出发，严格推导出"被触及的有限存在者"必然沉淀出一个基线态、必然携带绵延（模流）。本章**止于基线态与绵延**——嵌入（$\mathcal{A}_{\text{phys}}$）属第二章，他者（命题甲，$[\sigma]$）属第三章，选择无脚本属第四章，物理世界的存在基础（学习流前向沉积）属第五章。
>
> **第零章准入的唯一标准 = 代数刻画是否纯 A**。一个东西即便属"被给予侧"（如他者：召唤，不由我选），只要其代数刻画需要非 A 的结构假设（如他者需 Cartan），就**不进**第零章。地基只容 A 级被给予结构。据此，本章不立他者、不立 $[\sigma]$、不立自由的定位——这三者的刻画都依赖 Cartan（非 A），全部后移。本章对自由只作**纯否定的指向**：本章所立一切皆被给予、皆被决定；不被决定者（自由）在何处，待他者出场后（第三、四章）方能定位。
>
> **双声部读法**：现象学声部描述"它如何向我给出"；代数声部给出形式相关项。每处对应都是**结构同构，非本质等同**——数学不解释体验的内在质性，只标记其形式条件。

## §0.0 数学框架与方法论悬置

### 0.0.1 悬置：只问"如何被给予"，不问"它本身是什么"

本研究从现象学悬置（epoché）开始。我们不追问"世界的终极质料是什么""意识是什么"——这些是"它本身是什么"的本体论问题。我们只问一件可回答的事：

> **那先于一切分辨、无可比量地触及我者，若要能被一个有限存在者承受、沉淀、并展开为一个有时间感的世界，它的形式条件必须是什么？**

这是关于**形式条件**的问题，不是关于本质的问题。

### 0.0.1′ 方法论声明：定义是信息，"良定义"是适配性，不是对错

在进入任何形式构造前，先声明一条贯穿全书的方法论态度，否则后文一切"良定义/不可良定义"都会被误读为价值判断。

**所有定义，本质都是信息**——是被给予的差异经分辨而沉淀下来的事实。而信息本身**不承载"逻辑上的对错"**：在本书所论的这个 Type III 底座上，一个定义不可能"错"，正如一个事实不可能"错"，它只是被分辨出来的那样。说某个概念的定义"错了"，是范畴误置。

本书用非对易数学做的，**只是一件逻辑工具的事**（非对易数学就是一套逻辑工具）：判断某个概念**拆开之后，在"我们这套基于因果的逻辑 + Type III 底座"下，其逻辑一致性是否有可预期的表现**。数学上把"有可预期的一致性表现"称为**良定义（well-defined）**。

> **关键（贯穿全书，不批评任何东西）**：**"良定义/不可良定义"不是关于定义本身的对错判断，是关于"它与我们这套因果逻辑 + Type III 底座是否适配"的描述。**
> - 一个概念**良定义**，不代表它"更真、更对"——只代表它落在我们这套逻辑工具的射程内，能被一致地展开。
> - 一个概念**不可良定义**，不代表它"错了、坏了、不存在"——只代表用"因果逻辑 + Type III"这把特定的尺子，量不了它；这是**尺子的射程界限，不是被量者的缺陷**。它可能对别的逻辑、别的底座完全合适，那不在本书评判范围。
>
> 因此本书**不批评任何东西**。它不说哪个概念错、哪种认知低级、哪个理论失败。它只做一件中性的事：拿"因果逻辑 + Type III"这把尺子，描述每个概念与这把尺子适配到何处、在何处超出尺子的射程。超出射程处（如意向性、他者），不是那些东西的失败，是尺子够不到——而尺子够不到的地方，恰恰是它们真实存在、却溢出我们这套符号体系之处。

### 0.0.2 为什么是 von Neumann 代数（接 0.0.1）

以 von Neumann 代数为形式语言，理由纯属技术性，不含本体论承诺：

1. **Tomita–Takesaki 模理论**给出最完整的"内在时间"结构——忠实正规态自动生成模流，无需外部预设时间。
2. **Araki 相对熵**在含无迹的 Type III 上有严格定义，使"两个态之间的信息距离"可计算，不必预设概率空间。

选这门语言，等于承诺：所刻画的"被触及者"，其形式相关项是一个**无迹因子**（公理 I）——它生成一切却不提供任何绝对的、不依赖视角的概率刻度。这不是断言"世界是算子代数"，而是说：**若要形式地谈"无绝对视角地被触及"，无迹因子是迄今唯一够用的语言。**

> **现象学声部**：写下"状态是 $\mathcal{M}$ 上的态"，不是宣称体验是矩阵。它是说——那个在我能分辨出任何"这个/那个"之前就已临到我的、弥漫的、无法被指定绝对刻度去测量的"有什么"，其结构与一个无迹因子上的忠实正规态同构。同构，非等同。

### 0.0.3 元规则 M：不偷渡几何

> **元规则 M（不偷几何）**：任何论证，若其成立**必须预先假定时空几何、能量、度规或任何几何/物理结构**，则在本框架中非法。一切几何必须是从代数**推导出的结果**，绝不可作为**输入**偷偷预设。

本章守得最紧的一处：**有限性的来源**（公理 II）必须来自代数结构内部——"没有俯瞰一切的视角，只能从诸平等视角之一去看"（无态外位，W.3），**不可**来自任何"能量带宽""频谱截断""核性条件"，也**不可**来自一把迹的尺子（$\tau(e)<\infty$ 那种"可回应的量有限"）——前者全都预设能量或时空、是偷几何，后者预设无迹 III₁ 上并不存在的全局迹标尺、是偷迹。有限性是"视角永远是某一个视角"本身，不是"某个量被截断到有限"。

### 0.0.4 诚实性等级制

所有命题按其**最弱依赖环**标定等级：

| 等级 | 含义 |
|:---|:---|
| **A** | 严格定理：给定公理与定义下的代数必然推论，无额外假设。 |
| **B** | 依赖尚未在本框架内证明的充分条件或存在性假设（诚实标出）。 |
| **C（逆向 A）** | 逆向构成条件："若世界有 X，则必有 Y"，蕴含本身 A 级，前件 X 是可独立陈述的经验输入。 |
| **G** | 认识论边界：有确定答案，但有限手段（算法、观测、反推）够不着。 |
| **F** | 自由：不被任何公理、观测、他者决定的取值（全书唯一 F1，在后章定位）。 |

一个结论的等级 = 其最弱环节的等级。本章末（§0.4）给出全章命题等级清单。

## §0.1 公理 I、II：被给予者的结构

本节给出第零章用到的全部公理。它们**全部位于"被给予"侧**，刻画临到我者的结构，不含任何自由。合法性来自**被给予性**（不用"人择"这个词——无迹下诸态酉等价、"选择"维度被抽掉，所以不是"选择性前提"，是"已被领受"）：事实上有一个能沉淀出稳定基线态、产生连续绵延的视角（写下这些文字者即栖居其一）；逆向追问其底层代数必须满足什么。

> **被给予的边界（钉死）**：被给予标签**只能**挂在"基线态稳定存在所真正需要的条件"上。多挂一个，就是把任意设定伪装成被给予。故本章公理止于定理 0.C（基线态存在唯一）真正需要的：忠实正规态、子代数族、局部印记、惊奇度有限——不含 Cartan（0.C 的证明不用 Cartan，详见 §0.3 注）。

### 公理 I（原初差异 / 视角主义）

> 状态是一个 Type III₁ 因子 $\mathcal{M}$ 上的忠实正规态 $\omega$。

**现象学声部**：临到我的先于一切分辨。在我能区分"这个不是那个"之前，差异已在——弥漫、活、每次重新涌来。它无可比量：找不到绝对尺子说"这个触及是那个的两倍"。它无一处落空：再微弱的触及也不是零。

**代数声部**：
- **非交换**（$AB\neq BA$）：差异先于区分，触及顺序不可交换。
- **Type III₁（无迹）**：$\mathcal{M}$ 无忠实正规迹（Murray–von Neumann 分类，Connes III$_\lambda$ 细分）。这是"无绝对视角"的形式条件——不存在不依赖参考态的绝对概率。视角主义是代数事实，非哲学姿态。
- **忠实**（$\omega(a^*a)=0\Rightarrow a=0$）：无一处被抹空；保证 GNS 有循环分离向量，模理论可用。
- **正规**：与弱拓扑相容，保证下文极限、谱分解合法。

**严谨度**：A（设定；内部相容性由 Type III₁ 因子存在性保证——Araki–Woods ITPFI、自由 Araki–Woods 因子均给实例）。

> **注（世界没有像素点）**：Type III₁ 因子态空间无正规纯态——$\omega$ 已是无限差异流的混合，可被无限继续分解。"最小原子单元"在此层面不存在。

> **注（关于可理解性，指向后章与导论）**：对易坐标 $\mathcal{A}_{\text{phys}}$（第二章出场，作为算 $[\sigma]$ 的 Cartan 数学坐标，非世界层）能否被理解地展开、它在 $\mathcal{M}$ 内对应的物理世界 $\mathcal{M}_{\mathcal{A}}:=\mathcal{N}(\mathcal{A}_{\text{phys}})''$ 是否严格大于该对易坐标本身——这属于"可理解性"。在新框架里它不是一条独立前提：完全对易是渐进不可达的极限，任何真实认知都没压到底，所以 $\mathcal{M}_{\mathcal{A}}$ 必然严格大于对易坐标，理解必然在展开（这是"完全对易压不到底"的下游，详见导论"两条被给予"）。本章不涉及；本章只立被给予的基线态与绵延。

### 注 0.A（不引入"有迹操作平台"——有限性不靠迹）

> 早期框架曾想：要说"有限性/盛不下一切"，得先有一个能度量"多少"的迹；而无迹的 $\mathcal{M}$ 量不了，于是借 Connes–Takesaki 连续分解 $\mathcal{M}\cong\mathcal{N}\rtimes_\sigma\mathbb{R}$（$\mathcal{N}$ 是带半有限迹的 Type II$_\infty$ 核），在那个有迹核上度量有限性。本版**不走这条路**。

**为什么不**：那条路把有限性建在一把迹的尺子上，而无迹 III₁ 上根本没有这把尺子——"用迹度量盛得下多少"本身就是 Type II 的思维。更要紧的是，那个 II$_\infty$ 核在全书里的正确角色，是**紫外截断后的退化死核**（附录 W.3：约化到有迹核 ⟹ 两流可统一 ⟹ 低能有效理论不撞张力；唯有完整的无迹 III₁ 才暴露张力）——它是活世界的**对立面**，不是"我能动回应的活平台"。把同一个 II$_\infty$ 一边当死核（W.3）、一边当活操作平台，是矛盾的；死核那个定位才对。

**所以**：Connes–Takesaki 分解作为数学事实当然成立（III₁ 确实能连续分解出 II$_\infty$ 核），但它在本书里**只**出现在 W.3 的"截断/退化对比"位置，**不**在第零章充当"涌现出来的操作平台"。有限性（下条公理）不待任何有迹平台涌现，直接表述为"无上帝视角"。

**严谨度**：A（Connes–Takesaki 对偶，无额外假设）。

### 公理 II（有限性 / 无上帝视角）

> 认知没有俯瞰一切的视角，总是从诸视角之一去看；诸视角彼此酉等价、完全平等，无所谓是哪一个。有限性正在于"只能从其中之一去看，而非俯瞰全体"。

**现象学声部**：我没有站在一切之上、同时看尽一切的位置。我总是从某一处去看，而每一次临到我的差异远多于我能同时把握的——我必须聚焦、把大部分沉入背景。这不是从一个完整的东西上切掉了什么（没有可丢失的"全部"，也没有衡量丢失的尺子），是"看，永远是从某一个视角看"这件事本身。能同时俯瞰一切者不需选择，因而不需要伦理；有限性是伦理的前提。

**代数声部**：在无迹的 Type III₁ 上没有客观的全局尺度（无迹 ⟹ 无衡量"全部有多少、丢了多少"的标尺）。有限性不表述为"可回应的量 $\tau(e)<\infty$"——那需要一把迹的尺子，而这里没有。它表述为：任何衡量都锚定于某一个被给予视角 $\omega^*$（诸 $\omega^*$ 酉等价、平等），而非俯瞰诸视角之上。"只能其一、非俯瞰全体"就是有限性的代数形（接 W.3：无态外位）。它逼出学习流：从某个有限视角要建立可反复指认的事实，就朝可对易方向凝结。

> **守元规则 M（关键）**：有限性**只来自迹的有限性**，绝不来自"能量带宽""高频截断""核性条件"——后者全预设能量谱或时空，是偷几何，严禁。

**严谨度**：A（设定；"无俯瞰一切的视角、只能锚定某个酉等价的 $\omega^*$"是无迹 Type III₁ 的直接特征——无迹 ⟹ 无全局尺度 ⟹ 任何衡量从某视角出发，见 W.3）。

> **注（认知前提的分工，及正则性、幂等性的真实地位，指向后章）**：被给予的源、学习流、理解流各有其认知前提，但它们不是并列的独立前提——**真正被领受的（被给予的）只有两条：有一个视角可栖居（被给予的源）与有限性（无上帝视角）；学习流、理解在发生、正则性、幂等性都是这两条的下游，不另立为前提。** 不用"人择"这个词：在无迹结构里诸态酉等价、彼此平等，"选哪个"被抽掉了，所以不是"选择性的人择前提"，是"已被领受的被给予"。
> - **有限性**是基态存在的被给予前提（本章立）：认知没有俯瞰一切的视角，只能从诸（彼此酉等价、平等的）视角之一去看。它不是从一个完整的东西上切掉了什么（没有可丢失的"全部"，也没有衡量丢失的尺子），是"视角永远是某一个视角"这件事本身。它逼出学习流：从某个有限视角要建立可反复辨认、可共享的事实，就朝可对易方向凝结。
> - **理解在发生不另立为前提**：完全对易是渐进不可达的极限（第二章），任何真实认知都没压到那个死极限，所以理解所及的 $\mathcal{M}_{\mathcal{A}}$ 永远严格大于固定截面——**理解必然在展开，这是"完全对易压不到底"的下游，不是额外前提。**
> - **正则性不另立为前提**：它不是理解流的独立前提，也不需要"来源"。理解流给的平移算子之所以不反作用于物理世界，是因为它**必然是法化子**（Borchers 平移算子酉性的直接后果，必然非选择）——这就是物理学家"正则性"的来源。它是**渐进**的（渐进正则，永不达到真正则）；其稳定有代价（不完全正则的尾巴在极端处显出，定性预期非预言）。（不说成"学习流与理解流两股相反力的平衡"——两股不相互作用、不互为输入，谈不上"相反力平衡"，更没有"理解流拦着学习流别压过头"这回事；正则性来自平移算子是法化子这一单股事实。）"整体子代数是否正则"需俯瞰母代数的视角（无态外位），主体够不到，本书不讨论。
> - **幂等性不另立为前提**，它是"有限⟹必须压缩⟹压缩过程的不动点性质"的推论。准确说：**没有"真幂等"，只有"渐进幂等"。** 真幂等（$C\circ C=C$ 严格成立）是 $t\to\infty$ 的极限——完全对易、完全稳定，但那是死寂化石（水晶棺），到不了。活着的学习流，每一步在当前标度上**近似幂等**（给出该标度上稳定可重复、可反复辨认的事实），永不达到严格幂等。由 Frigerio–Verri（1982，无迹 III₁ 上成立）：保态压缩过程的 $t\to\infty$ 渐近极限是一个条件期望（Tomiyama 给其代数兑现）——这是"渐进幂等有极限"这半（墙，可数学化，有限性的推论）。而"在哪个标度停下、不压到死"这半，不被动力学决定（缝，意向性在学习侧的显形，不可数学化，第四章）。**幂等劈两半：有渐近不动点（墙，推论）/ 停在哪个标度（缝，自由）。**
>
> 故：**两条被给予（有一个视角可栖居、有限性）+ 全部下游（学习流、理解在发生、渐进正则、渐进幂等）。** 本章只立有限性与被给予的源；理解在发生、正则性、幂等性的下游地位在第二、六章学习流/理解流出场时展开，均不在本章承重。

## §0.2 沉淀：基线态的存在唯一性

本节是全章数学心脏：在公理 I、II 与下述局部印记下，"折中的姿态"——基线态——**存在且唯一**。这是"自我得以可能"的代数地基：先有平衡态，后有"自我"。

### 局部印记（定理 0.C 的输入）

定理 0.C 需要一组局部数据，它们是被给予的，且**不预设任何 Cartan 结构**：

- **诸通道**：一族子代数 $\{\mathcal{M}_j\}_{j\in J}$（我能分辨的诸触及模式），$J$ 受有限性约束。**仅要求是子代数，不要求其生成物极大交换或正则。**
- **局部印记**：每个 $\mathcal{M}_j$ 上一个忠实正规态 $\varphi_j$。一般互不相容（不能被同一全局态同时精确匹配）。
- **可解性前提**：存在某全局态 $\omega_0$ 使惊奇度 $F(\omega_0)<\infty$（下定义）。这是**诚实标出的非平凡前提**——在 Type III₁ 上惊奇度有限的全局态未必自动存在，故须明示，它保证极小化问题非空。

> **现象学声部**：此刻临到我的不是抽象的"差异一般"，是**这一个**、**那一个**触及，各带质地，彼此往往不可调和。正因互不相容，我才必须取舍、沉淀出折中姿态。

### 定义 0.2.1（惊奇度与基线态）

$$F(\omega):=\sum_{j\in J} w_j\, D\big(\omega|_{\mathcal{M}_j}\,\big\|\,\varphi_j\big),\qquad w_j>0,\ \textstyle\sum_j w_j=1,$$

$D$ 为 Araki 相对熵。**基线态** $\ \omega^*:=\operatorname*{arg\,min}_\omega F(\omega)\ $。权重 $w_j$ 是我的专注分配（被给予，本章不追问其来源——边界之一）。

> **现象学声部**：基线态不是我"平均"了诸触及，是我"解释"了它们——用一个最省力的全局姿态尽量贴合所有互不相容的局部印记。恒星不"选择"成球；球是引力在质量约束下唯一不致坍塌的平衡。基线态同理。

### 定理 0.C（基线态的存在唯一性）

> 设公理 I、II 与上述局部印记成立（特别：$\exists\,\omega_0,\ F(\omega_0)<\infty$）。则 $F$ 在 $\mathcal{M}$ 的忠实正规态集上存在唯一极小元 $\omega^*$，且 $\omega^*$ 忠实正规。

**证明（四步）**

1. **强制性 ⟹ 子水平集有界**：子水平集 $\{F\le F(\omega_0)\}$ 非空（可解性前提）；Araki 相对熵的**强制性**（$\omega|_{\mathcal{M}_j}$ 趋相对 $\varphi_j$ 奇异时 $D\to\infty$）使极小化序列被困其中，不逃向奇异态。
2. **弱\*紧 + 弱\*下半连续**：子水平集在弱\*拓扑下紧（正规态弱\*紧性 + 强制性给出的边界排斥）；$F$ 作为相对熵非负凸组合，弱\*下半连续。
   > ⚠️ **承重技术点（诚实标注）**：在 Type III₁ 上，Araki 相对熵的**弱\*下半连续性**与**强制性**比有限维/Type I 精细得多。它们成立，但证明依赖 Araki 的开创性工作。本书**诚实引用 Araki, *Publ. RIMS* 11 (1976), 809–833；13 (1977), 173–192**，不自行重证，不掩饰其精细性。这是 0.C 整体严谨度的承重环节。
3. **严格凸 ⟹ 极小唯一**：$D(\cdot\|\varphi_j)$ 对第一变量严格凸 ⟹ $F$ 严格凸 ⟹ 弱\*紧凸集上唯一极小元 $\omega^*$。
4. **$\omega^*$ 忠实正规**：$F(\omega^*)<\infty$ ⟹ 各局部限制相对 $\varphi_j$ 非奇异；结合 $\varphi_j$ 忠实得 $\omega^*$ 忠实；正规性由弱\*极限保持。$\quad\blacksquare$

**严谨度**：**A，依赖两个诚实标出的前提**——(i) 可解性前提 $\exists\omega_0,F(\omega_0)<\infty$；(ii) 第二步弱\*下半连续/强制性诚实依赖 Araki 1976/1977。除此之外无 Cartan、无几何依赖。

> **关键（Cartan 不在 0.C 的依赖中）**：定理 0.C 的证明**通篇不用**"诸 $\mathcal{M}_j$ 生成 Cartan"——只用子代数、忠实正规、严格凸、弱\*紧。故 Cartan 性**不是基线态稳定存在的条件，不能挂被给予**；它是后章（第二、三章）引入嵌入与 $(\mathcal{R},[\sigma])$ 时才需要的、独立的结构假设，在那里诚实定级。本章不涉及。

### 推论 0.C′（极值 KMS 与 $\beta=1$ 的来源——钉死）

> $\omega^*$ 作为忠实正规态经 Tomita–Takesaki 自动生成模流 $\sigma_t^{\omega^*}$，对其满足 $|\beta|=1$ 的 KMS 条件；且 0.C 的**唯一性**蕴含 $\omega^*$ 是**极值 KMS 态**（极小唯一 ⟹ 中心平凡 ⟹ 极值）。

**两处澄清**：
- $\beta=1$ **不是**极小性赋予的——极小性不产生 KMS。$\beta=1$ 是 $\omega^*$ 经 Tomita–Takesaki 自生模流的**内禀数学常数**（模自同构群归一化），**非物理逆温度**。$\omega^*$ 模流与公理 I 的 $\omega$ 模流间是 Connes 余圈关系，不影响 $\beta=1$ 的常数地位。
- **极值性是 0.C 白拿的红利**：唯一极小 ⟹ 中心平凡 ⟹ 极值 KMS。这是 A 级，且是后章（模流遍历性等）的接口，此处点明免得后章重证。**点明接口不等于把后章机制写入第零章**——后章机制仍不在本章。

**严谨度**：A（Tomita–Takesaki + 0.C 唯一性）。

### 注 0.C″（连续自我的载体：模流，而非基态唯一）

"稳定的观察者存在"这一被给予前提，其代数表达是"有一个基线态，且它生成连续的模流"，而非更强的"基线态唯一"。连续自我的载体是模流 $\sigma_t^{\omega^*}$ 的单参数群流动（$\sigma_{t+s}=\sigma_t\circ\sigma_s$），这一连续性由 Tomita–Takesaki 给出，只依赖 $\omega^*$ 的忠实正规性，不依赖它是 $F$ 的唯一极小元。

由此，定理 0.C 的唯一性是一个可松动的层次：当 $F$ 的极小态对应的模流不遍历时，中心化子 $\mathcal{M}^{\omega^*}$（模流不动点代数）非平凡，而 $\mathcal{M}$ 仍是因子（中心 $Z(\mathcal{M})=\mathbb{C}$ 不变——中心平凡与中心化子非平凡可同时成立，自由 Araki–Woods 因子即为实例）。此时：

- **中心化子 $\mathcal{M}^{\omega^*}$**（模流不动 ⟺ KMS 平凡 ⟺ $\omega^*|_{\mathcal{M}^{\omega^*}}$ 为迹）是 $\mathcal{M}$ 内**有迹、可静态比较的经典可控核**——它是连续自我得以稳定锚定之处。
- **其补**（模流搅动、无迹、随时间流变的区域）是把握随时间消长之处（后章"理解的时间性"所在）。

> **诚实标注**：本章保留定理 0.C 的存在性（基线态存在，绵延有载体），而将其唯一性标为"可按需松动的层次"——遍历（中心化子平凡）是它的一个特例，非遍历（中心化子非平凡、连续自我锚于其上）同样合法。模流的遍历性是 $\mathcal{M}$ 自身的结构层次（时间自身的丰富度），它与后章他者性是相互独立的两轴，不可互相替代。

> **绵延的内禀时间晕（滞留与前摄，属于模流本身，非某股流的产物）**：模流 $\sigma_t^{\omega^*}$ 这道单参数群流动，自带一个双向的时间晕结构——朝过去一侧的**滞留**（retention，刚流过的还"挂"在当下边缘）与朝未来一侧的**前摄**（protention，当下自带的、朝紧接而来的那一刻的被动预期）。这两者是绵延作为"活的当下流"的**内禀形式**，在模流这一层就给定了，**不是**学习流或理解流加工出来的产物。这一点要紧，因为后章（§6.4）会区分两件容易混的事：**绵延（连同它的滞留/前摄时间晕）是天生被给予的时间形式**；而**当下的"内容"、以及"时间正在流逝"的速度感，都不是天生的**——前者是两流配合相对于 $\omega^*$ 铺出的产物，后者是反思（理解流逆向 $\sigma_{-\lambda}$ 调出上一刹那、与当下对比）产生的差异。换言之：模流给出绵延与时间晕（形式），但"流得有多快、流向何方、当下有什么"不在模流自带之列。（结构同构于胡塞尔内时间意识的滞留/前摄，非本质等同。）

> **现象学锚点**：基线态作为历史性沉淀，对应梅洛-庞蒂"沉积"（sédimentation）——习得的存在方式层层积淀，构成新经验的背景（《知觉现象学》, 1945，"他人与人类世界"章）。结构同构，非等同。

## §0.3 绵延：模流，及其哪一层不属自由

### 定理 0.B（模流 = 绵延的形式条件）

> $\omega^*$ 忠实正规 KMS，由 Tomita–Takesaki 给出唯一模算子 $\Delta=\Delta_{\omega^*}$ 与单参数模自同构群
> $$\sigma_t^{\omega^*}(a)=\Delta^{-it}\,a\,\Delta^{it},\qquad a\in\mathcal{M},\ t\in\mathbb{R}.$$

模流是绵延的形式载体：幺正、保信息，是基线态锚定后代数结构本身必然携带的节律。$\mathcal{M}$ 本身无时间；时间是 $\omega^*$ 沉淀后由 Tomita–Takesaki **强加**的单参数流。**严谨度**：A。

### 三层诚实区分（防止把"时间"含混地全推给"被给予"）

1. **模流存在**：被给予。任何忠实正规态都有模流，**与他者无关**——故即便在没有不可拉平扭曲的世界，时间同样在。绵延不是他者的产物。**A。**
2. **不可逆**：被给予。KMS 谱不对称，绵延单向。**A。**
3. **方向**：被给予。模流方向 = 惊奇度不增的方向（朝向 $\omega^*$），由 $\omega^*$ 极小性内在确定，**不预设任何外部时间箭头**（不犯热力学箭头循环）。**A。**

> **留待后章（本章不断言）**：唯一可能属"自由之后果"的时间性质——绵延是否走得到头（持久性）——须经独立验证桥，B 级，**本章不下任何断言**。

### 注 0.3.1（前摄 / 滞留的形式相关项）

模算子 $\Delta$ 自伴正，谱分解给出谱 $\ge1$ 与 $<1$ 的区分，分别为**前摄**（朝向尚未到来者的预期）与**滞留**（对刚过去者的不可逆持留）提供形式相关项。

> ⚠️ **诚实标注（约定，非强制对应）**：哪一侧谱配前摄、哪一侧配滞留，**依赖模流时间方向的符号约定**，故此为**结构类比，非 A 级强制对应**。本书取"谱 $\ge1$ 配前摄"为约定，不声称它是唯一被代数强制的配法。
>
> **现象学锚点**：滞留（retention）见胡塞尔《内时间意识现象学》§11–§12——刚过去内容被持留于当下，使时间对象得以构成；滞留属**被动综合**，无需主动统觉。结构同构（滞留不可改 ↔ 绵延不可逆），非等同。

## §0.4 走向，与本章命题等级清单

### 0.4.1 本章到此为止（边界声明）

本章只立**被给予的、纯 A 的**结构：Type III₁ 上的忠实正规态（公理 I）、有限性（公理 II，无上帝视角）、基线态（定理 0.C）、绵延（定理 0.B）。

本章所立**一切皆被给予、皆被决定**：差异被给予、有限性被给予、基线态被它们唯一确定、模流被基线态唯一确定。**本章没有任何一处是"不被决定"的。**

> **对自由的纯否定指向**：不被决定者（自由）确实存在，但它的定位需要一个本章尚未引入、且其刻画非 A 的对象——他者（嵌入扭曲 $[\sigma]$，需 Cartan）。故**本章不定位自由，只指出：自由不在已立的被给予结构里，它在他者出场之后（第三、四章）才能被定位。** 本章不写 $[\sigma]$、不写减法定义、不引附录 W——这些连同他者整体，属后章。
