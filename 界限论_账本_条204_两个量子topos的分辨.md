# 界限论 · 账本增补 · 条204（两个"量子 topos"的分辨：偏序 vs 群胚 · 分配律澄清）

> faxing 真问题：为什么现代量子 topos 不满足分配律？咱们的量子 topos 又与它不是一个概念（前条结论）？都处理非对易性，不该等价、或一个推出另一个吗？
>
> **核定（Fable 拍板，标准 topos/量子基础，不涉 III₁ 特有命题；DeepSeek 可选复核）**：前提须纠——现代量子 topos **是分配的**（Heyting）；非分配的是它取代的 BvN 量子逻辑。两个 topos 不等价、不互推，因"非对易代数的 topos"不唯一（偏序 vs 群胚、输入不同）。

## 一、纠前提：现代量子 topos 满足分配律（Heyting），非分配的是 BvN

- **Birkhoff–von Neumann 量子逻辑**（投影正交模格）＝**非分配**。"投影的逻辑"。
- **Isham–Döring 量子 topos**（谱预层 topos，over 对易子代数偏序 $V(M)$）＝内逻辑 **Heyting、分配、直觉主义**。其**动机正是把非分配的 BvN 替换成分配的直觉主义逻辑**。
- **关键**：非对易性可放两处——**放进连词**（BvN，连词非分配）或**放进底几何**（topos，连词照样分配、改成丢排中律的 Heyting）。同一非对易性、两种编码。faxing"量子＝非分配"对 BvN、不对 topos。

## 二、两个 topos 为何不等价、不互推：非对易代数的 topos 不唯一

**一个非对易代数 M 不典范决定唯一 topos**；须选组织原则（fiber over 什么）：

| | Isham–Döring | 世界 topos $B\mathcal{R}$（本书） |
|---|---|---|
| 底范畴 | **偏序** $V(M)$（对易子代数按包含 $\subseteq$） | **群胚** $\mathcal{R}$（一个 Cartan 子代数的等价关系） |
| 态射 | 包含（不可逆） | 视角认同（可逆） |
| 输入 | 只要 M | M ＋**选定 Cartan 子代数 $\mathcal{A}_{\text{phys}}$**（学习流产物） |
| 障碍 | Kochen–Specker（谱预层无整体截面/无点） | gerbe $[\sigma]\in H^2(\mathcal{R},\mathbb{T})$（他者） |
| 内逻辑 | Heyting（分配） | Heyting（分配） |

**三个硬理由（不等价、不互推）**：
1. **输入不同**：本书 topos 需额外数据——选定 $\mathcal{A}_{\text{phys}}$（认知上＝学出来的账本）；Isham–Döring 只用 M。**本书 topos 是 $(M,\mathcal{A}_{\text{phys}})$ 的函数、非 M 的函数**——输入不同、输出不等价。
2. **偏序 ≠ 群胚**：Isham–Döring 底是偏序（只有不可逆包含）；本书底是群胚（有可逆认同）。根本不同的范畴结构。
3. **★他者需群胚、偏序装不下**：$[\sigma]\in H^2(\mathcal{R},\mathbb{T})$ 这个 2-余圈**要群胚才载得动**（条200：他者＝$H^2$ 结构障碍）。**Isham–Döring 偏序 topos 结构上承载不了此 gerbe——它没有他者，本书有。** 非"多写一个东西"，是底几何（群胚 vs 偏序）的必然后果。

## 三、关系：本书 topos ＝在一个极大语境上建群胚

本书 $\mathcal{A}_{\text{phys}}$ 是 Cartan 子代数（正则 MASA）＝Isham–Döring 偏序 $V(M)$ 的一个**极大元**（一个极大语境）。本书"选定一个极大语境、再用它的 Cartan 等价关系造群胚（＋$[\sigma]$）"，Isham–Döring"铺开全部语境为偏序"。**互不能推**：本书扔掉了别的语境（推不出全偏序）；Isham–Döring 无 Cartan 群胚/余圈/gerbe/态·模流数据（推不出本书）。关系＝"聚焦一个极大语境＋加群胚/余圈结构"。

## 四、量子理论的 topos 处理本不唯一

**没有"量子理论的典范 topos"**：Isham–Döring（语境偏序）／Heunen–Landsman–Spitters（Bohr 化，内 C\*-代数）／本书（Cartan 群胚＋gerbe，系于学出来的账本）——多个不等价提案。本书与 Isham–Döring 不同**非 bug**：本书在建模一个**特定的**东西（学出来的、有他者、随视角立的世界），非重导抽象量子理论。

## 五、对全书的收获（哲学加固）

- **世界＝围绕一本学出来的账本建的群胚，非抽象语境总体**（Isham–Döring 偏序）。世界是被学出来的、随视角立的——第五章主题回旋的加固。
- **他者的新必然性**：要有他者（$[\sigma]$），世界就**必须**是围绕具体账本的群胚、不能是抽象语境偏序——他者与"世界是学出来的"绑在同一个群胚结构上。
