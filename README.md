# 《界限论》

[在线阅读 →](https://paynebrian713-stack.github.io/juexianlun/)

## 简介

全书围绕一个核心追问展开：**为什么"活着"这件事，注定要有"够不到的东西"？**

它不把哲学当意见，也不把数学当装饰。每一步结论都落在算子代数（Type III₁ von Neumann 因子）的可证事实上——无限性、无迹、非破缺、非超选择——然后用这些数学事实去接住"对世界的常识感是怎么来的""他者为什么不可能被公理推出""理解为什么天然有缝"这些现象学难题。

## 本地使用

```bash
# 修改源章节后，一键更新所有输出
python assemble_full.py

# 浏览器直接打开 docs/index.html 即可阅读
# (KaTeX 已内嵌，无需联网)
```

## 脚注系统

HTML 阅读器支持双向跳转：
- 正文脚注标记点击 → 跳到章末定义
- 定义处点 ↩ → 返回正文引用处
- 跨章引用自动跳到定义所在章节
