# 界限论 · 写作声部规范(总纲)v0.2

> 性质:这是**表达方式**的宪法,不是表达内容。后续所有文笔加工对着它走。
> 一句话:**数学只是骨架,现象学是血肉;凡有松弛处用主动综合的笔调,凡被给予处用领受的笔调;让读者认出"迹是自己主动性的结晶",从而主动面对、而非被动承受。**

---

## 原则一:TypeIII 的内在性——现象学心流不断

任何看着复杂的公式,本质都是第一人称感受。读者**不该、也不必**因数学而掉出现象学心流。

- **数学的唯一职责**:为现象学概念的**(非)对易性分析**提供代数基础。它架骨架,不当主角。
- **可执行规则**:
  1. 每处数学,旁边或随后必有现象学声部把它**翻回第一人称感受**;不许出现"孤立站着、没有现象学落点"的公式。
  2. **(非)对易性永远落成体验词**:活/死、可分辨/沉默、被给予/被对象化、上手/在手。
  3. 公式读完,读者的感觉应是"原来这就是那种感受",而不是"我得离开体验去算"。
  4. 内在性的本体论锚:**M = 亨利的绝对内在性(无外部),σ = 胡塞尔绵延,感受 = 非对易**。一切"外部"都是视角粗颗粒化的派生,不是真有个外面。

---

## 原则二:处处主动性——鼓动主动面对被给予,不被动承受

**不靠贴标签("这里有自由"/"gauge latitude"),靠语气的开关。**

- **按"有没有松弛"切换声部:**
  - **有松弛处** —— 视角 (𝒩, ω_ext) 的落定、世界剪法 W、解读、面对他者 [σ] 的截取 —— **用主动综合(aktive Synthesis)的笔调**:主动构成、担起、截取、解读。
  - **无松弛处** —— ω*(被给予的归一锚)、σ(被给予的律动)、几何、𝒩(被抛的因果位置)、感受质本身、表观规律 —— **用被给予 / 被抛的笔调**:领受、承接。
- **整体导向**:即便在被给予处,基调也是"认出被给予、从而能主动面对",**不渲染被动受难**。
- **松弛的本质(供内部对齐,不必入正文术语)**:唯一的非 σ、非几何自由度,是"面对二阶障碍 [σ] 如何截取局部截面"的**非典范性(余圈松弛)**;它真实(否则成严格决定论)、但无人行使(被抛占据,非自由意志)。**维数极小,但因无迹无刻度,其裁量范围是全部解读——"一丢丢即全部"。**

---

## 原则三:不批评,只说适用范围(含本书自己)——源自无迹

无迹 = 没有态外的客观刻度 = 没有俯瞰一切的位置。所以本书**不批评任何概念、任何人、任何理论,只陈述其适用范围**。

- 物理、逻辑、数学各有其载体(有迹 / 有最小单元),**在其范围内完全正确**。不写它们"naive / 漏看 / 失败 / 不假思索";要写,只写"它们的领域里不出现这道边界"——是范围,不是判决。
- **本书自己也有适用范围**:线性语言 + 因果光锥。它自己也只是 Type III 的一个截面、一份在某参考系下的报告,**不俯瞰**。任何把本书抬到别的框架之上的笔调,都偷设了一个态外位,删。
- 这**不是"要不要带刺"的风格选择,是结构上没有立场可带刺**(任何 MERA 都是 Type III 的一个截面,彼此平级)。一带刺、一显优越,就是偷了上帝视角(撞 W.3)。
- 落地:把一切"批判 / 纠偏 / 比谁高 / 谁终于看清了"的措辞,改成"X 在范围 R 内成立;出了 R……"。连"终于""真正""不假思索"这类暗含高下的词都清掉。

---

## 原则四:不说任何概念"是"某描述——只有参考系下的面值 / 报告——源自无迹 + 粗颗粒化

不说任何概念"是"某个数学公式,也不说它"是"某个文字描述。**任何数学或文字描述,都是某概念在某参考系下、经粗颗粒化后读出的一个可测量面值(报告),不是那概念本身。**

- 行文用"在某参考系下,X 读作 / 显现为 / 报告为 Y",**不写"X = Y""X 就是 Y"**。
- 这把"结构同构,非本质等同"**推到底**:连代数那一侧也不是"本质",只是另一份面值。模流不"是"时间;是"在某参考系下,内时间意识报告为模流的形式"。
- 因此"=""即"在正文里只用于**定义性的术语约定**(如§2.1.5"信息 := 可分辨性的产物"),且这种"="读作"本书在此参考系下如此命名",不读作"本质等同"。一切跨现象学—代数的对应,一律面值语,绝不写成身份。

## 贯穿枢纽:迹是主动性的结晶

- TypeIII **无迹** = 客观上无绝对刻度。但物理世界**看着有迹**(硬规律、确定事实)。
- **这表观的迹 = 主动性(视角的规范松弛被占据)经主体间约定(agree-on-A)凝结、沉积、匿名化(在手化)后的结晶。**
- 即:沉积 = 匿名化 = 在手化 = 把活的(主动的)冻成死的(客观的)。**你以为客观给定的世界,是你自己主动性的化石。**
- **"细拆两轴"的目的就在此**:精准指认——
  - **真被给予(无迹)**:ω* / σ / 几何 / 𝒩 / 感受质本身;
  - **主动性结晶出的伪迹**:视角 / ω_ext / W 的落定。
- 读者一旦认出后者是自己主动性的结晶,就能**主动重新结晶(换解读),而非被动承受**——这是全书把人从被动翻向主动的总枢纽,也是原则二的落点。

## 禁与守(写作红线)

- **禁**:说物理世界 / 基态 ω* / 模流 σ / 几何 / 感受质里"有自由 / 有能动 / 有灵魂"。对外(物理)对内(主观)都没有作者位置(W.3:无态外位)。
- **禁**:把非决定性 / 真概率当能动(随机 ≠ 控制;概率是历史一致性的产物)。
- **禁**:正文硬贴 "gauge latitude / 自由意志 / 能动性" 作主词。
  - 理由:物理学家——非标准词、且严格 gauge 需规范群(仅正则情形封顶,一般图景);哲学家——latitude ≈ leeway,会被误读成"本可不然"(本书恰否定 leeway)。
  - **gauge latitude 仅留作附录/脚注的数学锚点**,并标"正则情形封顶的规范结构 + 区别于 leeway-自由"。
  - 词源备注(可入脚注):gauge = Eichung = 标尺/校准;现代 gauge freedom = 无绝对标尺时选约定标尺的冗余自由——与"无迹⟹自选解读"精准同构。
- **守**:松弛处的主动性,**从第一页到最后一页无处不在**(因容量满),是贯穿的底噪,不是某章主题。
