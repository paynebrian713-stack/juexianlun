# 条 262 · 数学基础总架构：一个 III₁ ＋ 持续维持一本账 ＝ 四张脸

> **性质**：全书数学基础的收口。核心——算子代数、Borel、topos、谱三元 **不是四个并列地基，是一个地基的四张脸**。posited 的只有 $\mathcal{M}$ 与"持续维持一本账"这个过程；其余全长出来。本条为文学/各线的**总标准**，措辞封顶（含老会话五处撕定的修正）。

## 一、唯一 posited：一个 III₁ ＋ 一个持续过程
$$\boxed{\text{posited}=\mathcal{M}\ (\text{III}_1\ \text{因子·无迹})\ +\ \textbf{系统持续维持}\ \text{一本}\textbf{对易且弥散}\text{的账}\ \mathcal{A}\subset\mathcal{M}}$$
⚠️ **弥散是独立的 posit，不是长出来的**（第七章 [^takesaki] 逐字：弥散是**账本的构造级要求**、与对易正则并列，**不是 III 型自动给的**；反例 $\mathbb{C}p\oplus\mathbb{C}(1-p)$ 交换有极小投影而 $\mathcal{M}$ 仍 III 型；"III 型强制弥散"只在**账本极大**时成立，而活账本从不极大）。⟹ **对易推不出弥散、III 型也推不出弥散**；下面依赖树里的 $\mathcal{A}\cong L^\infty$ 是**这个 posit 的形式**（可分弥散交换 vN 代数的抽象同构），**不是从 $\mathcal{M}$ 导出的结论**。
⚠️ **但弥散与对易来路不同型，勿并追**（第八章 §8.4 已划定）：**对易有"谁"**——学习流施加的要求、账本被压向的理想（动力学产物、可追来源链）；**弥散没有"谁"**——无任何东西把账本推向弥散，它是**相容性条件**，理由是反证（原子账本＋遍历 ⟹ I 型死格 ⟹ $[\sigma]\equiv0$ ⟹ 连他者都容不下）。**故第八章来源链只追对易、把弥散单记一笔，是对的；把弥散塞进来源链会逼它长出一个它没有的"谁"。**
⚠️ **"维持一本账"不是瞬时 posit，是持续付功的过程**（条237/259附：维持＝代谢＝逆模流付功，Takesaki $\sigma_t^\omega(\mathcal{A})=\mathcal{A}$）。账不是取了就有，是一直维持才有。**这一字（维持 vs 取）决定本图是发生学、不是静态本体论。**

## 二、四张脸从 $\mathcal{M}$＋维持$\mathcal{A}$ 长出（依赖树）
```
𝓜(III₁·无迹·本体) + 持续维持 𝓐
  ├─ 𝓐 ≅ L^∞        ── 弥散基底（测度支 Borel）＝ 钝-代数〔⚠️ posit 的形式，非导出〕
  ├─ R̄ ＝ 商掉迷向的底  ── DST 支 Borel ──分类topos──▶ topos B R̄
  │     （活账本处典范但不完全·非 Cartan；FM 完全适用仅 Cartan 情形）
  ├─ 𝓛 ＝ 诸账空间     ── 账本间 Borel·wild
  └─ D_F(代数打桩)+D_M(模流转贷) ─▶ 谱三元 D ；  𝓜⋊R ─▶ II∞ 核（迹）
```
**读法**：topos 是 Borel 的下游（B R̄）；谱三元 D 是算子代数（打桩）＋模流（转贷）两半拼的；Borel 第一支（弥散基底）是第二支（R̄）的**底**；迹从交叉积 II∞ 核来。

## 三、四张脸分工
| 脸 | 管什么 | 是 $\mathcal{M}$ 的哪一面 |
|---|---|---|
| **算子代数**（$\mathcal{M}$/模流/余圈） | 有什么·怎么演化·缝·$[\sigma]$·两态 | 本体自己 |
| **Borel**（三支·两 branch） | 分类难度·走到哪本账·维数盲·表能力·$[\sigma]/K$ 的家 | 维持账后的可测结构 |
| **topos**（$B\bar{\mathcal{R}}$） | 哪些命题真·某物·无点·Heyting | R̄ 的逻辑面 |
| **谱三元 $D$** | 多远·几维·多久 | 打桩＋转贷拼的度量 |

## 四、贯穿全图的切线：无迹 ⟹ 无俯瞰 ⟹ 内蕴免费 / 绝对付费
$\mathcal{M}$ 无迹 ⟹ 主体注定"无俯瞰"（第一人称·无态外位）。这条命根把全图切成两半：
$$\text{内蕴 / 态-相对}\ \textbf{免费}\quad\big|\quad\text{绝对 / 俯瞰}\ \textbf{借来·付费}$$
⚠️ **但付费半不是一刀切、内部三债不同**（图形层最易焊错，钉死）：
$$\underbrace{\text{Borel-可测}}_{\textbf{不欠}(\text{本分·维数盲})}\ \ /\ \ \underbrace{\text{topos}}_{\text{欠}\ \textbf{参考态债}}\ \ /\ \ \underbrace{D\ /\ \text{绝对单位}}_{\text{欠}\ \textbf{态外位/迹债}}$$
**免费半**：缝、$[\sigma]$的存在、两态、模流（序）、可测（②本分）、逻辑结构、给定$\omega$的模流时间（态-相对）。**"内蕴免费/绝对付费"是全书那张脸的反复显形**（钝的形/尺、③序/④尺、核对/造尺、给定$\omega$模流时间/绝对物理单位——同一张脸）。

## 五、五处封顶要点（老会话撕定，务必守）
1. **topos：存在免费、内容不免费**。$B\bar{\mathcal{R}}$ 构造上由 R̄ 白给（不用额外 posit）；但**无点/某物 $F(1)$≠完整对象（缝）/Heyting 是 topos 层的新事实，Borel 层说不出**。⚠️ **不得叫 topos"影子"**——命根"无点≠无某物"只在 topos 层说得出，降格为影子就抽掉它的位格。免费的是 topos 的**存在**、不是它的**结论**。
2. **R̄ 非 Cartan**。R̄＝**商掉迷向的底**，活账本处**典范但不完全（非 Cartan）**；FM"正规化子轨道等价关系"完全适用**仅 Cartan 情形**。⚠️ 活账本**非极大**（极大＝$K{=}1$＝死点，正是书要拒的）——依赖树不得默认 Cartan。
3. **分析者俯瞰合法、但不免费**。数学家事后提"整体 wild 等级"合法（反身纪律 W.6.1.6），**但仍是账本内落盘**（落盘≠免费）；整体等级须标"R★-类/第一人称够不到的全局"或改**否定式**（"不存在 $\mathcal{L}$ 到可数不变量的 Borel 满射"＝内蕴地说无地图）。⚠️ **全书不认任何免费俯瞰位，包括数学家的**。
4. **三债非一刀切**（见§四）：Borel-可测不欠、topos 欠参考态、$D$ 欠态外位/迹——boxed 二分须带这一句，勿在图形层压成一刀切。
5. **维持非瞬时**（见§一）：账靠持续付功维持，非取一次就有——发生学、非静态本体论。

## 六、降不下去（结构性、核心逼出，非工具没选好）
- 无俯瞰 ⟹ 谈**分类难度**（要俯瞰比较复杂度）必外接会俯瞰的工具＝**Borel/DST**（算子代数、HoTT 都拒俯瞰、补不了）；
- 无迹 ⟹ 谈**绝对度量/单位**（距离/几维/秒）必外接一把尺＝**谱三元 $D$**（付态外位；迹从 II∞ 核来，时长/熵/距离同欠一笔迹债、只到缩放）；
- topos 存在免费搭在 Borel 上（内容不免费）。
$$\boxed{\text{最小配置}=\text{算子代数}(\mathcal{M})\ +\ \text{Borel}(\text{分类/俯瞰·含弥散基底})\ +\ \text{谱三元 }D(\text{度量·借来});\ \ \text{topos 存在免费}}$$

## 七、守的分辨（本条不重述，指针）
- **三个 Borel**：弥散基底 $\mathcal{A}$（测度支·账本内-base）/ R̄（DST·群胚）/ $\mathcal{L}$（DST·账本间·wild）——第一支是第二支的底；限定防混定理（Maharam＝测度支、Kuratowski/turbulence＝DST 支）、非否认关系（条258 附/方法论）；
- **两个 topos**：$B\bar{\mathcal{R}}$（自用·有某物）/ 谱预层（引用·KS 无全局截面）——不得并列（否则顶掉"无点≠无某物"）；
- **钝二根**：弥散（测度支·不欠）/ 无刻度（Connes·欠）——骑态外位分界两边（锐钝文档）；
- **打桩两角色**：立账投影（喂引擎·参考态-无关）/ $D_F$-几何（度量骨架）——历史一致性用前者、不循环（条259附）；
- **D 对不对称**：$D_F$ 参考态-无关 / $D_M$ 参考态-依赖（条259附）。
- ⚠️ **舞台 $\mathcal{M}_{\mathcal{A}}=\mathcal{N}(\mathcal{A})''$ 不继承公理 I**（W.1.3.1 逐字：**一定是子代数**〔双对易子定理白给〕；**不一定因子**；**不一定 III₁**〔型未定，可能半有限甚至有迹〕）。⟹ **凡要用"无迹／因子／III₁"的论证，只能在 $\mathcal{M}$ 上做、不能在舞台上做**（附录守此：W.7 开放1 护栏"有迹的是舞台非世界"）。⚠️ **两处已冒头**：(a) `[^nocenter]` 写"$\mathcal{M}$ 因子 $\Rightarrow Z(\mathcal{M}_{\mathcal{A}})=\mathbb{C}1$"缺**生成**前件；(b)"$\mathcal{A}$ 在 $\mathcal{M}_{\mathcal{A}}$ 里正则"是**恒等式**（$\mathcal{N}_{\mathcal{M}_{\mathcal{A}}}(\mathcal{A})=\mathcal{N}_{\mathcal{M}}(\mathcal{A})$），内容不在"正则成立"、在"工作在舞台而非世界"这个选择（由 R★ 承担）。⚠️ **但别把这条挂错账**：舞台是 R★／FM 定的，正则只是其副产品——代价是"在舞台上工作"的价签，不是"白拿正则"的价签。
  ⟹ **纪律不是"这三项不能用"，是"每条性质都要逐条问传不传"；判据如下（两秒可查）**：
  | | 例 | 为什么 |
  |---|---|---|
  | **向下传**（$\mathcal{M}\Rightarrow\mathcal{M}_{\mathcal{A}}$） | 可分／可数生成 | 子代数不会"更大"（可分 Hilbert 空间上的 vN 代数，预对偶仍可分）——**白给，但白给的步也是步** |
  | **不传** | 因子性 | $Z(\mathcal{N})=\mathcal{N}\cap\mathcal{N}'$ 可以**更大**（$\mathcal{N}'\supseteq\mathcal{M}'$） |
  | **不传** | 型（III₁／无迹） | III 型因子里装得下 II₁ 子因子 |
  ⟹ **一句判据：「小」的性质（基数／生成度）向下传；「结构刚性」的性质（中心平凡、型）不向下传。** ⚠️ 故某一步"这三项用不上"**不等于"检查可略"**——是**命中项为零、检查照做**（说"空转"会给下次的跳过发许可证）。实例：第五章"可分 ⟹ 可数标准 Borel"那一步，因子／型两项落零、可分一项落一。

## 九、给读者的投影（四域表·文字版；SVG 待落点确认，勿先画）
⚠️ **262 的技术树（§二）是内部地图，不给读者**（管道非风景、会犯"机器前置"的进步主义病）。但它逼出一张适合读者的表——**同一件事四种问法，每种沿同一条线裂开：局部相对的你有，绝对整体的你借**：
| | 你够得着的（内蕴·局部相对） | 你够不到的（绝对·整体） |
|---|---|---|
| **空间** | 谁挨着谁 | 隔多远 |
| **时间** | 先后、方向 | 多少秒 |
| **逻辑** | 有个"某物"（局部拼得起） | 拼成完整无缺的整体（总差一道缝） |
| **分类** | 认得出彼此 | 一张完整的地图 |
⚠️ **两格硬约束（老会话撕定，画/写前必守）**：
- **逻辑·够不到 ＝ 完整无缺的整体（缝，$F(1)$≠完整对象）**，**绝不写"视角无关的真"**——代数事实（对易/包含/$[\sigma]$）恰恰视角-无关、客观、所有视角公认（反相对主义的墙），把"客观真"放进"够不到"是硬伤、会把全书读成相对主义。且**不写"无全局截面"**（那是谱预层 KS、非 $B\bar{\mathcal{R}}$，会混两个 topos）。
- **分类·够不到 ＝ 一张完整的地图（没有地图＝无典范分类/wild）**，**不写"绝对的难易序"**——那会让读者以为"有个绝对序、只是够不到"；wild 是"根本没有地图"，非"有地图但够不到"。
⚠️ **三条呈现约束**：①不带技术标签（写空间/时间/逻辑/分类，不写 topos/Borel/谱三元）；②晚放（§9.5 或收尾，读者走完现象学旅程后·回望位），不前置；③与路线图分工——路线图＝旅程图（动态·走一遍），此表＝收束图（静态·回望"四处是一回事"），别重叠。**SVG 待两格落定＋落点确认后再画（图形层焊比条目层焊危险）。**

## 八、诚实与开口
- **承重（本条封）**：四张脸从一个 $\mathcal{M}$＋维持$\mathcal{A}$ 长出；无迹⟹无俯瞰⟹内蕴免费/绝对付费；三债；最小配置降不下去；
- **图景桥（勿焊定理）**："四张脸""内蕴/绝对同一张脸多次显形"是组织性对齐，非单一定理的多推论；
- **开口**：$\mathcal{A}_0$ 非平凡押 7b；维持代价定量接 Landauer；活账本非-Cartan 正则配对的 III₁ 良定义（DeepSeek 靶）。
