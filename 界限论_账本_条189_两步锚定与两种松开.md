## 甲 · 纪律:论文无"同一者"原语
"同一者/我"是伪概念——只有模流流动(绵延)+他者必被响应(余圈)+两步锚定动作。**"我"=两动作合成物,非实体**(接条169 占位符/条188 被攥的一道差异)。全书无"同一者"原语。

## 乙 · 两步锚定(拆原单步 J3,账上新增)
- **第一步·锚定基态**→凸凹场 + 前反思不动中心(基态相对自己不动=同义反复,**尚未主题化为同一性,无我他之分**)。凸凹=基态自带(条183 模谱)。
- **第二步·引入参考态**测跨时同一性(=历史一致性 $\sigma_{-t}$ 搬运,条183)→**参考态=尺子=测量立场="我"**/被测流动="他者"。
- **我他之分生于第二步,非第一步;历史一致性=第二步的启动**。大部分人两步一次完成,故体验里直接是"我经历他者",看不出中间层。

## 丙 · 两种松开(核心,区分活空/绷空)
| | 松参考态(松第二步、保基态) | 松基态(松第一步、全松) |
|---|---|---|
| 保留 | 基态在(凸凹在) | 全松→原初差异满溢 |
| 状态 | 有凸凹无我他,绷着防同一性 | 无我无参考,差异满溢 |
| 响应他者 | **无法响应**(响应需我他之分,被抑制) | **自在响应但不认领**(差异互动照常,无我登记) |
| 用力 | 反向用力(防定义同一) | 松开(不用力) |
| 视角 | 维持态视角、无对象 | 认出算子视角=无固定基态(条182) |
| 味道 | 刻意止/无念,断响应 | 自在,响应而不知在响应 |
- **两种"空"判别(修行歧路警示,同型旁注非教义)**:绷着的空(松参考态,断响应)≠活的空(松基态,响应自在);压念≠真松。
- **响应他者=凸凹主动变化=差异互动**:有我时经"我响应他者"登记(费力)/无我时直接是差异互动(自在)。

## 丁 · 语言到头处(诚实标)
第一步的"前反思不动中心"在语言上游:描述它需第二步(概念化/引入参考态),而它在第二步之前⟹**说出来就已定义它、已引入参考态**。盲区引理形(条131),非能力不足——对象在"说清"这个动作的构成条件的上游。

## 戊 · 落点
- 第零章锚定:J3 拆两步(基态锚定/参考态引入),我他之分生于第二步。
- 第五章历史一致性:=第二步(同一性引入=参考态选定,接条183)。
- 尾声/修行现象:两种松开之别(活空 vs 绷空),同型旁注。
- 第二章意向性/第四章他者:我他之分非本有、生于第二步,回改相关表述。
