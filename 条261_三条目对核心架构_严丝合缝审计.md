# 界限论 · 三条目（258/259/260）对核心数学架构 咬合审计（为第五章打地基）

> **⚠️ v2 改判（老会话再审 + 数学线自认）**：v1"无一处顶撞、严丝合缝"**过强、犯方法论病**——任务设定（"严丝合缝"+"无一丝错位"）让 v1 只找合不找缝、为整齐**硬凑**。**改判：大体咬合，三处错位（已修）**：①三根↔三源第三对硬凑（KS≠撞他者/荷，两个不同层）；②表二"空间化天然分荷/他者"与 T2"不可判定"内部顶撞（本体/认识没分）；③"客观身体=换锚不变者"混装封与开当地基。**"严丝合缝"四字降级**——真正咬合的系统不需宣布自己咬合。

> **审计目标**：建侧三条目必须与原核心架构**无一丝错位**——核心＝**无迹** + **三个不可知**（横向/参考态 $\omega_{\text{ext}}$、纵向/未来 $\sigma_{+\lambda}(\mathcal{N})$、脚手架/断言板 $D$）+ **对易现实撞他者/荷**。
> **对第五章尤关键**：**横向（参考态）的完整戏份就在第五章（历史一致性）**（第三章 3.1 逐字：横向的锚→第五章）。三条目必须为它备好地基。

## 一、核心架构基准（逐字核准，第三章 3.1/3.4）
- **无迹**（根）：无态外位、无绝对尺；显形为**禁格**（可居+典范那格空着＝无迹）。三个不可知共同来源。
- **三个不可知**（沿"可居/典范"两坐标）：
  - **横向＝参考态锚 $\omega_{\text{ext}}$**：可居、**非典范**（随 Connes 余圈漂移，无天生原点）。→ **第五章（历史一致性）**。
  - **纵向＝未来 $\sigma_{+\lambda}(\mathcal{N})$**：无物、不可居。→ 第六章（逻辑）。
  - **脚手架＝断言板 $D$**：典范可构想、**不可居**（外部对象化）；维数是其上一层。→ 第七章。
- **对易现实撞他者/荷**：账本（对易）碰到不可拉平者 → **荷**（可吸收＝纤维迷向 $K$）或 **他者**（$[\sigma]$，不可吸收）；**哪个是哪个不可判定**（需知 $\mathcal{B}$/舞台，而 $\mathcal{B}$ 要态外位＝无）。完全性＝判定停机＝死点。

## 五、一句结论
**三条目与核心架构大体咬合，三处错位已修**：前两对根源对应（无迹↔三不可知、嵌入↔R★）真；第三对（非对易→KS+荷/他者）是**两 manifestation、非一一对上**（撤硬凑）。为第五章：横向/参考态（定性已备、公式开）、topos、可交流三块地基（标清封/开）；T1 + T2（本体有别/认识不可判，已落 260 §3.5）补齐。
> **方法论记一笔（老会话点破）**：自我审计报"完美"时，最该被再审的就是那个"完美"。v1 为"严丝合缝"硬凑第三对、collapse 本体/认识、混装封/开——三病同源于"任务要我找合"。**此后"严丝合缝"降级使用**；框架（基准表+逐条核+T1/T2）保留、可作第五章前标准动作，但结论只报"咬合/错位几处"，不报"完美"。

## 六、方法论纪律（八条·从本审计及其后各轮现场撕出）

> **性质**：以下八条不是数学结论，是**工序纪律**——每一条都对应一次真实发生过的错（多数是本会话或老会话当场犯的）。它们不进正文，是防正文漂的护栏。

①**交接稿压缩＝焊接高发区**——凡"本书自用"与"本书引用"的对象，压缩时必须保持标注、不得并列（本轮 $B\bar{\mathcal{R}}$/谱预层即此类）；
②**同名分辨做完后，凡再提该名必带限定**——**三个 Borel**（$\bar{\mathcal{R}}$ 群胚·DST / $\mathcal{L}$ 账本间·DST / 弥散基底 $\mathcal{A}\cong L^\infty$·测度支·账本内-base）、两个 topos（$B\bar{\mathcal{R}}$/谱预层），提名即带限定。⚠️ **但限定是防混用定理、非否认结构关系**：第一支（测度/弥散基底）是第二支（DST/$\bar{\mathcal{R}}$）的**底**（R̄ 建在 $\mathcal{A}$ 的谱上，Cartan/正则情形）——引定理时分清哪一支（Maharam＝测度支、Kuratowski/turbulence＝DST 支），但结构上第一支是第二支下层，该用就用（勿因守"别混"而不敢引底层）。
③**借例子 ≠ 借框架**——借一个现成结果当例子（一次性、不带预设进来）可以；借整套框架当地基（会把它的纲领预设一起拖进来、顶书的核心）不可。典型：可借"非构造存在"这个例子，不可借 Curry–Howard/BHK/HoTT 的构造主义框架（拖可构造性预设、顶不可判定核心）。
④**工具层 ≠ 内容层，不得互冒**（同名近形防线，同两个 Borel/两个 topos）：参数化（书写 $\sigma_t$ 用的 $\mathbb{R}$）、外接结构（交叉积的轴与迹）属**工具层**；系统内部可见证的量属**内容层**。工具层为真（"$t_2-t_1$ 有定义""交叉积后有迹"）**不得读成**内容层为真（"$\mathcal{M}$ 内部见证等长""世界自带尺度"）。判据：问单位/原点由谁钉住——"我们写的/外接的"＝工具层，"系统内某物"＝内容层。⚠️ **但"工具层"≠"arbitrary"**：给定 $\omega$ 的模流时间 canonical-given-$\omega$（KMS 钉死、非随便选），只是"不在 M 内见证、无物理单位"——态外位债在"绝对物理单位"（III₁ 钉不住）、非在"任何 $t_2-t_1$"。（防两向过读：既非"内部给不出时长"，亦非"$t$ 外部强加"。）
⑤**未入框架者：可指认空位、不可安放**——凡要说"X 欠某笔债／属某一层／在某个位置"，先查 X 有没有进框架（它在 $\mathcal{M}$ 里是什么）。⚠️ **界线（否则与"限制性结论"打架）**：对未入框架的 X，**可说**"框架给不出 X 的数值"（**限制性**·关于框架的能力边界·不需 X 在场·合法）；**不可说**"X 落在第④层"（**安放性**·关于 X 的位置·需 X 在场·非法）。同族前例：$\Theta$ 未定义却盖成非典范、$\mathcal{L}$ 未严格却当截面用、$G_\Theta$ 未定义却定义客观身体、"多费劲"（现象学）安成"能量"（物理）、未入框架的质量安成"欠某笔债"。
⑥**说"框架给不出 X"时，必须指明是哪一种给不出**（两轮内三次同族错、两会话各犯，故升为一等纪律）：**①开口**（还没做·**价格**·将来可能可答，如维持代价定量接 Landauer）／**②纪律或立场**（不做·如 W.6.4 descriptive 非 prescriptive、尾声逐字不给规则）／**③结构上不可能**（**墙**·如无迹 ⟹ 无绝对尺）。⚠️ **两条附则**：(a) **不得抓一个不对题的结论当墙**——量纲论证挡不住**无量纲**的比（配比是两代价之比）；"例级不可知"（讲的是因子的分类不变量）挡不住**相对量**；(b) **同一个问题可能一半是价格、一半是纪律**——如"配比"：描述性那半押代价函数（开口/价格），规范性那半押尾声与 W.6.4（纪律，非开口，做完 Landauer 也不开）。**混报三者，或用不对题的结论封口，是本轮反复犯的错。**
⑦**记诚实的方式是"承重点定位"，不是"承认可能错"**（老会话本轮立）：写下一处脆弱性时，不要写"这条可能是错的"，要写"**这一族结论唯一挂在哪一根钉子上；那根钉子倒了会带塌哪些**"。⚠️ **定位要精确到步**（如：时长五步链第 1、2 步对整个 III 型成立，III₁ 专进来的是第 3 步权流分类 ⟹ 承重点是"权流平凡"这一个条件）。⚠️ **两条附则**：(a) **承重点须是框架内可陈述的对象**，不得跨到物理去（"世界自带最小时间刻度"跨了，且犯⑤；"$S(\mathcal{M})\neq[0,\infty)$"没跨）；(b) **区分"posited 的假设"与"可查证的条件"**——公理的一半是前者，写成后者会把一个赌注说成一次体检。
⑧**recap 双向纪律：不得领先、也不得滞后于被 recap 的东西**（老会话补全）。**不得领先**：上游还没落定就先写汇总，写出的是指向空气的 recap（同"撤图留死引用"的镜像）。**不得滞后**：上游一落定，**立刻把所有 recap／汇总处列进待核清单**——recap 是全书最大的一处"引用"，而附录反复犯的"改口径没核引用处"（方法论第 11 条）正是它的小尺度版。⟹ **完整工序**：先落上游 → 上游落定的同一轮把 recap 列进清单 → 清完再发版。⚠️ **适用范围**：§8.4（发生学链）、§9.5（够不到/表墙）、W.0（三定理与依赖图）、十二步表、承重状态一览——这五处是全书的 recap 面，任一上游变动都要过它们一遍。
⑨**分辨要跨线登记，不只进它自己的条目**（老会话本轮立；⑧的**线际轴**版本）。⑧管的是**时间轴**（同一份文档、跨版本没核引用）；本条管**线际轴**——**一条分辨在一条会话线上撕定了，在另一条线上会被当成全新发现**（实例：「$\mathcal{M}_{\mathcal{A}}$ 不一定因子」两轮前修 `[^nocenter]` 时已落，本轮在另一条线上作为"稿子完全没提"重新冒头）。⟹ **工序**：凡一条分辨被撕出来，须**同时**进 (a) 它所在的条目 与 (b) **分辨清单**（条262 §七）；**缺 (b) 就会重新发现一次**。⚠️ **分辨清单的作用因此不是"备查"，是跨线同步的载体**——它是各线共用的那一份，条目是各线各自的。
