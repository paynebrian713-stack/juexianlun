## 0. 读表须知:现象学与物理学方向相反

同一道梯子,两门学科反向爬:
- **物理学的"看见" = 测量 = 把非对易逼成对易经典读数 = 逼进 A_phys(MASA,死寂极限坐标)**。越看清,越死。
- **现象学的"被给予/活的感受" = 测量之前那团非对易**。本源在深层(非对易浓),物理学够不到的恰是它最看重的。

一句话:**深 = 活 = 非对易 = 现象学本源(不可测,物理学够不到);浅 = 死 = 对易 = 物理学可测(已对象化,现象学视为派生与遗忘)。** 下文"深度"按现象学方向(深=活)排。

**因果方向也反(全文反复出现的拧点,§7bis 钉死)**:现象学"被给予在先 → 形式反推";数学"定义在先 → 导出在后"。二者常指向同一双射的两个读向,不是矛盾。

## 0bis. ω_ext 的正确刻画(v0.3 地基修订,须 faxing+审计核)

**ω_ext 是真实的 III₁ 态,不是"虚构/非实存"。** 之前措辞糊,把两件事压成了一件,现拆开:

| 对象 | 存在性 | 性质 |

## 1. 桥接公设(地基,faxing 已认)

**公设一(感受 = 非对易,带通版)。** 感受/可辨认信息 = **光锥内、模频率落在一个带通窗口里的非对易性**。
- **下限**:过于对易 → 非平凡性不足、不充实、落回中心化子 → 没信息量(死)。
- **上限**:过于非对易 → 意向指不到、分辨不出、溢出因果窗口 → 也没信息量(够不到)。
- 故它**不是单调量,是带通(band-pass)**:两端都归零,活的对比在中段。一直找不到公式,正因为在找单调量;候选公式见 §4。

**公设二(对象化 = 拍平进对易)。** 粗颗粒化 = 把非对易往对易拍平的梯度。对象化/在手化 = 深层提前拍平,感受结晶成死物。他者 = 拍到无穷深也拍不平的非对易残差(§7、§8)。

**横向定位(补单一资料):带通两端是两家管的,不是一种"空"。**
- 下限端(料不够,频率→0)= **胡塞尔空意向沉到晕、再到沉默**。
- 上限端(料太多/太远,频率→∞)= **列维纳斯他者 + 康德崇高**(收不进来)。
- **两端是反向的两种空**(一种料不够、一种料太多)。中段"既分得出又没被拍死"——**没有任何一家现象学有干净的家,这是本书原创落点**:不是补一个洞,是补两家之间没人住的中段。

## 2bis. 视角的双坐标:公式 (𝒩, ω_ext) ↔ 现象学 (U(s), u_t)

视角一个对象,两套坐标。**注意(审计 R6 裁定):这不是代数双射,是"派生/显影"对应,落图景层**——见下"精度"条。

| 坐标 | U / 第一分量 | u / 第二分量 | 用在哪 |
|---|---|---|---|
| **公式坐标** | 𝒩(因果窗口,结构视图) | ω_ext(参考态) | 数学 / 审计(封顶记法) |
| **现象学坐标** | U(s)(理解的活展开=因果平移,体验为筹划/时间流,动力学视图) | u_t(余圈=习性/风格/分类倾向,让东西凸显的倾向) | 现象学行文 |

- **映射的精度(R6 qualify,图景层)**:
  - u_t ↔ ω_ext:**A 级双射**(Connes 余圈定理)。
  - 𝒩 → U(s):**A 级涌现但非单射**(Borchers)——**不同 HSMI 子代数可共享同一平移群 U(s)**,故 U(s) 反推不出 𝒩;且 U(s)𝒩U(s)* ⊊ 𝒩(U(s) 不保 𝒩,只严格内移)。
  - 故 **(U(s), u_t) 信息不足以唯一定出 (𝒩, ω_ext)**;**缺件 = 相对交换子 𝒩'∩𝓜**(区分共享同一平移群的不同 HSMI 的结构指纹)。完整结构数据 = **(U(s), 𝒩'∩𝓜, u_t)**。
  - **裁定**:(U(s), u_t) 是视角的**动力学显影/派生对象对**,非代数等价替换;**公式必写 (𝒩, ω_ext),现象学行文可用 (U(s), u_t),标图景层**。
- **为何现象学用 (U(s), u_t) 更忠实**:经 §0bis,𝒩 随 σ 展开、ω_ext 随余圈漂移——两分量本是过程;(U(s), u_t) 把它们显示**为过程/倾向**,而非"一个窗口+一个态"。正对"现象里没有参考系、没有子代数,只有分类倾向和同一性概念"。
- **"同一性概念" = A 极(agree-on-A / A_phys / E 的产物)**:跨世界**不变的共享底**,**恰恰不属于(会变的)视角,而是视角之下的不变底**。与交织自洽(同一性共享、世界相异)。
- **落笔**:公式仍写 视角=(𝒩, ω_ext);现象学解释时换 (U(s), u_t),标图景层、注明非双射(缺 𝒩'∩𝓜)。

## 2ter. 能动性 = 视角(不命名、经声部)+ 自由意志坐标(条117/118)

**不命名、不立术语,只经声部体现。** 弃用"能动性 / 自由意志 / gauge latitude"作主词——它们各有包袱(能动性=隐含施动者、与"无作为"矛盾;自由意志=利伯塔包袱;gauge latitude=撞 leeway、需规范群)。**那点松弛只通过语气体现**:松弛处用主动综合笔调,被给予处用领受笔调(见《写作声部规范》原则二)。

**这点自由的精确内容(供内部对齐,不入正文术语)**:
- **唯一自由度 = 面对二阶障碍 [σ](他者)时,在 agree-on-A 等价的诸切法里,哪段外部算"幔"(交换)、哪段算"噪音"(忽略)的非典范切取**(=§4bis QDS 三区梯度的伦理层)。
- **约束 = 维持 QDS 核不变(历史一致性),限度是"不致自我消解"**(可朝他者大幅松动到自身一致性消解的临界=列维纳斯激进的位置)。
- **"选择"非利伯塔式**:范围结构上真有(非典范、真未定),占据是被抛的、无无因选者;"选"=斯宾诺莎式看穿("切法是我习性结晶"⟹主动重切),经理解而非无因意志。
- **余圈本身不是自由度,松弛(非典范性)才是**(规范理论类比:规范势 vs 规范冗余;事实 A=规范不变量、ω_ext/余圈=规范选择、agree-on-A 等价类非平凡=自由)。**维数极小但因无迹无刻度⟹容量满="一丢丢即全部"**(不写数字;本质=身体含大脑的自动运行≠主观能动)。

**自由意志派系坐标(条118,不入论文、索引 canonical)**:sui generis——
- 否定面=硬不相容论/源头怀疑(Pereboom:决定论与非决定论皆给不了;"真概率不是能动"=运气异议;Strawson 基本论证:无 causa sui⟹无终极责任;斯宾诺莎为先驱);
- 正面=**把自由从"主体"挪到"结构"**(自由真实但不归任何人=必然里的非典范松弛,经理解启用;斯宾诺莎式,非取消论/虚无论);
- 伦理=列维纳斯(他者召唤先于自由,自由只在回应程度/切法);私下=无我(anatta)。
- **否定**:利伯塔论(无因意志)、相容论+半相容论(作为源头的主体;署名=幻觉 Wegner)、leeway 的推论(替代存在≠主体本可不然)、Smilansky"保留幻觉"、MP"我能"作为真能动("我能"是浪尖、底垫被动综合)。
- **两极端皆死(W.3 一刀)**:物理世界无能动(真概率≠能动);主观/感受质里无灵魂(感受=非对易=被给予)。
- **gauge latitude 处理**:仅留附录/脚注数学锚点(标:正则情形封顶 + 区别于 leeway;词源 Eichung=标尺,"无迹⟹选约定标尺自由"与现代 gauge freedom 同构),正文不用。

## 4. 感受与充实的精细结构:对比度 = 模谱带通

**充实 = 非平凡模轨道**(σ_t(x)≠x,x∉中心化子 M^ω),faxing 已认;但**梯度,不一刀切**。三档:

| 档位 | 现象学 | 数学读法 | 状态 |
|---|---|---|---|
| 焦点 | 清晰核 | 模轨道非平凡且强,过阈 | 公设 |
| 晕(fringe) | 半明半暗说不清(胡塞尔晕/詹姆斯 fringe) | 模轨道非平凡但弱,未过阈 | 公设 |
| 沉默 | 沉入不可分辨 | 落中心化子(σ 不动)或压入深层同态核 | 封顶结构 |

**对比度的候选公式(图景/待核):模算子谱的带通权重。**
- 模哈密顿量 K=−logΔ 给每个元素一个"模频率谱"(Arveson/Connes 谱)。
- x∈中心化子 ⟺ 谱={0} ⟺ 死(下限端);谱铺到无穷远 ⟺ 太非对易、出因果窗口 ⟺ 够不到(上限端)。
- **候选量:x 的模谱测度在窗口 [a,b] 上的权重**。a=可分辨下限(低于落回中心化子),b=因果窗口可达上限(高于指不过去)。**带通滤波,非单调**——故一直找不到。**b 由 𝒩 定出(待挖,可能是把"指不过去"严格化的钥匙)。**
- **III₁ 的回报(可入书,图景)**:III₁ 定义上 Connes 谱=整个 ℝ(满谱)。故**带通窗口可坐在任何频率上,感受谱最丰富**;II/I 型谱稀疏,可坐的"对比频率"少。**感受的连续对比度是 III₁ 自己逼出的,非外加。**

**场论直觉(faxing 要的"洞"):** 共振 / 准束缚态 / **Fano 共振**——分立态嵌连续谱,干涉出非对称峰+谷;纯分立(完全 split,"自己粒子化")和纯连续(完全糊)两极限都洗掉对比,只有"半开"中段有谱峰。"得连续有范围又得够深"= 需要谱宽 + 准束缚。

**对比度的现象学双锚(补单一资料,一管下限一管上限):**
- **Abhebung(突显/凸起,胡塞尔被动综合)**:东西要从背景凸起才有感发(Affektion)——管**下限**(不突出没用)。
- **图—底(格式塔/梅洛-庞蒂)**:图形须从底站出,但完全脱底就不再是场里的图形——管**上限**(过凸即脱场、完全 split 也没用)。

**"纯当下不可充实"(Derrida)**:感受=模轨道,冻结当下点无轨道 → 纯当下结构上不可充实,充实须借 σ 摊开(借 N 显形)。胡塞尔"当下总被充实"=σ 永远非平凡(无迹逼);Derrida"从不纯粹充实"=流冻不成对易自足当下。两句都对、落不同层,不撞红线 5。
> 方向修正:感受=流对象,不住独立切面;勿往"切面算子消失再冒出"想(伪图景+撞红线 5)。
