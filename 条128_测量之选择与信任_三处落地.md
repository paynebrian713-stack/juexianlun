## 一、第一章 §1.2 末 · 补 [^notmera]（挡物理读者三连问）

### 1.1 为什么必要（封顶依据）

物理读者读到"学习流＝保态条件期望、压成对易账本"会经 MERA 映射**连锁触发三问**，三问的答案全部已裁（W.2.2′④ MERA 加固／§1.4"补不回"／§4.6 无通用公式），但现在只在附录与第四章——物理读者在第一章就卡住，等不到。**一条主脚注三段挡三问，一次接住。**

### 1.2 脚注文字（可直接落）

**[^notmera]:** （给熟悉重整化的读者：学习流不是 MERA，这里说清三件容易混的事。）

**其一，学习流不是 MERA。** MERA（多尺度纠缠重整化）是一列**幺正**的 disentangler 加等距映射，可逆、保全信息，是一个变分**态**的高效表示。学习流是**保态条件期望** $E:\mathcal{M}\to\mathcal{A}_{\text{phys}}$：幂等、保基线态（$\omega^*\circ E=\omega^*$）、值域是模不变子代数，而且**丢信息**——有非平凡的核 $\ker E$（正是被压掉的那层非对易信息）。一个可逆保信息、一个不可逆丢信息，不是同一类对象；两者只在"逐层粗粒化"这个精神上相通，机制相反。

**其二，正因如此，压缩才不可逆。** MERA 可逆，于是有人会问：压缩怎么会单向？答案在其一——$E$ 不是幺正的，是投影，$\ker E$ 一旦丢掉就取不回。要重建丢掉的，需要一把"第二尺子"校准丢了多少、怎么丢的；而这台机器无迹，所以压缩是结构性单向的（详 §1.4 那个"补不回"，及第二章那道更硬的墙）。把学习流当 MERA，就会误以为压缩能反演。

**其三，"没有通用公式"不是遗漏，是无迹的后果。** 熟悉开放系统的读者会想到 GKSL（Lindblad）给这类耗散半群统一的生成元形式。但 GKSL 的通用形式活在 Type I（有迹）上；在无迹的 $\text{III}_1$ 上没有这样的通用形式，容许的生成元类只能相对基线锚 $\omega^*$ 的模结构才定义得出（详第四章 §4.6 与 [^db]）。反过来说，MERA 看似"不要参照、只要变分目标"，其实把参照藏进了那个变分目标——disentangler 要相对一个基态去优化，没有那个基态就不知道该 disentangle 什么。所以 MERA 不是"不需要基线态"的反例，是"需要、只是藏起来了"的又一例（详附录 W.2.2′）。

### 1.3 落地红线

- 脚注只**挡疑问＋指去处**（§1.4／§4.6／W.2.2′），不在脚注里展开证明；
- 其三守两条：①"GKSL 通用形式在 Type I、$\text{III}_1$ 上没有"可以说（已裁）；②**不得**说成"$\text{III}_1$ 上什么都算不出"——是"没有**独立于参照**的通用公式"，相对参照仍定型；
- 其三与 §4.6 QDS 段是同一件事两次出现（$E$／$\Phi_t$ 两个部件），措辞呼应、明确回指 §4.6，不打架。

## 二、导论 · 加一小段入口（"选择与信任测量仪器"＝全书指称）

### 2.1 封顶概念（依据）

**框架的每个动力学部件，其指称即物理学家"计算之外"的一个选择/信任动作**——这不是应用，是指称本身（根据：观察者入理论内 ⟹ 观察者每个选择/信任动作须在理论内有对应物）：

| 部件 | 指称的动作 |
|---|---|
| 参考态 $\omega_{\text{ext}}$ | 选视角（从哪里量） |
| 基线态 $\omega^*$ | 信背景/真空（凭什么这个真空） |
| 学习流 $E$／账本 $\mathcal{A}$ | 选基／信这组读数是事实（凭什么这组可观测量能同时测） |
| 理解流 $\mathcal{N}$ | 信历史读数连贯 |
| $D$ 对 | 信刻度/距离 |
| 稳态 $\Phi_t$＝身体 | 信仪器（含我这具身体）在测量中稳定不塌 |

### 2.2 入口段草稿（内容已封顶，voice 交文学会话）

> 物理学家每天做两件事。一件写在方程里：给定仪器、给定基、给定初态，算演化、算预测。另一件从不写进方程：**选择**这台仪器、**信任**这组读数、**认定**这个基没选错、**默认**仪器（连同他自己的身体）在测量过程中稳定不塌。前一件他们做了三百年，做到了登峰造极；后一件他们每天都在做，却从不回头看——它是全部计算的地基，又不在任何计算之内。
>
> 这本书讲的就是后一件事的结构。你往后会遇到的每一个部件——参考态、两股流、那块几何的板——翻回日常语言，都是这后一件事里的一个动作：选一个看的地方，信一组读数是事实，认一段历史连贯，托付一具会稳住的身体。本书不教你怎么把这些动作做对（往后会证明：那个"做对"的判据，恰恰是够不到的）；它做的是把这些动作一个个摊开，看清每个动作凭什么可能、以及它的底在哪里断掉。

### 2.3 落地红线（硬）

- ⚠️ **元规则 M（最狠的一处）**：**不得**写成"框架指导/预测怎么选仪器、哪个测量更可信"——"信任仪器"恰是最不可内生判定的（仪器客观稳定＝R★ 环境级、够不到）。只写"**刻画**选择与信任这个动作的结构与**界限**"；
- 入口段必须含"本书不教你做对、那个判据够不到"这半句（草稿已含），防读者带着"求一套选仪器指南"的期待进来；
- 位置：导论 §一（现象学主场）之后、§二之前为宜，或并入 §一收尾——由文学会话按节奏定，内容红线如上。

### 3.2 钥匙段草稿（内容已封顶，voice 交文学会话）

> 还有一把回望的钥匙，留给拿仪器吃饭的读者。你这一路遇到的所有部件，其实都在说你实验室里每天发生、却从不写进论文的那部分：参考态，是你选定从哪里量；基线，是你托付的那个真空；学习流压出的账本，是你认定"这组读数是事实"的那一下；历史一致性，是你信这台仪器昨天和今天是同一台；那块几何的板，是你从不追问的刻度；而身体那个稳态，是你默认自己的手、眼、仪器，在读数落下的那一瞬没有塌。这些动作没有一个在方程里，可没有它们，一个方程也立不起来。这本书没有告诉你哪台仪器值得信——它证明了那个判据够不到。它只是把你每天咽下去的那半个物理学，摊开给你看了一遍：计算之内的那半，你们做到了登峰造极；计算之外的这半——选择，然后信任——它一直没有底，而你们一直站在上面。
